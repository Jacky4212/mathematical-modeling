9. 存储论（库存论）

灰色系统 →

一、存储论概述

存储论（或称为库存论）是定量方法和技术最早的领域之一，是研究存储系统的性质、运行规律以及如何寻找最优存储策略的一门学科，是运筹学的重要分支。存储论的数学模型一般分成两类：确定性模型（不含随机因素）和随机存储模型。

1.1 存储问题的基本要素

要素符号含义
需求率D 或 R单位时间内对某种物品的需求量
订货批量Q一次订货中包含某种货物的数量
订货间隔期T两次订货之间的时间间隔

1.2 基本费用

费用类型符号含义
存储费C₁单位货物单位时间的存储费用（资金利息、库房、人力等）
短缺损失费C₂单位货物单位时间的缺货损失
订货费C₃每组织一次订货的固定费用（与订购数量无关）

二、常见存储策略

策略描述
t-循环策略每隔固定时间t补充固定量Q，不论实际存储状态
(t, S)策略每隔固定时间t补充一次，补充量 Q = S − I（补足到最大存储量S）
(s, S)策略存量为I时：若 I ≤ s，补充 Q = S−I；若 I > s，不补充。s为订货点（安全存量）
(t, s, S)策略每隔t盘点一次，若 I ≤ s 则补充到 S。是最完整的策略

三、确定性存储模型

3.1 模型一：EOQ（不允许缺货，补充时间极短）

EOQ = Economic Ordering Quantity（经济订购批量）

最经典的存储模型。假设：不允许缺货（C₂=∞）、补充瞬时完成（P=∞）、需求连续均匀（R为常数）。

核心结论 — EOQ公式：
$$t^* = \sqrt{\frac{2C_3}{C_1 R}} \quad \text{（最佳订货周期）}$$
$$Q^* = \sqrt{\frac{2C_3 R}{C_1}} \quad \text{（经济订购批量）}$$
$$C^* = \sqrt{2 C_1 C_3 R} \quad \text{（最小平均费用）}$$
例题：某商品单位成本5元，每天保管费为成本的0.1%，每次订购费10元，需求100件/天，不允许缺货。
C₁ = 5×0.1% = 0.005元/件·天，C₃ = 10元，R = 100件/天
t* = 6.32天，Q* = 632件，C* ≈ 3.16元/天
结论：每隔6.32天进货632件，总费用最少。

3.2 模型二：允许缺货+补充需要时间（经济生产批量）

假设条件

需求连续均匀（R常数）；生产连续均匀，速度P>R；允许缺货（C₂有限）；订购费C₃。

存储周期 [0, T]：t₁开始生产 → t₂缺货补足 → t₃停止生产达最大存量A → T时刻存量为零。

需求一方面以速度R满足，另一方面以速度P−R增加存储/补充缺货。

最优存储策略参数：
最优存储周期：$T^*$   经济生产批量：$Q^*$   最大存储量：$A^*$   最大缺货量：$B^*$

推导路线：从各时段关系建立时间参数 → 计算存储费和缺货费 → 求得平均总费用 → 对决策变量求导得最优值。

3.3 模型三：不允许缺货+补充需要时间

推导

在模型二中令 C₂ → ∞（不允许缺货），t₂ = 0，即得模型三。此时没有缺货阶段，生产的产品一部分满足需求，剩余部分作为存储。

参数公式
最优存储周期$T^* = \sqrt{\frac{2C_3}{C_1 R} \cdot \frac{P}{P-R}}$
经济生产批量$Q^* = \sqrt{\frac{2C_3 R}{C_1} \cdot \frac{P}{P-R}}$
最大存储量$A^* = \frac{P-R}{P} \cdot Q^*$
例题（订货点问题）：月需求30件，进价300元，月存储费为进价的2%，订购费20元/次，订购后5天到货，到货速度2件/天。
P=2件/天，R=1件/天，C₁=0.2元/天·件，C₃=20元/次，t₀=5天，L=5件
最优：Q*=20件，T*=20天，A*=10件，C*=2元

3.4 模型四：允许缺货+补充时间极短

推导

在模型二中令 P → ∞（补充瞬时），即得模型四。每个周期：订货Q后先支付缺货S，剩余Q-S为最高存储量。

$$T^* = \sqrt{\frac{2C_3}{C_1 R} \cdot \frac{C_1 + C_2}{C_2}}, \quad Q^* = \sqrt{\frac{2C_3 R}{C_1} \cdot \frac{C_1 + C_2}{C_2}}$$
例题：订货费12000元/次，存储费0.3元/件·月，缺货损失1.1元/件·月，月需求8000件。
以年为单位：C₁=3.6，C₂=13.2，R=96000
全年订货3次，每次32000件，最大缺货量6857件，最优费用81257元。

3.5 模型五：经济订购批量折扣模型

折扣模型特点

商品价格K(Q)不固定，随订货量递增而递减（分段函数）。存储费C₁(Q) = r·K(Q)，其中r为存储费率。

目标函数由三部分组成：年平均存储费 + 年平均订货费 + 商品购买费用。由于价格分段，需在各价格区间分别计算EOQ，取总费用最小的订货量。

四、单周期随机库存模型（报童问题）

4.1 问题描述

报童问题（Newsboy Problem）

在整个需求期内只订购一次货物（订货量Q）。需求r为随机变量，概率P(r)已知。每卖出一件盈利k，未售出每件损失h。求最优Q使期望利润最大。

4.2 离散需求

最优订货量 Q* 满足：

$$\sum_{r \leq Q^*} P(r) \geq \frac{k}{k+h} > \sum_{r \leq Q^*-1} P(r)$$
例题：报亭每天出售报纸服从Poisson(500)分布，每份盈利0.15元，未售出亏损0.40元。
k/(k+h) = 0.15/0.55 ≈ 0.2727 → 在Poisson(500)中找累积概率≥0.2727的Q
Q* = 487份，每天盈利约70.93元。

4.3 连续需求

设需求r的概率密度函数为f(r)，累积分布函数为F(Q)，则最优订货量满足：

$$F(Q^*) = \frac{k}{k+h}$$

即取需求分布的 k/(k+h) 分位点作为最优订货量。

五、数学建模竞赛中的应用

存储论的竞赛应用场景：
1. 供应链优化：原材料采购批量、生产批量决策
2. 库存管理：超市补货策略（如2023国赛C题蔬菜补货）
3. 报童模型变体：易逝品（生鲜、报纸、机票）单周期订货
4. 折扣采购：大量采购享受价格折扣时的最优策略
5. 安全库存：需求不确定时设置安全库存水平

Python求解示例

import numpy as np from scipy.optimize import minimize_scalar from scipy.stats import norm # === EOQ模型 === def eoq(C1, C3, R): return np.sqrt(2 * C3 * R / C1) # 示例：C1=0.005, C3=10, R=100 Q_star = eoq(0.005, 10, 100) # 632.46 # === 报童模型（正态需求）=== k, h = 0.15, 0.40 # 盈利、损失 mu, sigma = 500, 22.36 # 需求均值、标准差 critical_ratio = k / (k + h) Q_opt = norm.ppf(critical_ratio, mu, sigma) print(f'最优订货量: {Q_opt:.0f}') # ~487
