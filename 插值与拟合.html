1. 插值与拟合

一、MATLAB 绘图基础

1.1 二维绘图 必须掌握

MATLAB 中最基本的绘图命令是 plot()，用于绘制二维线图。

基本语法
plot(x, y, '线型颜色标记')

其中 x 和 y 是相同长度的向量，分别存储横坐标和纵坐标数据。

常用线型选项：实线 '-'、虚线 '--'、点线 ':'、点划线 '-.'
颜色选项：红 'r'、绿 'g'、蓝 'b'、黑 'k'、白 'w'
标记选项：星号 '*'、圆圈 'o'、加号 '+'、点 '.'

1.2 图形标注

函数作用
title('文本')添加图形标题
xlabel('文本')添加 x 轴标签
ylabel('文本')添加 y 轴标签
text(x, y, '文本')在指定坐标添加文字
legend('曲线1','曲线2')添加图例
grid on / off添加/取消网格线
hold on / off保持/覆盖当前图形
subplot(m,n,p)将图形窗口分成 m×n 个子图，当前为第 p 个

1.3 三维绘图

三维曲线：plot3
plot3(x, y, z, '选项') — 绘制三维空间曲线
三维曲面：mesh / surf

绘制三维曲面的标准流程：

  1. 定义 x、y 的取值范围
  2. 用 [X, Y] = meshgrid(x, y) 生成网格点矩阵
  3. 计算 Z = f(X, Y)
  4. 用 mesh(X,Y,Z)（网格图）或 surf(X,Y,Z)（着色表面图）绘制
函数效果
mesh(X,Y,Z)三维网格曲面（连线着色）
surf(X,Y,Z)三维着色曲面
meshc(X,Y,Z)网格 + 底部等高线
meshz(X,Y,Z)网格 + 底部幕帘
surfc(X,Y,Z)着色曲面 + 底部等高线
surfl(X,Y,Z)带光照效果的着色曲面

二、插值方法（Interpolation）

2.1 什么是插值 重点

插值是根据已知离散数据点，构造一个连续函数来估算未知点的函数值。插值函数 必须严格经过所有已知数据点。

插值 vs 拟合：插值要求曲线经过每个已知点；拟合不要求经过所有点，而是追求整体误差最小。
选型原则：数据精确无噪声 → 插值；数据含噪声或规律待发现 → 拟合。

2.2 常用插值方法

方法原理优点缺点
最近邻插值
nearest		 取最近已知点的值 速度最快 不连续，阶梯状
线性插值
linear		 相邻两点连线取中间值 简单快速，连续 在节点处不可导
三次样条插值
spline		 分段三次多项式，C²连续 非常光滑，二阶连续 可能产生过冲振荡
保形分段三次插值
pchip		 分段三次 Hermite 插值 保持单调性，无振荡 光滑度略低于 spline
Lagrange 插值 构造 n 次多项式经过所有点 理论简洁优美 高次多项式 Runge 现象
Runge 现象：使用高次（如 ≥ 7 次）多项式插值等距节点时，在区间两端会产生剧烈振荡，插值误差反而增大。解决方法是使用分段低次插值（样条/Hermite）或选择 Chebyshev 节点。

2.3 Lagrange 插值公式

$$L(x) = \sum y_k \cdot l_k(x)，\text{其中 } l_k(x) = \prod_{j \neq k} \frac{x - x_j}{x_k - x_j}$$
% lagrange.m — 自定义 Lagrange 插值函数 function y = lagrange(x0, y0, x) n = length(x0); m = length(x); for i = 1:m z = x(i); s = 0.0; for k = 1:n p = 1.0; for j = 1:n if j ~= k p = p * (z - x0(j)) / (x0(k) - x0(j)); end end s = p * y0(k) + s; end y(i) = s; end end

三、一维插值 — interp1 高频考点

interp1 语法
yi = interp1(x, y, xi, 'method')

x：已知点横坐标（向量） y：已知点纵坐标（向量）
xi：待插值点的横坐标 method：插值方法

method 可选值：

选项含义
'nearest'最近邻插值
'linear'线性插值（默认）
'spline'三次样条插值
'pchip' / 'cubic'保形分段三次 Hermite 插值

示例 1：比较五种插值方法

已知 12 小时整点温度数据，估算每 0.1 小时的温度，比较五种方法的效果。

% example1.m — 五种插值方法对比 t = 1:12; temp = [5,8,9,15,25,29,31,30,22,25,27,24]; t0 = 1:0.1:12; y_nearest = interp1(t, temp, t0, 'nearest'); y_linear = interp1(t, temp, t0, 'linear'); y_spline = interp1(t, temp, t0, 'spline'); y_pchip = interp1(t, temp, t0, 'pchip'); y_lagrange = lagrange(t, temp, t0); subplot(2,3,1), plot(t, temp, 'r*'); subplot(2,3,2), plot(t0, y_nearest, 'r'); subplot(2,3,3), plot(t0, y_linear, 'r'); subplot(2,3,4), plot(t0, y_spline, 'r'); subplot(2,3,5), plot(t0, y_pchip, 'r'); subplot(2,3,6), plot(t0, y_lagrange, 'r');

示例 2：Spline vs PCHIP — 振荡对比

% example2.m — Spline 与 PCHIP 差异 x = 0:0.3:3; y = (x.^2 - 4*x + 2) .* sin(x); x0 = 0:0.01:3; y_spline = interp1(x, y, x0, 'spline'); y_pchip = interp1(x, y, x0, 'pchip'); plot(x, y, 'o'); hold on plot(x0, y_spline, 'r'); plot(x0, y_pchip, 'b');
关键区别：Spline 追求最大光滑性（C² 连续），但可能在数据突变处产生过冲振荡。PCHIP 牺牲部分光滑性来保持数据的单调性，不会产生额外振荡。 竞赛建议：数据变化平缓 → Spline；数据有突变 → PCHIP。

四、二维插值

4.1 网格数据：interp2 重点

interp2 语法
ZI = interp2(X, Y, Z, XI, YI, 'method')

X, Y：已知网格点坐标（可用 meshgrid 生成） Z：已知函数值矩阵
XI, YI：待插值网格 method：'nearest' | 'linear'(默认) | 'cubic' | 'spline'

示例 3：二维网格插值

% example3.m — 二维温度数据插值 x = 1:5; y = 1:3; z = [82 81 80 82 84; 79 63 61 65 81; 84 84 82 85 86]; x0 = 1:0.1:5; y0 = 1:0.1:3; z_cubic = interp2(x, y, z, x0', y0, 'cubic'); mesh(x0, y0, z_cubic);

4.2 散点数据：griddata 竞赛常用

griddata 语法
ZI = griddata(x, y, z, XI, YI, 'method')

用于不规则散点的二维插值。method：'linear' | 'cubic' | 'nearest' | 'v4'

示例 5：散点水深数据插值

% example5.m — 散点水深数据网格化 x = [129 140 103.5 88 185.5 195 105 157.5 107.5 77 81 62 162 117.5]; y = [7.5 141.5 23 147 22.5 137.5 85.5 -6.5 -81 3 56.5 -66.5 84 -33.5]; h = -[4 8 6 8 6 8 8 9 9 8 8 9 4 9]; x0 = 75:1:200; y0 = -50:1:125; z = griddata(x, y, h, x0', y0, 'cubic'); mesh(x0, y0, z);

五、数据拟合（Curve Fitting）高频考点

5.1 最小二乘法原理

最小二乘法的目标是找到函数 f(x)，使所有数据点的残差平方和最小：

$$\min \sum [y_i - f(x_i)]^2$$
与插值的区别：拟合不要求曲线经过已知点，而是追求整体趋势的最佳逼近。数据含噪声或测量误差时，拟合优于插值。

5.2 多项式拟合：polyfit / polyval 必须掌握

polyfit 语法
p = polyfit(x, y, n) — 用 n 次多项式拟合数据
y_fit = polyval(p, x) — 计算拟合多项式在 x 处的值

示例 6：不同次数多项式拟合对比

% example6.m — 1/2/3 次多项式拟合对比 x = [0.3 0.4 0.7 0.9 1.2 1.9 2.8 3.2 3.7 4.5]; y = [1 2 3 4 2 6 9 2 7 8]; a1 = polyfit(x, y, 1); % 一次拟合 a2 = polyfit(x, y, 2); % 二次拟合 a3 = polyfit(x, y, 3); % 三次拟合 z1 = polyval(a1, x); z2 = polyval(a2, x); z3 = polyval(a3, x); plot(x, y, '+'); hold on plot(x, z1, 'r', x, z2, 'b', x, z3, 'g');
如何选择多项式次数？次数太低 → 欠拟合（偏差大）；次数太高 → 过拟合（方差大，对新数据泛化差）。可使用交叉验证或 AIC/BIC 准则选最优次数。

5.3 超定方程组求解

多项式拟合也可以用超定方程组直接求解：

$$A = [x^n \dots x \ 1]，\text{系数} = A \backslash y$$

即求解 $\beta = (A^T A)^{-1} A^T y$，MATLAB 中直接使用反斜杠运算符 A \ y。

5.4 非线性拟合 重点

函数语法适用场景
lsqcurvefitx = lsqcurvefit(@fun, x0, xdata, ydata)非线性最小二乘曲线拟合
nlinfitbeta = nlinfit(x, y, @fun, beta0)非线性回归
lsqnonlinx = lsqnonlin(@fun, x0)非线性最小二乘（残差形式）

示例 8：指数衰减模型拟合

% fun1.m — 拟合目标函数 function f = fun1(x, tdata) f = x(1) + x(2) * exp(-0.02 * x(3) * tdata); end % example8.m — 非线性拟合 tdata = 100:100:1000; cdata = 0.001 * [4.54 4.99 5.35 5.65 5.90 6.10 6.26 6.39 6.50 6.59]; x0 = [0.1 0.1 0.1]; % 初始猜测值 x = lsqcurvefit('fun1', x0, tdata, cdata); f = fun1(x, tdata); plot(tdata, cdata, '*', tdata, f); % 或用 nlinfit betafit = nlinfit(tdata, cdata, 'fun1', x0);
初值选择：非线性拟合需要提供初始猜测值 x0，不同的初值可能收敛到不同的局部最优解。
竞赛技巧：先用线性化方法（取对数、变量变换等）得到粗略估计作为初值，再用非线性拟合精化。
