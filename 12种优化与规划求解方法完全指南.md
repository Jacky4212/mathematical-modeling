## 1. 线性规划 (Linear Programming)

### 1.1 数学原理

线性规划研究**线性目标函数**在线性约束条件下的极值问题。标准形式为：

**目标函数**:
$$\min \; \mathbf{c}^T \mathbf{x} \quad \text{或} \quad \max \; \mathbf{c}^T \mathbf{x}$$

**约束条件**:
$$\begin{cases} \mathbf{A}\mathbf{x} \leq \mathbf{b} \\ \mathbf{x} \geq \mathbf{0} \end{cases}$$

其中 $\mathbf{c} \in \mathbb{R}^n$ 为价值系数向量，$\mathbf{A} \in \mathbb{R}^{m \times n}$ 为约束矩阵，$\mathbf{b} \in \mathbb{R}^m$ 为资源约束向量。

**核心定理**:
- **可行域是凸多面体**: 线性约束定义的交集是凸集
- **最优解在顶点处取得**: 若最优解存在，必在可行域的某个极点(顶点)处达到
- **单纯形法 (Simplex Method)**: 从某一顶点出发，沿边迭代至更优顶点，直到所有检验数满足最优性条件
- **对偶理论**: 每个LP问题(原问题)存在对偶问题，弱对偶性和强对偶性定理保证了解的界

**单纯形法核心步骤**:
1. 将问题化为标准型（引入松弛变量/剩余变量/人工变量）
2. 建立初始单纯形表
3. 计算检验数 $\sigma_j = c_j - \mathbf{c}_B^T \mathbf{B}^{-1} \mathbf{a}_j$
4. 若所有 $\sigma_j \geq 0$（min问题），则当前解为最优；否则选择入基变量
5. 通过最小比值法选择出基变量: $\theta = \min\left\{\frac{(\mathbf{B}^{-1}\mathbf{b})_i}{(\mathbf{B}^{-1}\mathbf{a}_k)_i} \;\middle|\; (\mathbf{B}^{-1}\mathbf{a}_k)_i > 0\right\}$
6. 进行基变换（Gauss-Jordan消元），转步骤3

**内点法**: Karmarkar(1984)提出，从可行域内部出发逼近最优解，复杂度 $O(n^{3.5}L)$，比单纯形法在最坏情况下更优。

### 1.2 适用场景

| 场景 | 示例 |
|------|------|
| 资源分配 | 原材料分配、资金预算分配、人员排班 |
| 生产计划 | 多种产品的产量规划、利润最大化 |
| 运输问题 | 最小化运输成本的物资调运 |
| 配料/混合 | 满足营养/成分要求的最低成本配方 |
| 投资组合 | 在约束下最大化预期收益（简化版） |
| 网络流 | 最大流问题、最小费用流问题 |

### 1.3 优点

- **理论成熟**: 有严格的数学理论支撑（单纯形法、对偶理论、灵敏度分析）
- **求解高效**: 商业求解器（Gurobi、CPLEX）和开源求解器（GLPK、HiGHS）都非常成熟
- **全局最优**: 凸优化问题的局部最优即全局最优
- **灵敏度分析**: 可以对约束右端项和目标系数进行灵敏度分析（影子价格）
- **确定性**: 给定相同的输入，总是得到相同的最优解

### 1.4 缺点

- **线性假设**: 实际问题往往是非线性的，线性近似可能不准确
- **连续变量要求**: 不允许整数约束（需转整数规划）
- **确定性假设**: 模型中所有参数必须是确定的（不考虑随机性）
- **规模限制**: 超大规模问题（百万级变量）需要专门求解器

### 1.5 Python代码实现

```python
"""
线性规划求解示例
使用 scipy.optimize.linprog (HiGHS求解器)
"""
import numpy as np
from scipy.optimize import linprog


# ==================== 示例1: 生产计划问题 ====================
# max Z = 3x1 + 5x2  (利润最大化)
# s.t.
#   x1 + x2 <= 4       (资源A)
#   2x1 + x2 <= 6      (资源B)
#   x1 + 2x2 <= 8      (资源C)
#   x1, x2 >= 0

def example1():
    print("=" * 60)
    print("示例1: 生产计划问题")
    print("=" * 60)
    
    # scipy的linprog默认求min，所以目标函数取负
    c = [-3, -5]  # min (-3x1 - 5x2) 等价于 max (3x1 + 5x2)
    
    A_ub = [
        [1, 1],    # x1 + x2 <= 4
        [2, 1],    # 2x1 + x2 <= 6
        [1, 2],    # x1 + 2x2 <= 8
    ]
    b_ub = [4, 6, 8]
    
    bounds = [(0, None), (0, None)]  # x1 >= 0, x2 >= 0
    
    result = linprog(c, A_ub=A_ub, b_ub=b_ub, bounds=bounds, method='highs')
    
    if result.success:
        x1, x2 = result.x
        print(f"最优解: x1 = {x1:.4f}, x2 = {x2:.4f}")
        print(f"最大利润: Z = {-result.fun:.4f}")
        print(f"迭代次数: {result.nit}")
    else:
        print(f"求解失败: {result.message}")
    
    # 灵敏度分析——影子价格
    print("\n各约束的影子价格（对偶变量值）:")
    if hasattr(result, 'ineqlin') and hasattr(result.ineqlin, 'marginals'):
        for i, sp in enumerate(result.ineqlin.marginals):
            print(f"  资源{i+1}: {sp:.4f}")


# ==================== 示例2: 运输问题 ====================
# 三个产地(A1,A2,A3) → 四个销地(B1,B2,B3,B4)
# 最小化运输成本
def example2():
    print("\n" + "=" * 60)
    print("示例2: 运输问题")
    print("=" * 60)
    
    # 产地供应量
    supply = [7, 4, 9]
    # 销地需求量
    demand = [3, 6, 5, 6]
    
    # 单位运输成本矩阵 (3产地 × 4销地)
    cost = np.array([
        [3, 11, 3, 10],
        [1, 9, 2, 8],
        [7, 4, 10, 5]
    ])
    
    m, n = cost.shape
    # 决策变量: x_ij, flatten成一维, 共 m*n 个
    c = cost.flatten().tolist()
    
    # 供应约束: 每个产地的出货量 = supply
    A_eq_supply = []
    for i in range(m):
        row = [0] * (m * n)
        for j in range(n):
            row[i * n + j] = 1
        A_eq_supply.append(row)
    b_eq_supply = supply
    
    # 需求约束: 每个销地的到货量 = demand
    A_eq_demand = []
    for j in range(n):
        row = [0] * (m * n)
        for i in range(m):
            row[i * n + j] = 1
        A_eq_demand.append(row)
    b_eq_demand = demand
    
    A_eq = A_eq_supply + A_eq_demand
    b_eq = b_eq_supply + b_eq_demand
    
    bounds = [(0, None)] * (m * n)
    
    result = linprog(c, A_eq=A_eq, b_eq=b_eq, bounds=bounds, method='highs')
    
    if result.success:
        x = result.x.reshape(m, n)
        print(f"最优运输方案 (x_ij):")
        print(x)
        print(f"最小运输成本: {result.fun:.2f}")
    else:
        print(f"求解失败: {result.message}")


# ==================== 示例3: 带灵敏度分析的LP ====================
def example3():
    print("\n" + "=" * 60)
    print("示例3: 灵敏度分析——资源价值评估")
    print("=" * 60)
    
    # 原问题
    c = [-40, -50]  # max 40x1 + 50x2
    A_ub = [[1, 2], [4, 3], [1, 0]]
    b_ub = [40, 120, 25]
    bounds = [(0, None), (0, None)]
    
    result = linprog(c, A_ub=A_ub, b_ub=b_ub, bounds=bounds, method='highs')
    
    if result.success:
        print(f"最优解: x1 = {result.x[0]:.2f}, x2 = {result.x[1]:.2f}")
        print(f"最大收益: {-result.fun:.2f}")
    
    # 手工进行影子价格分析: 逐个增加资源量
    print("\n影子价格分析 (每增加1单位资源对目标函数的边际贡献):")
    for k in range(3):
        b_new = b_ub.copy()
        b_new[k] += 1
        r = linprog(c, A_ub=A_ub, b_ub=b_new, bounds=bounds, method='highs')
        shadow_price = (-r.fun) - (-result.fun) if r.success else None
        print(f"  资源{k+1}: 影子价格 = {shadow_price:.4f}")


if __name__ == "__main__":
    example1()
    example2()
    example3()
```

**关键库说明**:
- `scipy.optimize.linprog`: 支持单纯形法、内点法和HiGHS求解器
- `pulp`: Python建模语言，支持调用多种求解器（CBC、GLPK、CPLEX、Gurobi）
- `cvxpy`: 凸优化建模框架，代码更直观
- `ortools` (Google): 线性规划、整数规划、约束规划的综合工具包

## 2. 整数规划 (Integer Programming)

### 2.1 数学原理

整数规划是在线性规划基础上，要求部分或全部决策变量取整数的一类优化问题。

**分类**:
- **纯整数规划 (Pure IP)**: 全部变量取整数
- **混合整数规划 (MIP)**: 部分变量取整数，部分取实数
- **0-1规划 (Binary IP)**: 变量只取 0 或 1

**数学模型**:
$$\min \; \mathbf{c}^T \mathbf{x}$$
$$\text{s.t.} \; \mathbf{A}\mathbf{x} \leq \mathbf{b}, \quad \mathbf{x} \in \mathbb{Z}^n_+$$

**核心算法——分支定界法 (Branch and Bound)**:

1. **定界 (Bounding)**: 松弛整数约束得到LP松弛问题，其最优值为原IP问题的下界（min问题）
2. **分支 (Branching)**: 选择一个取值为分数的变量 $x_j = f$，分为两支:
   - $x_j \leq \lfloor f \rfloor$
   - $x_j \geq \lceil f \rceil$
3. **剪枝 (Pruning)**: 三种剪枝条件:
   - 该节点LP不可行
   - 该节点LP解已满足整数要求
   - 该节点LP目标值不超过当前最优值（min问题中，若下界 >= 当前上界，剪枝）

**核心算法——割平面法 (Cutting Plane)**:
- 在LP松弛问题的基础上，逐次添加线性约束（割平面），"切掉"不含整数解的可行域部分
- **Gomory割**: 从最优单纯形表中导出，是最经典的割平面

**0-1规划的特殊性**:
- 变量取值空间为 $2^n$，枚举不可行时需用隐枚举法
- 逻辑约束等价转换:
  - $y = x_1 \land x_2$: $y \leq x_1$, $y \leq x_2$, $y \geq x_1 + x_2 - 1$
  - $y = x_1 \lor x_2$: $y \geq x_1$, $y \geq x_2$, $y \leq x_1 + x_2$
  - 互斥约束: $x_1 + x_2 \leq 1$

**计算复杂性**: IP 是 NP-hard 问题，不存在多项式时间算法。

### 2.2 适用场景

| 场景 | 示例 |
|------|------|
| 0-1背包问题 | 有限容量下选择最有价值的物品组合 |
| 指派问题 | n个人分配n项任务的最小总成本 |
| 设施选址 | 在候选地中选择开设仓库的数量和位置 |
| 排班调度 | 护士排班、航班机组调度、考试时间表 |
| 投资组合 | 含固定交易费的离散投资选择 |
| 集合覆盖 | 选择最少的设施覆盖所有需求点 |
| 旅行商问题(TSP) | 访问所有城市恰好一次的最短回路 |
| 下料/切割 | 一维/二维切割中的整数长度优化 |

### 2.3 优点

- **建模能力强**: 能处理"是/否"决策、逻辑约束、计数约束等离散决策问题
- **0-1变量灵活**: 可表示开关、选择、互斥、依赖等逻辑关系
- **求解器成熟**: Gurobi、CPLEX等商业求解器可高效处理中等规模的MIP

### 2.4 缺点

- **NP-hard**: 最坏情况下计算时间指数增长
- **求解时间不确定**: 同一规模的不同实例求解时间差异可能极大
- **大规模困难**: 变量超过10万时，需要启发式算法或分解方法
- **需要调整参数**: 求解器通常需要调整割平面、启发式参数等

```python
"""
整数规划求解示例
使用 pulp (调用开源CBC求解器) 和 scipy + 手工分支定界
"""
import numpy as np
from scipy.optimize import linprog
from itertools import combinations


# ==================== 示例1: 0-1背包问题 (pulp求解) ====================
def example1_pulp():
    """用PuLP求解0-1背包问题"""
    try:
        import pulp
    except ImportError:
        print("请先安装: pip install pulp")
        return
    
    print("=" * 60)
    print("示例1: 0-1背包问题 (PuLP)")
    print("=" * 60)
    
    # 物品价值
    values = [60, 100, 120, 80, 50, 40]
    # 物品重量
    weights = [10, 20, 30, 15, 10, 5]
    # 背包容量
    capacity = 50
    n = len(values)
    
    # 创建问题: 最大化
    prob = pulp.LpProblem("Knapsack", pulp.LpMaximize)
    
    # 决策变量: x_i  ∈ {0, 1}
    x = [pulp.LpVariable(f"x{i}", cat='Binary') for i in range(n)]
    
    # 目标函数
    prob += pulp.lpSum([values[i] * x[i] for i in range(n)])
    
    # 容量约束
    prob += pulp.lpSum([weights[i] * x[i] for i in range(n)]) <= capacity
    
    # 求解
    prob.solve(pulp.PULP_CBC_CMD(msg=False))
    
    print(f"状态: {pulp.LpStatus[prob.status]}")
    print(f"最大总价值: {pulp.value(prob.objective):.0f}")
    for i in range(n):
        print(f"  物品{i+1} (价值{values[i]}, 重{weights[i]}): "
              f"{'选中' if pulp.value(x[i]) == 1 else '未选'}")

                print(f"  工人{i+1} → 任务{j+1} (成本{cost_matrix[i][j]})")


# ==================== 示例3: 手工分支定界法 ====================
def branch_and_bound(c, A_ub, b_ub, bounds):
    """
    手工实现分支定界法求解整数规划
    c, A_ub, b_ub: LP问题的参数 (min问题)
    bounds: [(l1,u1), (l2,u2), ...] 整数变量的下界和上界
    """
    best_x = None
    best_obj = np.inf  # 当前最优目标值 (min问题)
    nodes_explored = [0]
    
    def solve_node(local_bounds):
        """求解一个节点的LP松弛，返回(成功标志, 解向量, 目标值)"""
        result = linprog(
            c, A_ub=A_ub, b_ub=b_ub,
            bounds=local_bounds, method='highs'
        )
        if result.success:
            return True, result.x, result.fun
        return False, None, None
    
    def is_integer(x, tol=1e-6):
        """检查解是否整数"""
        return all(abs(v - round(v)) < tol for v in x)
    
    def branch(node_bounds):
        nonlocal best_x, best_obj
        nodes_explored[0] += 1
        
        success, x_relax, obj = solve_node(node_bounds)
        
        # 剪枝条件1: LP不可行
        if not success:
            return
        
        # 剪枝条件3: 当前下界 >= 已知最优值(再加-gap容忍)
        if obj >= best_obj - 1e-6:
            return
        
        # 剪枝条件2: 找到整数解
        if is_integer(x_relax):
            best_obj = obj
            best_x = x_relax.copy()
            return
        
        # 分支: 选第一个分数变量
        for i, val in enumerate(x_relax):
            if abs(val - round(val)) >= 1e-6:
                floor_val = int(np.floor(val))
                
                # 左分支: x_i <= floor(val)
                left_bounds = node_bounds.copy()
                left_bounds[i] = (node_bounds[i][0], floor_val)
                branch(left_bounds)
                
                # 右分支: x_i >= ceil(val)
                right_bounds = node_bounds.copy()
                right_bounds[i] = (floor_val + 1, node_bounds[i][1])
                branch(right_bounds)
                
                break
    
    branch(bounds)
    return best_x, best_obj, nodes_explored[0]


def example3_bnb():
    """分支定界法示例"""

    # 仓库容量
    capacity = [100, 150, 120]
    # 客户需求
    demand = [30, 40, 35, 45, 50]
    # 运输成本矩阵 [仓库i → 客户j]
    transport_cost = [
        [4, 6, 8, 5, 7],
        [7, 4, 5, 9, 6],
        [6, 8, 5, 6, 9]
    ]
    
    n_wh = len(fixed_cost)
    n_cust = len(demand)
    
    prob = pulp.LpProblem("Facility_Location", pulp.LpMinimize)
    
    # 决策变量
    y = [pulp.LpVariable(f"open_{i}", cat='Binary') for i in range(n_wh)]
    x = [[pulp.LpVariable(f"ship_{i}_{j}", lowBound=0) for j in range(n_cust)]
         for i in range(n_wh)]
    
    # 目标: 固定成本 + 运输成本
    prob += (pulp.lpSum([fixed_cost[i] * y[i] for i in range(n_wh)]) +
             pulp.lpSum([transport_cost[i][j] * x[i][j]
                          for i in range(n_wh) for j in range(n_cust)]))
    
    # 每个客户需求被满足
    for j in range(n_cust):
        prob += pulp.lpSum([x[i][j] for i in range(n_wh)]) == demand[j]
    
    # 仓库容量约束
    for i in range(n_wh):
        prob += (pulp.lpSum([x[i][j] for j in range(n_cust)])
                 <= capacity[i] * y[i])
    
    prob.solve(pulp.PULP_CBC_CMD(msg=False))
    
    print(f"状态: {pulp.LpStatus[prob.status]}")
    print(f"最小总成本: {pulp.value(prob.objective):.0f}")
    print("开设的仓库:")
    for i in range(n_wh):
        if pulp.value(y[i]) == 1:
            print(f"  仓库{i+1} (固定成本{fixed_cost[i]})")
            for j in range(n_cust):
                ship = pulp.value(x[i][j])
                if ship > 0:
                    print(f"    → 客户{j+1}: {ship:.0f}单位 "
                          f"(运输成本{transport_cost[i][j]}/单位)")


if __name__ == "__main__":
    example1_pulp()
    example2_assignment()
    example3_bnb()
    example4_facility_location()
```

## 3. 非线性规划 (Nonlinear Programming)

### 3.1 数学原理

非线性规划是目标函数或约束条件中至少有一个是非线性函数的优化问题。

**一般形式**:
$$\begin{aligned}
\min \quad & f(\mathbf{x}) \\
\text{s.t.} \quad & g_i(\mathbf{x}) \leq 0, \quad i = 1, \ldots, m \\
& h_j(\mathbf{x}) = 0, \quad j = 1, \ldots, p \\
& \mathbf{x} \in \mathbb{R}^n
\end{aligned}$$

**KKT条件 (Karush-Kuhn-Tucker)** — 一阶必要条件:

如果 $\mathbf{x}^*$ 是局部最优解且满足约束品性（如LICQ），则存在拉格朗日乘子 $\boldsymbol{\mu} \geq \mathbf{0}$ 和 $\boldsymbol{\lambda}$，使得:

$$\nabla f(\mathbf{x}^*) + \sum_{i=1}^{m} \mu_i \nabla g_i(\mathbf{x}^*) + \sum_{j=1}^{p} \lambda_j \nabla h_j(\mathbf{x}^*) = \mathbf{0}$$

$$\mu_i g_i(\mathbf{x}^*) = 0 \quad \text{(互补松弛条件)}$$

$$g_i(\mathbf{x}^*) \leq 0, \quad h_j(\mathbf{x}^*) = 0, \quad \mu_i \geq 0$$

**核心算法分类**:

**1. 无约束优化**:
- **梯度下降法**: $\mathbf{x}_{k+1} = \mathbf{x}_k - \alpha_k \nabla f(\mathbf{x}_k)$
- **牛顿法**: $\mathbf{x}_{k+1} = \mathbf{x}_k - [\nabla^2 f(\mathbf{x}_k)]^{-1} \nabla f(\mathbf{x}_k)$
- **拟牛顿法 (BFGS/L-BFGS)**: 用近似Hessian矩阵代替精确Hessian
- **共轭梯度法**: 适用于大规模问题

**2. 约束优化**:
- **序列二次规划 (SQP)**: 每一步求解一个二次规划子问题，收敛快
- **内点法**: 用障碍函数将不等式约束转化为"惩罚项"，从可行域内部向最优解逼近
- **增广拉格朗日法**: 结合拉格朗日法和罚函数法
- **信赖域方法**: 在信赖域内用二次模型近似原问题

**凸优化特殊性质**:
- 如果 $f$ 是凸函数，$g_i$ 是凸函数，$h_j$ 是仿射函数，则任何局部最优解就是全局最优解
- 凸优化可以通过内点法在多项式时间内求解

### 3.2 适用场景

| 场景 | 示例 |
|------|------|
| 参数估计 | 最小二乘法拟合经验模型参数 |
| 机器学习训练 | 神经网络、SVM、逻辑回归的损失函数最小化 |
| 工程最优设计 | 结构优化、形状优化、电路设计 |
| 投资组合 | Markowitz均值-方差模型（二次规划） |
| 化学平衡 | Gibbs自由能最小化 |
| 最优控制 | 轨迹规划、燃料最省轨道 |
| 图像处理 | 图像去噪的TV正则化模型 |

### 3.3 优点

- **建模精度高**: 可精确描述真实物理/经济过程中的非线性关系
- **算法多样**: 针对不同问题结构(凸/非凸、光滑/非光滑)有特定算法
- **凸优化理论强大**: 凸问题的局部=全局性质保证了解的质量

### 3.4 缺点

- **局部最优陷阱**: 非凸问题可能陷入局部最优解
- **对初值敏感**: 解的质野往往依赖于初始点选择
- **可能不收敛**: 非光滑、病态条件数等问题可能导致不收敛
- **计算成本高**: Hessian矩阵计算和存储代价大 ($O(n^2)$ 或 $O(n^3)$)
- **缺乏灵敏度分析**: 不像LP那样有成熟的对偶/灵敏度理论

```python
        """投资组合方差 = w^T Σ w"""
        return w @ Sigma @ w

        return w @ mu

    constraints.append(
        {'type': 'ineq', 'fun': lambda w: portfolio_return(w) - target_return}
    )
    
    w0 = np.ones(n_assets) / n_assets  # 等权重初始
    result = minimize(portfolio_variance, w0, method='SLSQP',
                      bounds=bounds, constraints=constraints)

        for i, w in enumerate(result.x):
            print(f"  资产{i+1}: {w:.4f}")

        print(f"投资组合标准差: {np.sqrt(result.fun):.4f}")

        print(f"优化失败: {result.message}")
```

## 4. 动态规划 (Dynamic Programming)

### 4.1 数学原理

动态规划是Bellman于1950年代提出的最优化方法，核心思想是将复杂问题分解为**重叠子问题**，通过**递推关系**自底向上求解。

**最优性原理 (Bellman Optimality Principle)**:
> 一个最优策略具有这样的性质：无论过去的状态和决策如何，对于当前决策所形成的状态而言，余下的决策必须构成一个最优子策略。

**基本要素**:

1. **阶段 (Stage)** $k$: 将过程划分为若干相互联系的阶段
2. **状态 (State)** $s_k$: 各阶段开始时的客观条件, $s_k \in S_k$
3. **决策 (Decision)** $x_k$: 从当前状态到下一阶段的选择, $x_k \in D_k(s_k)$
4. **状态转移方程**: $s_{k+1} = T_k(s_k, x_k)$
5. **报酬函数**: $v_k(s_k, x_k)$ — 阶段k在状态$s_k$下采取决策$x_k$的即时收益
6. **最优值函数**: $f_k(s_k)$ — 从阶段k的状态$s_k$出发到过程结束的最优总收益

**Bellman递推方程**:

**逆推法** (从最终状态向前递推):
$$f_k(s_k) = \text{opt}_{x_k \in D_k(s_k)} \left\{ v_k(s_k, x_k) + f_{k+1}(s_{k+1}) \right\}$$

其中 $s_{k+1} = T_k(s_k, x_k)$, 边界条件 $f_{N+1}(s_{N+1})$ 已知。

**顺推法** (从初始状态向后递推):
$$f_k(s_{k+1}) = \text{opt}_{x_k \in D_k(s_k)} \left\{ v_k(s_k, x_k) + f_{k-1}(s_k) \right\}$$

**解法对比**:
- **DP vs 分治法**: DP子问题重叠（用记忆化避免重复计算），分治法子问题不重叠
- **DP vs 贪心法**: DP保证全局最优，贪心只需局部最优选择

**DP的两个实现范式**:

1. **自顶向下（记忆化递归 / Memoization）**: 递归求解，用哈希表/数组缓存已计算结果
2. **自底向上（表格法 / Tabulation）**: 从小到大填表，消除递归开销

**维数灾难 (Curse of Dimensionality)**: 当状态变量个数增加时，计算量和存储量呈指数增长，Bellman称之为"维数灾难"。

### 4.2 适用场景

| 场景 | 示例 |
|------|------|
| 最短路径 | 有向无环图(DAG)的最短路 |
| 背包问题 | 0-1背包、完全背包、多重背包 |
| 资源分配 | 多阶段资源分配决策 |
| 序列决策 | 设备更新、库存管理、生产调度 |
| 字符串匹配 | 编辑距离(Levenshtein)、最长公共子序列(LCS) |
| 区间DP | 矩阵链乘、最优二叉搜索树 |
| 树形DP | 在树上做决策（独立集、支配集） |
| 状态压缩DP | TSP（Held-Karp算法）、集合覆盖 |
| 概率DP/期望DP | 马尔可夫决策过程(MDP)、强化学习基础 |

### 4.3 优点

- **保证全局最优**: 基于最优性原理，解的质量有理论保证
- **结构清晰**: 递推关系清晰表达了问题结构
- **灵活性高**: 可处理多种类型的约束和非线性关系
- **多解性**: 可同时得到多个阶段的最优解

### 4.4 缺点

- **维数灾难**: 高维状态空间导致计算和存储爆炸
- **无后效性要求**: 要求当前状态仅依赖于前一阶段，不适用的场景需要扩展
- **建模困难**: 复杂问题的状态定义和转移方程构建有挑战性
- **连续状态**: 连续状态空间需要离散化或函数近似

```python
"""
动态规划经典问题实现
"""
import numpy as np
from functools import lru_cache


# ==================== 示例1: 0-1背包问题 (二维DP) ====================
def knapsack_01(values, weights, capacity):
    """
    0-1背包问题 DP解法
    O(n*W) 时间, O(n*W) 空间
    """
    n = len(values)
    # dp[i][w]: 前i个物品,容量w的最大价值
    dp = [[0] * (capacity + 1) for _ in range(n + 1)]
    
    for i in range(1, n + 1):
        for w in range(1, capacity + 1):
            if weights[i-1] <= w:
                dp[i][w] = max(
                    dp[i-1][w],                          # 不选i
                    dp[i-1][w - weights[i-1]] + values[i-1]  # 选i
                )
            else:
                dp[i][w] = dp[i-1][w]
    
    # 回溯找出选中的物品
    selected = []
    w = capacity
    for i in range(n, 0, -1):
        if dp[i][w] != dp[i-1][w]:
            selected.append(i - 1)
            w -= weights[i-1]
    
    return dp[n][capacity], selected[::-1]


def knapsack_01_optimized(values, weights, capacity):
    """空间优化: O(W) 一维DP"""
    n = len(values)
    dp = [0] * (capacity + 1)
    
    for i in range(n):
        # 必须倒序! 否则一个物品可能被选多次
        for w in range(capacity, weights[i] - 1, -1):
            dp[w] = max(dp[w], dp[w - weights[i]] + values[i])
    
    return dp[capacity]


# ==================== 示例2: 编辑距离 (Levenshtein Distance) ====================
def edit_distance(s1, s2):
    """
    Levenshtein编辑距离: 将s1转换为s2所需的最少操作数
    操作: 插入/删除/替换 (每次操作代价为1)
    """
    m, n = len(s1), len(s2)
    # dp[i][j]: s1前i个字符 → s2前j个字符的最小编辑距离
    dp = [[0] * (n + 1) for _ in range(m + 1)]
    
    for i in range(m + 1):
        dp[i][0] = i  # 全部删除
    for j in range(n + 1):
        dp[0][j] = j  # 全部插入
    
    for i in range(1, m + 1):
        for j in range(1, n + 1):
            if s1[i-1] == s2[j-1]:
                dp[i][j] = dp[i-1][j-1]  # 无需操作
            else:
                dp[i][j] = 1 + min(
                    dp[i-1][j],      # 删除s1[i-1]
                    dp[i][j-1],      # 在s1[i-1]后插入s2[j-1]
                    dp[i-1][j-1]     # 替换s1[i-1] → s2[j-1]
                )
    
    return dp[m][n]


# ==================== 示例3: 最长公共子序列 (LCS) ====================
def lcs(s1, s2):
    """最长公共子序列 (Longest Common Subsequence)"""
    m, n = len(s1), len(s2)
    dp = [[0] * (n + 1) for _ in range(m + 1)]
    
    for i in range(1, m + 1):
        for j in range(1, n + 1):
            if s1[i-1] == s2[j-1]:
                dp[i][j] = dp[i-1][j-1] + 1
            else:
                dp[i][j] = max(dp[i-1][j], dp[i][j-1])
    
    # 回溯LCS
    i, j = m, n
    lcs_chars = []
    while i > 0 and j > 0:
        if s1[i-1] == s2[j-1]:
            lcs_chars.append(s1[i-1])
            i -= 1
            j -= 1
        elif dp[i-1][j] > dp[i][j-1]:
            i -= 1
        else:
            j -= 1
    
    return dp[m][n], ''.join(reversed(lcs_chars))


# ==================== 示例4: 资源分配问题 (多阶段DP) ====================
def resource_allocation(total_resource, n_projects, profit_func):
    """
    资源分配DP: total_resource分给n_projects个项目的最大利润
    profit_func(k, x): 第k个项目分配x资源后的利润
    """
    # dp[k][r]: 前k个项目,用掉r资源的最大利润
    dp = [[0] * (total_resource + 1) for _ in range(n_projects + 1)]
    # choice[k][r]: dp[k][r]时第k个项目分配的资源量
    choice = [[0] * (total_resource + 1) for _ in range(n_projects + 1)]
    
    for k in range(1, n_projects + 1):
        for r in range(total_resource + 1):
            best = 0
            best_x = 0
            for x in range(r + 1):
                val = dp[k-1][r - x] + profit_func(k, x)
                if val > best:
                    best = val
                    best_x = x
            dp[k][r] = best
            choice[k][r] = best_x
    
    # 回溯分配方案
    alloc = [0] * n_projects
    remaining = total_resource
    for k in range(n_projects, 0, -1):
        alloc[k-1] = choice[k][remaining]
        remaining -= alloc[k-1]
    
    return dp[n_projects][total_resource], alloc


# ==================== 示例5: 旅行商问题 (状态压缩DP) ====================
def tsp_dp(dist_matrix):
    """
    TSP的Held-Karp DP算法 O(n^2 * 2^n)
    dp[mask][i]: 已访问城市集mask, 最后在城市i的最短路径
    """
    n = dist_matrix.shape[0]
    INF = float('inf')
    
    # dp[mask][i], mask用位表示
    dp = [[INF] * n for _ in range(1 << n)]
    parent = [[-1] * n for _ in range(1 << n)]
    
    # 从城市0出发
    dp[1][0] = 0
    
    for mask in range(1 << n):
        for u in range(n):
            if dp[mask][u] == INF:
                continue
            # 尝试从u去v
            for v in range(n):
                if mask & (1 << v):  # v已访问
                    continue
                new_mask = mask | (1 << v)
                new_dist = dp[mask][u] + dist_matrix[u][v]
                if new_dist < dp[new_mask][v]:
                    dp[new_mask][v] = new_dist
                    parent[new_mask][v] = u
    
    # 找最短回路
    full_mask = (1 << n) - 1
    best = INF
    last_city = -1
    for i in range(1, n):
        total = dp[full_mask][i] + dist_matrix[i][0]
        if total < best:
            best = total
            last_city = i
    
    # 回溯路径
    path = [0]
    mask = full_mask
    v = last_city
    while v != -1:
        path.append(v)
        u = parent[mask][v]
        mask ^= (1 << v)
        v = u
    path.append(0)
    
    return best, path[::-1]


# ==================== 示例6: 矩阵链乘 (区间DP) ====================
def matrix_chain_order(dims):
    """
    矩阵链乘最优括号化 — 区间DP
    dims: [p0, p1, ..., pn], 第i个矩阵尺寸为 dims[i-1] × dims[i]
    """
    n = len(dims) - 1
    INF = float('inf')
    # dp[i][j]: A_i × ... × A_j的最小乘法次数
    dp = [[0] * n for _ in range(n)]
    split = [[0] * n for _ in range(n)]
    
    for length in range(2, n + 1):      # 区间长度
        for i in range(n - length + 1):  # 区间起点
            j = i + length - 1            # 区间终点
            dp[i][j] = INF
            for k in range(i, j):
                cost = (dp[i][k] + dp[k+1][j] +
                        dims[i] * dims[k+1] * dims[j+1])
                if cost < dp[i][j]:
                    dp[i][j] = cost
                    split[i][j] = k
    
    def build_parens(i, j):
        if i == j:
            return f"A{i}"
        k = split[i][j]
        left = build_parens(i, k)
        right = build_parens(k+1, j)
        return f"({left} × {right})"
    
    return dp[0][n-1], build_parens(0, n-1)
```

## 5. 多目标优化 (Multi-objective Optimization)

### 5.1 数学原理

多目标优化研究在给定区域内，使多个可能冲突的目标函数同时达到最优的问题。

**一般形式**:
$$\begin{aligned}
\min \quad & \mathbf{F}(\mathbf{x}) = [f_1(\mathbf{x}), f_2(\mathbf{x}), \ldots, f_m(\mathbf{x})]^T \\
\text{s.t.} \quad & \mathbf{x} \in \Omega
\end{aligned}$$

**Pareto最优 (Pareto Optimality)**:

- **Pareto支配**: 解 $\mathbf{x}^{(1)}$ 支配 $\mathbf{x}^{(2)}$ (记作 $\mathbf{x}^{(1)} \prec \mathbf{x}^{(2)}$), 当且仅当对所有 $i \in \{1,\ldots,m\}$ 有 $f_i(\mathbf{x}^{(1)}) \le f_i(\mathbf{x}^{(2)})$，且至少有一个严格不等式成立
- **Pareto最优解**: 不存在其他解支配它的解
- **Pareto前沿 (Pareto Front)**: 所有Pareto最优解在目标空间的映射

$$\text{PF} = \{ \mathbf{F}(\mathbf{x}) \mid \mathbf{x} \in \Omega \text{ 是Pareto最优} \}$$

**主要求解方法**:

**方法1: 权重法 (Weighted Sum)**
$$\min \; \sum_{i=1}^{m} w_i f_i(\mathbf{x}), \quad w_i \ge 0, \;\sum w_i = 1$$
- 改变权重可得到Pareto前沿上的不同点
- **局限**: 只能得到凸Pareto前沿上的点

**方法2: ε-约束法 ($\varepsilon$-Constraint)**
$$\begin{aligned}
\min \; & f_k(\mathbf{x}) \\
\text{s.t.} \; & f_j(\mathbf{x}) \le \varepsilon_j, \quad j \neq k
\end{aligned}$$
- 将 $m-1$ 个目标转为约束
- 可得到任意Pareto前沿（包括非凸的）

**方法3: 目标规划 (Goal Programming)**
$$\min \; \sum_{i=1}^{m} (d_i^+ + d_i^-)$$
$$\text{s.t.} \quad f_i(\mathbf{x}) + d_i^- - d_i^+ = g_i, \quad d_i^+, d_i^- \ge 0$$
其中 $g_i$ 是第 $i$ 个目标的理想值，$d_i^+, d_i^-$ 分别是正负偏差变量。

**方法4: 多目标进化算法 (MOEA)** — 见第7-10章

### 5.2 适用场景

| 场景 | 示例 |
|------|------|
| 投资组合 | 最大化收益 vs 最小化风险 |
| 工程优化 | 最大化强度 vs 最小化重量/成本 |
| 供应链 | 最小化成本 vs 最大化服务水平 vs 最小化碳排放 |
| 水资源管理 | 经济效益 vs 生态保护 vs 社会公平 |
| 车辆路径 | 最小化距离 vs 最小化车辆数 vs 均衡各路线 |
| 医学 | 最大化疗效 vs 最小化副作用 |

### 5.3 优点

- **全面性**: 同时考虑多个决策维度
- **灵活性**: 决策者可事后从Pareto前沿选择偏好解
- **客观性**: 不预设目标间的重要程度（权重法除外）

### 5.4 缺点

- **解的数量多**: 完整Pareto前沿可能包含大量解
- **计算成本高**: 特别是进化算法需要大量函数评估
- **展示困难**: 超过3个目标时Pareto前沿难以可视化
- **无"最优"**: 实际只有一个"解集"，最终选择仍需主观判断

```python
"""
多目标优化求解示例
"""
import numpy as np
from scipy.optimize import minimize
import matplotlib
matplotlib.use('Agg')
import matplotlib.pyplot as plt


# ==================== 示例1: 权重法 ====================
def weighted_sum_method(f_list, x0, bounds, constraints, n_points=20):
    """
    权重法: 均匀采样权重, 求解单目标优化问题
    """
    n_obj = len(f_list)
    pareto_solutions = []
    pareto_objectives = []
    
    # 生成均匀分布的权重
    if n_obj == 2:
        w1_vals = np.linspace(0.01, 0.99, n_points)
        weight_pairs = [(w, 1-w) for w in w1_vals]
    elif n_obj == 3:
        # 三个目标: 在单纯形上采样
        w1_vals = np.linspace(0.01, 0.98, n_points)
        weight_triplets = []
        for w1 in w1_vals:
            for w2 in np.linspace(0.01, 0.99 - w1, n_points // 2):
                if w1 + w2 < 1:
                    weight_triplets.append((w1, w2, 1-w1-w2))
        weight_pairs = weight_triplets
    else:
        weight_pairs = []
    
    for weights in weight_pairs[:n_points]:
        def weighted_obj(x):
            return sum(w * f(x) for w, f in zip(weights, f_list))
        
        result = minimize(weighted_obj, x0, method='SLSQP',
                          bounds=bounds)
        if result.success:
            obj_vals = [f(result.x) for f in f_list]
            # 非支配选择
            pareto_solutions.append(result.x.copy())
            pareto_objectives.append(obj_vals)
    
    return np.array(pareto_solutions), np.array(pareto_objectives)


# ==================== 示例2: ε-约束法 ====================
def epsilon_constraint_method(f_list, x0, bounds, epsilons, constraints=None):
    """
    ε-约束法: 优化第1个目标, 其余目标作为ε约束
    constraints: 已有的等式/不等式约束
    """
    n_obj = len(f_list)
    pareto_solutions = []
    pareto_objectives = []
    
    for eps_vec in epsilons:
        cons = list(constraints) if constraints else []
        # 添加 ε-约束
        for j in range(1, n_obj):
            eps_j = eps_vec[j-1]
            cons.append({'type': 'ineq',
                          'fun': lambda x, j=j, eps=eps_j: eps - f_list[j](x)})
        
        result = minimize(f_list[0], x0, method='SLSQP',
                          bounds=bounds, constraints=cons)
        if result.success:
            obj_vals = [f(result.x) for f in f_list]
            pareto_solutions.append(result.x.copy())
            pareto_objectives.append(obj_vals)
    
    return np.array(pareto_solutions), np.array(pareto_objectives)


# ==================== 示例3: Pareto非支配排序 ====================
def is_pareto_dominated(a, b):
    """返回True如果a被b支配 (都是最小化)"""
    return all(b_i <= a_i for b_i, a_i in zip(b, a)) and \
           any(b_i < a_i for b_i, a_i in zip(b, a))


def pareto_front(objectives):
    """从目标值矩阵中提取Pareto前沿"""
    n = len(objectives)
    dominated = [False] * n
    for i in range(n):
        for j in range(n):
            if i == j:
                continue
            if is_pareto_dominated(objectives[i], objectives[j]):
                dominated[i] = True
                break
    return [i for i in range(n) if not dominated[i]]


def nsga2_fast_non_dominated_sort(objectives):
    """
    NSGA-II的快速非支配排序
    输入: objectives shape = (N, M), M为目标数
    返回: fronts = [[索引...], ...] 按非支配层级排序
    """
    N = len(objectives)
    M = objectives.shape[1]
    
    # 对每个个体, 统计支配它的个体数 n_p, 和它支配的个体集合 S_p
    n_p = np.zeros(N, dtype=int)
    S_p = [[] for _ in range(N)]
    
    for p in range(N):
        for q in range(N):
            if p == q:
                continue
            if is_pareto_dominated(objectives[q], objectives[p]):
                # p 支配 q
                S_p[p].append(q)
            elif is_pareto_dominated(objectives[p], objectives[q]):
                # q 支配 p
                n_p[p] += 1
        if n_p[p] == 0:
            S_p[p] = S_p[p]
    
    fronts = []
    current_front = [i for i in range(N) if n_p[i] == 0]
    
    while current_front:
        fronts.append(current_front[:])
        next_front = []
        for p in current_front:
            for q in S_p[p]:
                n_p[q] -= 1
                if n_p[q] == 0:
                    next_front.append(q)
        current_front = next_front
    
    return fronts
```

## 6. 图论与网络优化 (Graph & Network Optimization)

### 6.1 数学原理

图论与网络优化研究的是以图（节点+边）为模型的优化问题。图 $G = (V, E)$ 由顶点集 $V$ 和边集 $E$ 组成。

**核心问题家族**:

**1. 最短路径问题**:

- **Dijkstra算法**: 非负权图单源最短路, 贪心策略, $O(|E| + |V|\log|V|)$
- **Bellman-Ford**: 可处理负权边, 检测负环, $O(|V|\cdot|E|)$
- **Floyd-Warshall**: 所有节点对最短路, DP, $O(|V|^3)$
- **A\*算法**: 启发式搜索, 用于有先验知识的场景

**2. 最小生成树 (MST)**:

- **Prim算法**: 从一点开始扩展, $O(|E|\log|V|)$
- **Kruskal算法**: 按边权排序, 用并查集避免环, $O(|E|\log|E|)$

**3. 网络流 (Network Flow)**:

- **最大流问题**: 在容量网络中使源到汇的流量最大
  - 增广路算法: Ford-Fulkerson $O(|E|\cdot f_{\max})$
  - Edmonds-Karp (BFS选最短增广路): $O(|V|\cdot|E|^2)$
  - Dinic算法: $O(|V|^2\cdot|E|)$

- **最小费用最大流**: 在保证最大流的前提下最小化费用
  - 连续最短路 (Successive Shortest Path)
  - 基于SPFA的费用流

- **最大流—最小割定理**: 网络中最大流的值等于最小割的容量
  $$\max\text{-flow} = \min\text{-cut}$$

**4. 二分图匹配**:

- 匈牙利算法: $O(|V|\cdot|E|)$
- 可转化为网络流问题

**5. 中国邮递员问题 (CPP) / 旅行商问题 (TSP)**:

- CPP（经过每条边至少一次）：欧拉回路+最小权匹配
- TSP（经过每个节点恰好一次）：NP-hard

|------|----------|------|
| 路径规划 | 最短路径 | 导航、AGV调度、机器人路径 |
| 网络设计 | MST | 通信网络布线、供电线路规划 |
| 交通流分配 | 网络流 | 城市交通拥堵分析、紧急疏散 |

| 供应链 | 最小费用流 | 多级配送网络优化 |
| 连通性分析 | 割/连通度 | 网络脆弱性评估、道路中断分析 |
| 中心性分析 | PageRank/中心度 | 社交网络影响力、交通枢纽识别 |

### 6.3 优点

- **理论基础扎实**: 图论有数百年的数学积累
- **算法高效**: 多项式时间算法覆盖了大多数经典问题
- **可视化直观**: 网络/图结构易于理解和展示
- **建模灵活**: 大量实际问题可抽象为图模型

### 6.4 缺点

- **规模限制**: 超大规模图（亿级节点）需要分布式算法
- **动态性**: 时变网络（动态图）的优化算法更复杂
- **NP-hard问题**: TSP等仍需要启发式求解
- **多目标**: 同时优化多条路径或多商品流的扩展困难

```python
"""
图论与网络优化 — 经典算法实现
"""
import numpy as np
import heapq
from collections import deque, defaultdict


# ==================== 1. Dijkstra最短路径 ====================
def dijkstra(graph, start):
    """
    Dijkstra单源最短路径 O((V+E)logV)
    graph: 邻接表 {u: [(v1, w1), (v2, w2), ...]}
    """
    INF = float('inf')
    dist = {node: INF for node in graph}
    prev = {node: None for node in graph}
    dist[start] = 0
    pq = [(0, start)]
    visited = set()
    
    while pq:
        d, u = heapq.heappop(pq)
        if u in visited:
            continue
        visited.add(u)
        
        for v, w in graph.get(u, []):
            if dist[v] > d + w:
                dist[v] = d + w
                prev[v] = u
                heapq.heappush(pq, (dist[v], v))
    
    return dist, prev


def reconstruct_path(prev, start, end):
    """从prev数组重建路径"""
    path = []
    cur = end
    while cur is not None:
        path.append(cur)
        cur = prev[cur]
    path.reverse()
    return path if path[0] == start else []


# ==================== 2. Kruskal最小生成树 ====================
class UnionFind:
    def __init__(self, n):
        self.parent = list(range(n))
        self.rank = [0] * n
    
    def find(self, x):
        if self.parent[x] != x:
            self.parent[x] = self.find(self.parent[x])  # 路径压缩
        return self.parent[x]
    
    def union(self, x, y):
        px, py = self.find(x), self.find(y)
        if px == py:
            return False
        if self.rank[px] < self.rank[py]:
            self.parent[px] = py
        elif self.rank[px] > self.rank[py]:
            self.parent[py] = px
        else:
            self.parent[py] = px
            self.rank[px] += 1
        return True


def kruskal(n, edges):
    """
    Kruskal最小生成树 O(ElogE)
    n: 节点数 (0..n-1)
    edges: [(u, v, weight), ...]
    返回: (总权重, 生成树边列表)
    """
    edges_sorted = sorted(edges, key=lambda e: e[2])
    uf = UnionFind(n)
    
    mst_weight = 0
    mst_edges = []
    
    for u, v, w in edges_sorted:
        if uf.union(u, v):
            mst_edges.append((u, v, w))
            mst_weight += w
            if len(mst_edges) == n - 1:
                break
    
    return mst_weight, mst_edges


# ==================== 3. Dinic最大流 ====================
class Dinic:
    """
    Dinic最大流算法 O(V^2 * E)
    """
    def __init__(self, n):
        self.n = n
        self.graph = [[] for _ in range(n)]
    
    def add_edge(self, u, v, cap):
        """添加有向边 u→v 容量cap"""
        self.graph[u].append([v, cap, len(self.graph[v])])
        self.graph[v].append([u, 0, len(self.graph[u]) - 1])
    
    def bfs(self, s, t, level):
        """BFS构建分层图"""
        for i in range(self.n):
            level[i] = -1
        level[s] = 0
        q = deque([s])
        while q:
            u = q.popleft()
            for v, cap, rev in self.graph[u]:
                if cap > 0 and level[v] < 0:
                    level[v] = level[u] + 1
                    q.append(v)
        return level[t] >= 0
    
    def dfs(self, u, t, f, level, it):
        """DFS寻找阻塞流"""
        if u == t:
            return f
        for i in range(it[u], len(self.graph[u])):
            it[u] = i
            v, cap, rev = self.graph[u][i]
            if cap > 0 and level[u] < level[v]:
                ret = self.dfs(v, t, min(f, cap), level, it)
                if ret > 0:
                    # 更新残留网络
                    self.graph[u][i][1] -= ret
                    self.graph[v][rev][1] += ret
                    return ret
        return 0
    
    def max_flow(self, s, t):
        """计算s到t的最大流"""
        flow = 0
        INF = float('inf')
        level = [0] * self.n
        while self.bfs(s, t, level):
            it = [0] * self.n
            while True:
                f = self.dfs(s, t, INF, level, it)
                if f == 0:
                    break
                flow += f
        return flow


# ==================== 4. 最小费用最大流 (SPFA) ====================
def min_cost_max_flow(n, graph_cap, graph_cost, s, t, maxf):
    """
    最小费用最大流 (Successive Shortest Path with SPFA)
    返回: (实际流量, 总费用)
    """
    INF = float('inf')
    n_nodes = n
    flow = 0
    cost = 0
    
    # 构建邻接表和残留网络
    # 每条边: [to, cap, cost, rev_idx]
    g = [[] for _ in range(n_nodes)]
    
    def add_edge(fr, to, cap, cst):
        g[fr].append([to, cap, cst, len(g[to])])
        g[to].append([fr, 0, -cst, len(g[fr]) - 1])
    
    for u in range(n_nodes):
        for v, cap in graph_cap.get(u, []):
            cst = graph_cost.get((u, v), 0)
            add_edge(u, v, cap, cst)
    
    while flow < maxf:
        dist = [INF] * n_nodes
        prevv = [-1] * n_nodes
        preve = [-1] * n_nodes
        in_queue = [False] * n_nodes
        
        dist[s] = 0
        q = deque([s])
        in_queue[s] = True
        
        while q:
            u = q.popleft()
            in_queue[u] = False
            for i, (v, cap, cst, rev) in enumerate(g[u]):
                if cap > 0 and dist[v] > dist[u] + cst:
                    dist[v] = dist[u] + cst
                    prevv[v] = u
                    preve[v] = i
                    if not in_queue[v]:
                        q.append(v)
                        in_queue[v] = True
        
        if dist[t] == INF:
            break
        
        # 找最小残量
        d = maxf - flow
        v = t
        while v != s:
            u = prevv[v]
            e = preve[v]
            d = min(d, g[u][e][1])
            v = u
        
        flow += d
        cost += d * dist[t]
        
        # 更新残留网络
        v = t
        while v != s:
            u = prevv[v]
            e = preve[v]
            g[u][e][1] -= d
            g[v][g[u][e][3]][1] += d
            v = u
    
    return flow, cost


# ==================== 5. 匈牙利算法 (二分图最大匹配) ====================
def hungarian(cost_matrix):
    """
    匈牙利算法求解指派问题 O(n^3)
    cost_matrix: n×n 成本矩阵
    返回: 最小总成本, 匹配方案 [(i, j), ...]
    """
    from scipy.optimize import linear_sum_assignment
    
    cost = np.array(cost_matrix)
    row_ind, col_ind = linear_sum_assignment(cost)
    total_cost = cost[row_ind, col_ind].sum()
    assignment = list(zip(row_ind, col_ind))
    
    return total_cost, assignment
```

## 7. 遗传算法 (Genetic Algorithm)

### 7.1 数学原理

遗传算法(GA)是受达尔文进化论启发的随机搜索算法，通过模拟自然选择和遗传机制寻找最优解。

**核心概念对应关系**:
| 生物学 | GA |
|--------|-----|
| 种群 (Population) | 一组候选解的集合 |
| 染色体 (Chromosome) | 一个解的编码 (个体) |
| 基因 (Gene) | 解的每个分量 |
| 适应度 (Fitness) | 目标函数值（越大越好） |
| 选择 (Selection) | 按适应度比例选优秀个体进入下一代 |
| 交叉 (Crossover) | 两个父代基因素材重组产生子代 |
| 变异 (Mutation) | 以较小概率随机改变某些基因 |
| 进化 (Evolution) | 多代迭代，种群适应度逐渐提高 |

**算法流程**:

```
1. 初始化: 随机生成规模为 N 的种群 P(0), t=0
2. 评估: 计算 P(t) 中每个个体的适应度
3. while 终止条件不满足:
   a. 选择 (Selection): 从 P(t) 中选 N 个个体 (有放回, 优胜劣汰)
   b. 交叉 (Crossover): 以概率 p_c 对选出的个体两两交叉
   c. 变异 (Mutation):   以概率 p_m 对新个体每个基因变异
   d. P(t+1) = 新种群
   e. t = t+1
   f. 评估 P(t) 的适应度
4. 输出最优个体
```

**选择算子**:
- **轮盘赌选择 (Roulette Wheel)**: 选择概率 $\propto$ 适应度
  $$p_i = \frac{f_i}{\sum_{j=1}^{N} f_j}$$
- **锦标赛选择 (Tournament)**: 随机选k个个体，其中最好的入选
- **精英保留 (Elitism)**: 直接将最优的几个个体保留到下一代

**交叉算子**:
- **单点交叉**: 在随机位置切断, 交换尾部
- **两点交叉**: 交换中间片段
- **均匀交叉**: 每个基因独立地从父代之一继承
- **SBX (Simulated Binary Crossover)**: 实值编码的模拟二进制交叉
- **PMX (Partially Mapped Crossover)**: 排列编码（适合TSP）

**变异算子**:
- **位翻转变异**: 二进制编码的 bit flip
- **高斯变异**: 实值编码 $x_i' = x_i + \mathcal{N}(0, \sigma^2)$
- **交换变异**: 排列编码中交换两个位置

**模式定理 (Schema Theorem) / 积木块假设 (Building Block Hypothesis)**:
- 短定义距、低阶、高于平均适应度的模式(积木块)在进化中被指数级采样
- GA通过不断组合这些积木块来构建更优解

### 7.2 适用场景

| 场景 | 示例 |
|------|------|
| 组合优化 | TSP、VRP、调度、排课 |
| 函数优化 | 不可微/多峰/黑箱目标函数的最优化 |
| 参数优化 | PID控制器参数、神经网络超参 |
| 特征选择 | 从高维特征中选择最优子集 |
| 结构设计 | 桁架拓扑优化、天线设计 |
| 规则发现 | 交易策略规则进化 |
| 多目标 | NSGA-II/MOEA/D 求Pareto前沿 |

### 7.3 优点

- **全局搜索能力强**: 种群并行搜索,不易陷入局部最优
- **不需要导数信息**: 适合黑箱函数、不可微函数
- **灵活**: 可以很容易地处理混合变量、复杂约束
- **天然并行**: 种群评估可并行化

### 7.4 缺点

- **收敛速度慢**: 特别是接近最优解时
- **早熟收敛 (Premature Convergence)**: 种群多样性丧失, 全体收敛到某个局部最优
- **参数敏感**: 种群大小、交叉率、变异率等超参对效果影响大
- **理论上弱**: 没有严格的收敛性证明（相比凸优化）
- **计算量大**: 每代需要评估N次目标函数

### 7.5 Python代码实现

```python
"""
遗传算法 (Genetic Algorithm) 完整实现
"""
import numpy as np
import matplotlib
matplotlib.use('Agg')
import matplotlib.pyplot as plt


class GeneticAlgorithm:
    """通用的遗传算法框架"""
    
    def __init__(self, fitness_func, bounds, pop_size=100,
                 crossover_rate=0.8, mutation_rate=0.1,
                 elite_ratio=0.05, max_generations=200,
                 verbose=True):
        """
        fitness_func: 适应度函数 (最大化), 接收一个解向量
        bounds: [(min, max), ...] 每个变量的范围
        """
        self.fitness_func = fitness_func
        self.bounds = np.array(bounds)
        self.dim = len(bounds)
        self.pop_size = pop_size
        self.crossover_rate = crossover_rate
        self.mutation_rate = mutation_rate
        self.elite_size = max(1, int(pop_size * elite_ratio))
        self.max_generations = max_generations
        self.verbose = verbose
        
        self.history = []  # 记录每代的最优适应度
    
    def initialize(self):
        """随机初始化种群"""
        pop = np.random.rand(self.pop_size, self.dim)
        for i in range(self.dim):
            lo, hi = self.bounds[i]
            pop[:, i] = lo + pop[:, i] * (hi - lo)
        return pop
    
    def evaluate(self, pop):
        """评估种群适应度 (向量化)"""
        return np.array([self.fitness_func(ind) for ind in pop])
    
    def select_tournament(self, pop, fitness, k=3):
        """锦标赛选择"""
        n = len(pop)
        selected = []
        for _ in range(self.pop_size - self.elite_size):
            idx = np.random.choice(n, k, replace=False)
            best_idx = idx[np.argmax(fitness[idx])]
            selected.append(pop[best_idx].copy())
        return np.array(selected)
    
    def select_roulette(self, pop, fitness):
        """轮盘赌选择 (适应度需非负)"""
        # 适应度缩放 (确保非负)
        f_min = fitness.min()
        if f_min < 0:
            fitness_adj = fitness - f_min + 1e-10
        else:
            fitness_adj = fitness + 1e-10
        
        probs = fitness_adj / fitness_adj.sum()
        n_select = self.pop_size - self.elite_size
        idx = np.random.choice(len(pop), n_select, p=probs, replace=True)
        return pop[idx].copy()
    
    def crossover_sbx(self, parent1, parent2, eta=15):
        """
        SBX (Simulated Binary Crossover) 用于实值编码
        eta越大, 子代越接近父代
        """
        child1, child2 = parent1.copy(), parent2.copy()
        
        for i in range(self.dim):
            if np.random.rand() < self.crossover_rate:
                lo, hi = self.bounds[i]
                if abs(parent1[i] - parent2[i]) > 1e-14:
                    if parent1[i] < parent2[i]:
                        y1, y2 = parent1[i], parent2[i]
                    else:
                        y1, y2 = parent2[i], parent1[i]
                    
                    # SBX公式
                    u = np.random.rand()
                    beta = 1 + 2 * (y1 - lo) / (y2 - y1 + 1e-14)
                    alpha = 2 - beta ** (-(eta + 1))
                    
                    if u <= 1.0 / alpha:
                        beta_q = (u * alpha) ** (1.0 / (eta + 1))
                    else:
                        beta_q = (1.0 / (2.0 - u * alpha)) ** (1.0 / (eta + 1))
                    
                    c1 = 0.5 * ((y1 + y2) - beta_q * (y2 - y1))
                    
                    beta = 1 + 2 * (hi - y2) / (y2 - y1 + 1e-14)
                    alpha = 2 - beta ** (-(eta + 1))
                    
                    if u <= 1.0 / alpha:
                        beta_q = (u * alpha) ** (1.0 / (eta + 1))
                    else:
                        beta_q = (1.0 / (2.0 - u * alpha)) ** (1.0 / (eta + 1))
                    
                    c2 = 0.5 * ((y1 + y2) + beta_q * (y2 - y1))
                    
                    # 边界裁剪
                    child1[i] = np.clip(c1, lo, hi)
                    child2[i] = np.clip(c2, lo, hi)
                else:
                    child1[i] = parent1[i]
                    child2[i] = parent2[i]
            else:
                child1[i] = parent1[i]
                child2[i] = parent2[i]
        
        return child1, child2
    
    def crossover_blend(self, parent1, parent2, alpha=0.5):
        """
        BLX-α 混合交叉
        """
        child1, child2 = parent1.copy(), parent2.copy()
        
        for i in range(self.dim):
            if np.random.rand() < self.crossover_rate:
                lo, hi = self.bounds[i]
                d = abs(parent1[i] - parent2[i])
                low = max(lo, min(parent1[i], parent2[i]) - alpha * d)
                high = min(hi, max(parent1[i], parent2[i]) + alpha * d)
                child1[i] = np.random.uniform(low, high)
                child2[i] = np.random.uniform(low, high)
        
        return child1, child2
    
    def mutate_gaussian(self, individual, generation):
        """
        高斯变异 (变异幅度随代数衰减)
        """
        sigma = 0.1 * (self.bounds[:, 1] - self.bounds[:, 0])
        sigma *= (1 - generation / self.max_generations)  # 逐步减小
        
        for i in range(self.dim):
            if np.random.rand() < self.mutation_rate:
                individual[i] += np.random.normal(0, sigma[i])
                individual[i] = np.clip(individual[i],
                                         self.bounds[i][0],
                                         self.bounds[i][1])
        return individual
    
    def mutate_polynomial(self, individual, eta_m=20):
        """
        多项式变异 (Polynomial Mutation)
        """
        for i in range(self.dim):
            if np.random.rand() < self.mutation_rate:
                lo, hi = self.bounds[i]
                u = np.random.rand()
                delta = min(individual[i] - lo, hi - individual[i]) / (hi - lo + 1e-14)
                
                if u <= 0.5:
                    delta_q = (2 * u + (1 - 2 * u) *
                               (1 - delta) ** (eta_m + 1)) ** (1/(eta_m + 1)) - 1
                else:
                    delta_q = 1 - (2 * (1 - u) + 2 * (u - 0.5) *
                                   (1 - delta) ** (eta_m + 1)) ** (1/(eta_m + 1))
                
                individual[i] += delta_q * (hi - lo)
                individual[i] = np.clip(individual[i], lo, hi)
        return individual
    
    def run(self):
        """运行遗传算法"""
        pop = self.initialize()
        fitness = self.evaluate(pop)
        
        # 精英保留
        elite_idx = np.argsort(fitness)[-self.elite_size:]
        elites = pop[elite_idx].copy()
        
        self.history = []
        
        for gen in range(self.max_generations):
            # 选择
            offspring = self.select_tournament(pop, fitness)
            # offspring = self.select_roulette(pop, fitness)
            
            # 交叉
            for i in range(0, len(offspring) - 1, 2):
                offspring[i], offspring[i+1] = self.crossover_sbx(
                    offspring[i], offspring[i+1]
                )
            
            # 变异
            for i in range(len(offspring)):
                offspring[i] = self.mutate_gaussian(offspring[i], gen)
                # offspring[i] = self.mutate_polynomial(offspring[i])
            
            # 精英保留
            pop = np.vstack([offspring, elites])
            fitness = self.evaluate(pop)
            
            # 更新精英
            elite_idx = np.argsort(fitness)[-self.elite_size:]
            elites = pop[elite_idx].copy()
            
            best_fitness = fitness.max()
            self.history.append(best_fitness)
            
            if self.verbose and (gen % 20 == 0 or gen == self.max_generations - 1):
                print(f"Generation {gen:4d}: Best Fitness = {best_fitness:.6f}")
        
        best_idx = np.argmax(fitness)
        return pop[best_idx], fitness[best_idx]

def ackley(x):
    """Ackley函数: 多峰、大量局部最优
    全局最小值 f(0,...,0) = 0
    """
    n = len(x)
    a, b, c = 20, 0.2, 2 * np.pi
    sum1 = np.sum(x**2)
    sum2 = np.sum(np.cos(c * x))
    return -(a * np.exp(-b * np.sqrt(sum1 / n)) +
             np.exp(sum2 / n) - a - np.exp(1))


def rastrigin(x):
    """Rastrigin函数: 大量规则分布的局部最优
    全局最小值 f(0,...,0) = 0
    """
    n = len(x)
    A = 10
    return -(A * n + np.sum(x**2 - A * np.cos(2 * np.pi * x)))
```

## 8. 模拟退火 (Simulated Annealing)

### 8.1 数学原理

模拟退火(SA)源于固体退火的物理过程：将固体加热至高温后缓慢冷却，原子在高温时自由运动，缓慢降温时在每一温度都达到热平衡，最终形成能量最低的晶格结构。

**Metropolis准则**:

在当前状态 $i$ (能量 $E_i$) 下，随机扰动产生新状态 $j$ (能量 $E_j$):

- 若 $E_j < E_i$ (更优), **总是接受**新状态
- 若 $E_j \ge E_i$ (更劣), 以概率 $P = \exp\left(-\frac{E_j - E_i}{kT}\right)$ 接受新状态

其中 $T$ 为当前温度, $k$ 为Boltzmann常数（通常取 $k=1$）。

**温度调度 (Cooling Schedule)**: 控制收敛速度和搜索精度最关键的因素。

- **指数降温**: $T_{k+1} = \alpha \cdot T_k, \quad \alpha \in (0.9, 0.99)$
- **对数降温**: $T_k = \frac{T_0}{\log(1+k)}$
- **线性降温**: $T_{k+1} = T_k - \Delta T$
- **快速SA**: $T_k = \frac{T_0}{1+k}$

**算法伪代码**:
```
1. 初始化: x = 随机解, T = T_max, k = 0
2. while T > T_min:
      for i = 1 to L_k (Markov链长度):
          y = neighbor(x)  # 邻域扰动
          ΔE = f(y) - f(x)
          if ΔE < 0 or random() < exp(-ΔE / T):
              x = y
      T = cooling(T, k)
      k += 1
3. 输出 x
```

**收敛性理论**:
- 在无限慢降温 (即 $T_k \ge \frac{C}{\log(1+k)}$) 下，以概率1收敛到全局最优
- 实际操作中不可能无限慢，需要权衡速度和精度

**关键超参数**:
- $T_0$ (初始温度): 足够高以使接受概率接近1
- $\alpha$ (降温系数): 0.85-0.99, 越接近1搜索越充分
- $L_k$ (Markov链长度): 每个温度下的迭代次数
- $T_{\min}$ (终温): 低到不再接受劣解

### 8.2 适用场景

| 场景 | 示例 |
|------|------|
| TSP/VRP | 大规模旅行商/车辆路径问题 |
| VLSI设计 | 芯片布局、布线优化 |
| 调度问题 | Job-shop调度、考试排表 |
| 图像处理 | 图像分割、图像恢复 |
| 机器学习 | 特征选择、聚类 |
| 结构优化 | 桁架设计、化工流程优化 |
| 组合优化 | 几乎所有NP-hard的组合优化问题 |

### 8.3 优点

- **可跳出局部最优**: Metropolis准则允许以一定概率接受劣解
- **理论上收敛**: 在适当降温速度下保证收敛到全局最优
- **简单**: 算法结构简单, 易于实现和并行化
- **通用**: 对问题的数学性质（可微性、连续性）要求低
- **约束处理灵活**: 可以通过罚函数或拒绝策略处理约束

### 8.4 缺点

- **速度慢**: 充分搜索需要大量评估
- **参数调节困难**: 初始温度、降温率、链长度等依赖经验
- **可能不收敛**: 降温太快则停留在局部最优
- **重复运行差异**: 随机性强, 不同运行可能得到不同解

```python
"""
模拟退火算法 (Simulated Annealing) 实现
"""
import numpy as np
import matplotlib
matplotlib.use('Agg')
import matplotlib.pyplot as plt


class SimulatedAnnealing:
    """通用模拟退火框架"""
    
    def __init__(self, objective_func, bounds, neighbor_func=None,
                 T_max=1000, T_min=1e-8, alpha=0.95,
                 steps_per_temp=100, max_iter=50000,
                 verbose=True):
        """
        objective_func: 目标函数 (最小化)
        bounds: [(min, max), ...]
        neighbor_func: 邻域函数, 默认高斯扰动
        """
        self.objective_func = objective_func
        self.bounds = np.array(bounds)
        self.dim = len(bounds)
        self.T_max = T_max
        self.T_min = T_min
        self.alpha = alpha
        self.steps_per_temp = steps_per_temp
        self.max_iter = max_iter
        self.verbose = verbose
        
        # 默认邻域函数
        self.neighbor_func = neighbor_func or self._default_neighbor
        
        self.history = []  # 记录最优值变化
    
    def _default_neighbor(self, x, T):
        """
        默认邻域函数: 高斯扰动
        扰动幅度正比于温度和变量范围
        「温度高时大步探索，温度低时小步精调」
        """
        sigma = 0.1 * (self.bounds[:, 1] - self.bounds[:, 0])
        sigma *= min(1.0, T / self.T_max)  # 温度低时扰动减小
        
        y = x.copy()
        for i in range(self.dim):
            if np.random.rand() < 0.5:  # 随机选择哪些维度扰动
                y[i] += np.random.normal(0, sigma[i])
                y[i] = np.clip(y[i], self.bounds[i][0], self.bounds[i][1])
        return y
    
    def run(self):
        """运行模拟退火"""
        # 初始解
        x = np.array([np.random.uniform(lo, hi)
                       for lo, hi in self.bounds])
        x_best = x.copy()
        f_x = self.objective_func(x)
        f_best = f_x
        
        T = self.T_max
        self.history = [(0, f_best)]
        
        iter_count = 0
        accepted_total = 0
        improved_total = 0
        
        while T > self.T_min and iter_count < self.max_iter:
            accepted_this_temp = 0
            
            for _ in range(self.steps_per_temp):
                y = self.neighbor_func(x, T)
                f_y = self.objective_func(y)
                
                delta_E = f_y - f_x
                
                if delta_E < 0 or np.random.rand() < np.exp(-delta_E / T):
                    x = y
                    f_x = f_y
                    accepted_this_temp += 1
                    
                    if f_x < f_best:
                        x_best = x.copy()
                        f_best = f_x
                        improved_total += 1
                
                iter_count += 1
                if iter_count >= self.max_iter:
                    break
            
            accepted_total += accepted_this_temp
            
            # 降温
            T *= self.alpha
            
            # 记录
            self.history.append((iter_count, f_best))
            
            if self.verbose and len(self.history) % 50 == 0:
                accept_rate = accepted_this_temp / self.steps_per_temp
                print(f"Iter {iter_count:6d}: T={T:.4e}, "
                      f"Best={f_best:.6f}, Accept Rate={accept_rate:.2%}")
        
        if self.verbose:
            print(f"\n总评估次数: {iter_count}")
            print(f"总接受率: {accepted_total/max(1, iter_count):.2%}")
            print(f"改进次数: {improved_total}")
        
        return x_best, f_best


# ==================== 离散邻域 TSP示例 ====================
def tsp_simulated_annealing(dist_matrix, T_max=1000, alpha=0.995,
                             steps_per_temp=200, verbose=True):
    """
    用SA求解旅行商问题(TSP)
    """
    n = dist_matrix.shape[0]
    
    def tour_length(tour):
        """计算回路总长度"""
        total = 0
        for i in range(n):
            total += dist_matrix[tour[i]][tour[(i+1) % n]]
        return total
    
    def neighbor_2opt(tour):
        """2-opt邻域: 反转子序列"""
        new_tour = tour.copy()
        i, j = sorted(np.random.choice(n, 2, replace=False))
        new_tour[i:j+1] = new_tour[i:j+1][::-1]
        return new_tour
    
    # 初始解: 随机排列
    tour = np.random.permutation(n)
    tour_best = tour.copy()
    length = tour_length(tour)
    length_best = length
    
    T = T_max
    history = [length_best]
    
    iter_count = 0
    
    while T > 1e-3:
        for _ in range(steps_per_temp):
            new_tour = neighbor_2opt(tour)
            new_length = tour_length(new_tour)
            
            delta = new_length - length
            
            if delta < 0 or np.random.rand() < np.exp(-delta / T):
                tour = new_tour
                length = new_length
                
                if length < length_best:
                    tour_best = tour.copy()
                    length_best = length
            
            iter_count += 1
        
        T *= alpha
        history.append(length_best)
        
        if verbose and len(history) % 100 == 0:
            print(f"  T={T:.3f}, Best={length_best:.2f}")
    
    return tour_best, length_best, history


# ==================== 自适应降温策略 ====================
def adaptive_cooling(T, accept_rate, target_rate=0.5, alpha_min=0.8, alpha_max=0.99):
    """
    自适应温度调度: 根据接受率调节降温速度
    接受率太高 → 降温慢 (多探索)
    接受率太低 → 降温快 (收敛)
    """
    if accept_rate > target_rate:
        alpha = alpha_min + (alpha_max - alpha_min) * (1 - accept_rate)
    else:
        alpha = alpha_max
    return alpha
```

## 9. 粒子群优化 (Particle Swarm Optimization)

### 9.1 数学原理

粒子群优化(PSO)是Kennedy和Eberhart于1995年提出的群智能算法，模拟鸟群/鱼群的社交行为。每个粒子代表一个候选解，通过个体经验和群体信息调整搜索方向。

**速度和位置更新**:

$$v_i^{t+1} = w \cdot v_i^t + c_1 r_1 \cdot (\mathbf{pbest}_i - \mathbf{x}_i^t) + c_2 r_2 \cdot (\mathbf{gbest} - \mathbf{x}_i^t)$$

$$\mathbf{x}_i^{t+1} = \mathbf{x}_i^t + v_i^{t+1}$$

其中:
- $\mathbf{x}_i^t$: 粒子 $i$ 在时刻 $t$ 的位置
- $v_i^t$: 粒子 $i$ 在时刻 $t$ 的速度
- $\mathbf{pbest}_i$: 粒子 $i$ 的历史最优位置（个体认知）
- $\mathbf{gbest}$: 整个群体的历史最优位置（社会认知）
- $w$: **惯性权重** — 平衡探索与开发
- $c_1$: **认知加速系数** — 向个体最优学习的权重
- $c_2$: **社会加速系数** — 向群体最优学习的权重
- $r_1, r_2 \sim U(0,1)$: 随机数, 增加搜索多样性

**三部分含义**:
$$\text{新速度} = \underbrace{w \cdot v_i^t}_{\text{惯性项}} + \underbrace{c_1 r_1 (\mathbf{pbest}_i - \mathbf{x}_i)}_{\text{个人认知项}} + \underbrace{c_2 r_2 (\mathbf{gbest} - \mathbf{x}_i)}_{\text{社会认知项}}$$

**参数分析与调优**:

| 参数 | 推荐值 | 含义 |
|------|--------|------|
| $w$ | 0.4-0.9 | 越大越全局搜索, 越小越局部精调 |
| $c_1, c_2$ | 1.5-2.0 | 通常$c_1 = c_2 = 2.0$ |
| 种群大小 | 20-50 | 越小越快, 越大越可靠 |
| 速度限制 | $[-v_{\max}, v_{\max}]$ | 防止粒子飞出搜索空间 |

**常用改进策略**:

1. **线性递减惯性权重 (LDIW)**:
   $$w(t) = w_{\max} - (w_{\max} - w_{\min}) \cdot \frac{t}{T_{\max}}$$
   前期大权重强探索, 后期小权重强开发

2. **收缩因子 (Constriction Factor, Clerc & Kennedy 2002)**:
   $$v_i^{t+1} = \chi \left[ v_i^t + c_1 r_1 (\mathbf{pbest}_i - \mathbf{x}_i) + c_2 r_2 (\mathbf{gbest} - \mathbf{x}_i) \right]$$
   $$\chi = \frac{2}{|2 - \varphi - \sqrt{\varphi^2 - 4\varphi}|}, \quad \varphi = c_1 + c_2 > 4$$

3. **环形拓扑 (Ring Topology / lbest)**: 粒子只与其邻居比较, 减慢信息传播, 保持多样性

4. **全面学习PSO (CLPSO)**: 每个维度可学习不同粒子的pbest

### 9.2 适用场景

| 场景 | 示例 |
|------|------|
| 连续参数优化 | PID控制器、滤波器参数调优 |
| 神经网络训练 | 权重初始化、超参数搜索 |
| 工程优化 | 结构设计、天线阵列优化 |
| 电力系统 | 经济调度、无功优化 |
| 机器学习 | SVM参数选择、聚类中心优化 |
| 多目标优化 | MOPSO扩展到多目标领域 |

### 9.3 优点

- **收敛速度快**: 信息共享机制使群体快速向最优靠拢
- **实现简单**: 仅需速度和位置更新公式，参数少
- **无梯度要求**: 适合黑箱优化
- **记忆功能**: 个体和群体的历史最优信息被保留

### 9.4 缺点

- **早熟收敛**: gbest模型下信息传播太快, 易全体陷入局部最优
- **参数敏感**: w, c1, c2 的设置显著影响性能
- **离散化困难**: 原始PSO针对连续空间，离散版的性能不如ACO等
- **理论上弱**: 收敛性分析主要在简化条件下
### 9.5 Python代码实现

```python
"""
粒子群优化 (Particle Swarm Optimization) 完整实现
"""
import numpy as np
import matplotlib
matplotlib.use("Agg")
import matplotlib.pyplot as plt


class PSO:
    """标准粒子群优化算法"""

    def __init__(self, objective_func, bounds, num_particles=30,
                 w=0.729, c1=1.49445, c2=1.49445,
                 max_iter=200, topology="global", verbose=True):
        self.objective_func = objective_func
        self.bounds = np.array(bounds)
        self.dim = len(bounds)
        self.num_particles = num_particles
        self.w = w
        self.c1 = c1
        self.c2 = c2
        self.max_iter = max_iter
        self.topology = topology
        self.verbose = verbose
        self.v_max = 0.2 * (self.bounds[:, 1] - self.bounds[:, 0])
        self.history = []

    def initialize(self):
        lo = self.bounds[:, 0]
        hi = self.bounds[:, 1]
        r = hi - lo
        self.positions = lo + np.random.rand(self.num_particles, self.dim) * r
        self.velocities = np.random.uniform(-1, 1,
            (self.num_particles, self.dim)) * self.v_max
        self.fitness = np.array([self.objective_func(p) for p in self.positions])
        self.pbest_pos = self.positions.copy()
        self.pbest_val = self.fitness.copy()
        self.gbest_idx = np.argmin(self.fitness)
        self.gbest_pos = self.positions[self.gbest_idx].copy()
        self.gbest_val = self.fitness[self.gbest_idx]

    def update_velocity_position(self):
        r1 = np.random.rand(self.num_particles, self.dim)
        r2 = np.random.rand(self.num_particles, self.dim)
        if self.topology == "global":
            social_best = self.gbest_pos
        elif self.topology == "ring":
            social_best = self._get_lbest_positions()
        cognitive = self.c1 * r1 * (self.pbest_pos - self.positions)
        social = self.c2 * r2 * (social_best - self.positions)
        self.velocities = self.w * self.velocities + cognitive + social
        self.velocities = np.clip(self.velocities, -self.v_max, self.v_max)
        self.positions += self.velocities
        self.positions = np.clip(self.positions, self.bounds[:, 0], self.bounds[:, 1])

    def _get_lbest_positions(self):
        n = self.num_particles
        lbest_pos = np.zeros_like(self.positions)
        for i in range(n):
            left = (i - 1) % n
            right = (i + 1) % n
            neighbors = [i, left, right]
            best_nb = neighbors[np.argmin([self.pbest_val[j] for j in neighbors])]
            lbest_pos[i] = self.pbest_pos[best_nb].copy()
        return lbest_pos

    def run(self, w_strategy="constant"):
        self.initialize()
        self.history = [self.gbest_val]
        w_start = 0.9
        w_end = 0.4
        for t in range(self.max_iter):
            if w_strategy == "linear_decay":
                self.w = w_start - (w_start - w_end) * t / self.max_iter
            elif w_strategy == "adaptive":
                diversity = np.mean(np.std(self.positions, axis=0))
                max_range = np.mean(self.bounds[:, 1] - self.bounds[:, 0])
                self.w = 0.4 + 0.5 * (diversity / (max_range + 1e-10))
            self.update_velocity_position()
            self.fitness = np.array([self.objective_func(p) for p in self.positions])
            improved = self.fitness < self.pbest_val
            self.pbest_pos[improved] = self.positions[improved]
            self.pbest_val[improved] = self.fitness[improved]
            current_best_idx = np.argmin(self.fitness)
            if self.fitness[current_best_idx] < self.gbest_val:
                self.gbest_idx = current_best_idx
                self.gbest_pos = self.positions[current_best_idx].copy()
                self.gbest_val = self.fitness[current_best_idx]
            self.history.append(self.gbest_val)
            if self.verbose and (t % 20 == 0 or t == self.max_iter - 1):
                print(f"Iter {t:4d}: Best = {self.gbest_val:.8f}, w = {self.w:.4f}")
        return self.gbest_pos, self.gbest_val


class ConstrainedPSO(PSO):
    """处理约束的PSO (罚函数法)"""
    def __init__(self, objective_func, bounds, constraints=None,
                 penalty_factor=1000, **kwargs):
        super().__init__(objective_func, bounds, **kwargs)
        self.constraints = constraints or []
        self.penalty_factor = penalty_factor

    def constrained_fitness(self, x):
        f = self.objective_func(x)
        penalty = 0
        for g in self.constraints:
            violation = max(0, -g(x))
            penalty += self.penalty_factor * violation ** 2
        return f + penalty

    def run(self, w_strategy="constant"):
        self.objective_func_original = self.objective_func
        self.objective_func = self.constrained_fitness
        result = super().run(w_strategy)
        self.objective_func = self.objective_func_original
        return result
```

## 10. 蚁群优化 (Ant Colony Optimization)

### 10.1 数学原理

蚁群优化(ACO)由Dorigo于1992年提出，模拟蚂蚁觅食中通过信息素(pheromone)合作寻找最短路径的行为。蚂蚁在路径上留下信息素, 其他蚂蚁倾向于选择信息素浓度高的路径, 形成正反馈循环。

**核心机制**:

**路径选择概率** (Ant System, AS):

在节点 i 的蚂蚁 k 选择下一节点 j 的概率为:

$$p_{ij}^k = \begin{cases}
\frac{[\tau_{ij}]^\alpha \cdot [\eta_{ij}]^\beta}{\sum_{l \in N_i^k} [\tau_{il}]^\alpha \cdot [\eta_{il}]^\beta}, & j \in N_i^k \\
0, & \text{otherwise}
\end{cases}$$

其中:
- $\tau_{ij}$: 边 (i,j) 上的信息素浓度
- $\eta_{ij}$: 启发式信息 (通常 $\eta_{ij} = 1/d_{ij}$, 即距离的倒数)
- $\alpha$: 信息素重要度因子
- $\beta$: 启发式信息重要度因子
- $N_i^k$: 蚂蚁 k 在节点 i 的可选邻居集合

**信息素更新**:

$$\tau_{ij} \leftarrow (1-\rho) \cdot \tau_{ij} + \sum_{k=1}^{m} \Delta\tau_{ij}^k$$

其中:
- $\rho \in (0, 1)$: 信息素蒸发率
- $\Delta\tau_{ij}^k$: 第 k 只蚂蚁在边 (i,j) 上释放的信息素增量

**ACO变体差异**:

| 变体 | 特点 |
|------|------|
| AS (Ant System) | 原始版本, 收敛慢 |
| ACS (Ant Colony System) | 收敛最快, 应用最广 |
| MMAS (Max-Min AS) | 防止早熟, 性能稳定 |
| RAS (Rank-based AS) | 按排名加权更新 |

### 10.2 适用场景

| 场景 | 示例 |
|------|------|
| TSP | 经典应用, ACO最擅长的领域 |
| VRP (车辆路径) | 带容量/时间窗的配送优化 |
| 二次分配 (QAP) | 工厂选址、电路板元件布局 |
| 调度问题 | Job-shop调度、项目调度 |
| 网络路由 | Ad-hoc网络、通信路由 |
| 集合覆盖 | 设施选址覆盖问题 |

### 10.3 优点

- **正反馈机制**: 优秀解自动加强, 搜索效率高
- **分布式计算**: 各蚂蚁独立构造解, 天然并行
- **鲁棒性强**: 对参数不特别敏感, 容易适配新问题
- **结合局部搜索**: 可与2-opt等局部搜索算子结合成混合算法

### 10.4 缺点

- **收敛速度不确定**: 可能早熟(信息素饱和)或慢收敛
- **参数多**: alpha, beta, rho, q0, m 等需要调试
- **理论分析少**: 收敛性证明不如SA充分
- **主要适用于组合优化**: 连续优化需先离散化
### 10.5 Python代码实现

```python
"""
蚁群优化 (Ant Colony Optimization) -- TSP求解 (ACS版本)
"""
import numpy as np
import matplotlib
matplotlib.use("Agg")
import matplotlib.pyplot as plt


class AntColonySystem:
    """ACS求解TSP"""

    def __init__(self, dist_matrix, n_ants=20, alpha=1.0, beta=3.0,
                 rho=0.1, q0=0.9, tau0=None, max_iter=200,
                 local_search=True, verbose=True):
        self.dist = np.array(dist_matrix)
        self.n = self.dist.shape[0]
        self.n_ants = n_ants
        self.alpha = alpha
        self.beta = beta
        self.rho = rho
        self.q0 = q0
        self.max_iter = max_iter
        self.local_search = local_search
        self.verbose = verbose

        self.eta = np.zeros_like(self.dist)
        for i in range(self.n):
            for j in range(self.n):
                if i != j:
                    self.eta[i][j] = 1.0 / (self.dist[i][j] + 1e-10)

        if tau0 is None:
            tour_len = self._greedy_tour_length()
            tau0 = 1.0 / (self.n * tour_len + 1e-10)
        self.tau = np.ones((self.n, self.n)) * tau0
        self.history = []

    def _greedy_tour_length(self):
        unvisited = set(range(self.n))
        current = 0
        unvisited.remove(current)
        tour_len = 0
        while unvisited:
            next_city = min(unvisited, key=lambda j: self.dist[current][j])
            tour_len += self.dist[current][next_city]
            current = next_city
            unvisited.remove(current)
        tour_len += self.dist[current][0]
        return tour_len

    def _construct_solution(self):
        tours = np.zeros((self.n_ants, self.n), dtype=int)
        tour_lengths = np.zeros(self.n_ants)

        for k in range(self.n_ants):
            start = np.random.randint(self.n)
            unvisited = set(range(self.n))
            unvisited.remove(start)
            tours[k][0] = start
            current = start

            for step in range(1, self.n):
                candidates = list(unvisited)
                unvisited_arr = np.array(candidates)
                tau_vals = self.tau[current][unvisited_arr] ** self.alpha
                eta_vals = self.eta[current][unvisited_arr] ** self.beta
                probs = tau_vals * eta_vals
                probs = probs / probs.sum()

                q = np.random.rand()
                if q <= self.q0:
                    next_idx = np.argmax(probs)
                else:
                    next_idx = np.random.choice(len(candidates), p=probs)

                next_city = candidates[next_idx]
                tours[k][step] = next_city
                unvisited.remove(next_city)

                self.tau[current][next_city] = (
                    (1 - 0.1) * self.tau[current][next_city]
                    + 0.1 * self.tau[0][0]
                )
                self.tau[next_city][current] = self.tau[current][next_city]
                current = next_city

            length = 0
            for i in range(self.n):
                u, v = tours[k][i], tours[k][(i+1) % self.n]
                length += self.dist[u][v]
            tour_lengths[k] = length

        return tours, tour_lengths

    def _two_opt(self, tour):
        n = len(tour)
        improved = True
        best_length = self._tour_length(tour)
        while improved:
            improved = False
            for i in range(n - 1):
                for j in range(i + 2, n):
                    if i == 0 and j == n - 1:
                        continue
                    a, b = tour[i], tour[(i+1) % n]
                    c, d = tour[j], tour[(j+1) % n]
                    delta = (self.dist[a][c] + self.dist[b][d] -
                             self.dist[a][b] - self.dist[c][d])
                    if delta < -1e-10:
                        new_tour = tour.copy()
                        new_tour[i+1:j+1] = new_tour[i+1:j+1][::-1]
                        new_length = self._tour_length(new_tour)
                        if new_length < best_length:
                            tour = new_tour
                            best_length = new_length
                            improved = True
            if not improved:
                break
        return tour

    def _tour_length(self, tour):
        return sum(self.dist[tour[i]][tour[(i+1) % self.n]]
                   for i in range(len(tour)))

    def run(self):
        best_tour = None
        best_length = float("inf")
        self.history = []
        for iteration in range(self.max_iter):
            tours, lengths = self._construct_solution()
            if self.local_search:
                for k in range(self.n_ants):
                    tours[k] = self._two_opt(tours[k])
                    lengths[k] = self._tour_length(tours[k])
            iter_best_idx = np.argmin(lengths)
            if lengths[iter_best_idx] < best_length:
                best_length = lengths[iter_best_idx]
                best_tour = tours[iter_best_idx].copy()
            self.tau *= (1 - self.rho)
            for i in range(self.n):
                u = best_tour[i]
                v = best_tour[(i+1) % self.n]
                self.tau[u][v] += self.rho / (best_length + 1e-10)
                self.tau[v][u] = self.tau[u][v]
            self.history.append(best_length)
            if self.verbose and (iteration % 20 == 0 or iteration == self.max_iter - 1):
                print(f"Iter {iteration:4d}: Best Length = {best_length:.2f}")
        return best_tour, best_length
```

## 11. 混合整数线性规划 (Mixed Integer Linear Programming)

### 11.1 数学原理

混合整数线性规划(MILP)是线性规划的扩展, 允许部分变量取整数(包括0-1变量), 其余变量取实数。MILP是运筹学和数学建模中最强大的建模工具之一。

**数学模型**:

$$\begin{aligned}
\min \quad & \mathbf{c}^T \mathbf{x} + \mathbf{d}^T \mathbf{y} \\
\text{s.t.} \quad & \mathbf{A}\mathbf{x} + \mathbf{B}\mathbf{y} \leq \mathbf{b} \\
& \mathbf{x} \in \mathbb{Z}^n_+, \; \mathbf{y} \in \mathbb{R}^p_+
\end{aligned}$$

其中 $\mathbf{x}$ 为整数变量, $\mathbf{y}$ 为连续变量。

**Branch and Cut (B&C)** = 分支定界 + 割平面, 是现代MILP求解器的标准框架。

**逻辑约束的MILP建模**:

| 逻辑关系 | MILP等价表示 |
|----------|-------------|
| 互斥: x 和 y 不能同时 >0 | $x \le M \cdot \delta, \; y \le M \cdot (1-\delta)$ |
| 蕴含: 若delta=1 则 ax<=b | $ax \le b + M(1-\delta)$ |
| 或约束 | $f_1 \le M\delta_1, f_2 \le M\delta_2, \delta_1+\delta_2 \ge 1$ |

**小M vs 大M**: M应尽量小(紧), 否则LP松弛会变得很弱, 导致B&B效率极低。

### 11.2 适用场景

| 场景 | 建模特点 | 示例 |
|------|----------|------|
| 设施选址 | 0-1决策 + 连续运输量 | 仓库开设、基站选址 |
| 生产计划 | 整数批次 + 连续产量 | 批量生产、配料混合 |
| 排班调度 | 0-1分配 + 连续时间 | 护士/飞行员/课程排班 |
| 网络设计 | 0-1链路选择 + 连续流 | 通信网/供应链设计 |
| 电力调度 | 0-1机组启停 | 发电机组调度 |

### 11.3 优点

- **建模能力极强**: 可以表达绝大多数实际优化问题
- **求解器成熟**: CPLEX、Gurobi每秒可处理数百万节点
- **最优性保证**: 在足够时间内可证明最优或给出gap保证

### 11.4 缺点

- **求解速度不稳定**: 类似规模的问题求解时间可差几个数量级
- **M的选择困难**: Big-M太小会切断最优解, 太大会造成数值问题
- **大规模困难**: 变量超过百万时需要分解方法
- **商业求解器昂贵**: Gurobi/CPLEX许可费用高
### 11.5 Python代码实现

```python
"""
混合整数线性规划 (MILP) 求解示例
使用 pulp (CBC)
"""
import numpy as np

try:
    import pulp
    HAS_PULP = True
except ImportError:
    HAS_PULP = False
    print("警告: pulp未安装")


def example1_production():
    """生产计划MILP: 多种产品 + 最小批次 + 固定成本"""
    if not HAS_PULP:
        return
    print("=" * 60)
    print("示例1: 生产计划MILP")
    print("=" * 60)

    products = [
        {"price": 100, "var_cost": 40, "fixed_cost": 1000,
         "min_batch": 50, "max_prod": 500},
        {"price": 150, "var_cost": 60, "fixed_cost": 1500,
         "min_batch": 40, "max_prod": 400},
        {"price": 200, "var_cost": 80, "fixed_cost": 2000,
         "min_batch": 30, "max_prod": 300},
    ]
    n = len(products)
    resource_limits = [800, 1200]
    resource_usage = [[1.5, 2.0, 2.5], [2.0, 1.5, 3.0]]

    prob = pulp.LpProblem("Production_MILP", pulp.LpMaximize)
    x = [pulp.LpVariable(f"x_{i}", lowBound=0, cat="Integer") for i in range(n)]
    y = [pulp.LpVariable(f"y_{i}", cat="Binary") for i in range(n)]

    prob += pulp.lpSum([
        (products[i]["price"] - products[i]["var_cost"]) * x[i]
        - products[i]["fixed_cost"] * y[i] for i in range(n)
    ])
    for r in range(len(resource_limits)):
        prob += pulp.lpSum(
            [resource_usage[r][i] * x[i] for i in range(n)]
        ) <= resource_limits[r]
    for i in range(n):
        prob += x[i] <= products[i]["max_prod"] * y[i]
        prob += x[i] >= products[i]["min_batch"] * y[i]

    prob.solve(pulp.PULP_CBC_CMD(msg=False))
    print(f"状态: {pulp.LpStatus[prob.status]}")
    print(f"最大利润: {pulp.value(prob.objective):.0f}")
    for i in range(n):
        produce = pulp.value(y[i])
        quantity = pulp.value(x[i])
        if produce:
            profit_per = products[i]["price"] - products[i]["var_cost"]
            total_profit = profit_per * quantity - products[i]["fixed_cost"] * produce
            print(f"  产品{i+1}: 生产, 产量={quantity:.0f}, 利润贡献={total_profit:.0f}")
        else:

def example2_facility():
    """设施选址MILP"""
    if not HAS_PULP:
        return
    print("\n" + "=" * 60)
    print("示例2: 设施选址与容量规划MILP")
    print("=" * 60)

    n_sites = 4
    n_customers = 8
    np.random.seed(123)
    customer_demand = np.random.randint(10, 50, n_customers)
    capacity_options = [(100, 5000), (200, 9000), (350, 15000)]
    n_cap = len(capacity_options)
    transport_cost = np.random.randint(2, 15, (n_sites, n_customers))

    prob = pulp.LpProblem("Facility_Location_MILP", pulp.LpMinimize)
    y = [[pulp.LpVariable(f"y_{i}_{k}", cat="Binary")
          for k in range(n_cap)] for i in range(n_sites)]
    x = [[pulp.LpVariable(f"x_{i}_{j}", lowBound=0)
          for j in range(n_customers)] for i in range(n_sites)]

    fixed_cost = pulp.lpSum(
        [capacity_options[k][1] * y[i][k] for i in range(n_sites) for k in range(n_cap)]
    )
    transport = pulp.lpSum(
        [transport_cost[i][j] * x[i][j] for i in range(n_sites) for j in range(n_customers)]
    )
    prob += fixed_cost + transport

    for j in range(n_customers):
        prob += pulp.lpSum([x[i][j] for i in range(n_sites)]) == customer_demand[j]
    for i in range(n_sites):
        prob += pulp.lpSum([y[i][k] for k in range(n_cap)]) <= 1
    for i in range(n_sites):
        total_ship = pulp.lpSum([x[i][j] for j in range(n_customers)])
        total_cap = pulp.lpSum([capacity_options[k][0] * y[i][k] for k in range(n_cap)])
        prob += total_ship <= total_cap
    for i in range(n_sites - 1):
        prob += pulp.lpSum([y[i][k] for k in range(n_cap)]) >= pulp.lpSum([y[i+1][k] for k in range(n_cap)])

    prob.solve(pulp.PULP_CBC_CMD(msg=False))
    print(f"状态: {pulp.LpStatus[prob.status]}")
    print(f"最小总成本: {pulp.value(prob.objective):.0f}")
    for i in range(n_sites):
        for k in range(n_cap):
            if pulp.value(y[i][k]) == 1:
                cap = capacity_options[k][0]
                cost = capacity_options[k][1]
                print(f"  仓库{i+1}: 容量{cap}, 固定成本{cost}")
                for j in range(n_customers):
                    ship = pulp.value(x[i][j])
                    if ship > 0.01:
                        print(f"    -> 客户{j+1}: {ship:.0f}单位 (需求{customer_demand[j]})")

    """电力机组组合 (Unit Commitment)"""
    if not HAS_PULP:
        return
    print("\n" + "=" * 60)
    print("示例3: 电力机组组合 (Unit Commitment)")
    print("=" * 60)

    n_units = 4
    n_hours = 6
    units = [
        {"Pmax": 200, "Pmin": 50, "cost": 10, "startup": 500, "init_on": 0},
        {"Pmax": 150, "Pmin": 30, "cost": 12, "startup": 400, "init_on": 0},
        {"Pmax": 100, "Pmin": 20, "cost": 15, "startup": 300, "init_on": 1},
        {"Pmax": 80,  "Pmin": 10, "cost": 20, "startup": 200, "init_on": 1},
    ]
    demand = [280, 250, 220, 260, 320, 300]

    prob = pulp.LpProblem("Unit_Commitment", pulp.LpMinimize)
    p = [[pulp.LpVariable(f"p_{i}_{t}", lowBound=0) for t in range(n_hours)] for i in range(n_units)]
    u = [[pulp.LpVariable(f"u_{i}_{t}", cat="Binary") for t in range(n_hours)] for i in range(n_units)]
    v = [[pulp.LpVariable(f"v_{i}_{t}", cat="Binary") for t in range(n_hours)] for i in range(n_units)]

    fuel_cost = pulp.lpSum([units[i]["cost"] * p[i][t] for i in range(n_units) for t in range(n_hours)])
    startup_cost = pulp.lpSum([units[i]["startup"] * v[i][t] for i in range(n_units) for t in range(n_hours)])
    prob += fuel_cost + startup_cost

    for t in range(n_hours):
        prob += pulp.lpSum([p[i][t] for i in range(n_units)]) == demand[t]
    for i in range(n_units):
        for t in range(n_hours):
            prob += p[i][t] >= units[i]["Pmin"] * u[i][t]
            prob += p[i][t] <= units[i]["Pmax"] * u[i][t]
    for i in range(n_units):
        for t in range(n_hours):
            prev_u = units[i]["init_on"] if t == 0 else u[i][t-1]
            prob += v[i][t] >= u[i][t] - prev_u

    prob.solve(pulp.PULP_CBC_CMD(msg=False))
    print(f"状态: {pulp.LpStatus[prob.status]}")
    print(f"最小总成本: {pulp.value(prob.objective):.0f}")
    print(f"\n{'时段':>6} {'需求':>6}", end="")
    for i in range(n_units):
        print(f" {'机组'+str(i+1):>10}", end="")
    print()
    for t in range(n_hours):
        print(f"{t+1:6d} {demand[t]:6d}", end="")
        for i in range(n_units):
            state = "开" if pulp.value(u[i][t]) > 0.5 else "关"
            power = pulp.value(p[i][t])
            print(f" {state} {power:.0f}MW", end="")
        print()
```

## 12. 蒙特卡洛优化 (Monte Carlo Optimization)

### 12.1 数学原理

蒙特卡洛方法通过大量随机采样来近似求解确定性和随机性问题。在优化领域, 蒙特卡洛方法利用随机抽样来探索解空间, 估计最优解。

**理论基础**:

**大数定律**:
$$\bar{X}_n = \frac{1}{n}\sum_{i=1}^{n} X_i \xrightarrow{p} \mathbb{E}[X] \quad \text{as } n \to \infty$$

**收敛速率**: 蒙特卡洛误差以 $O(1/\sqrt{n})$ 衰减, 与维数无关 (这是其最大优势)。

**主要类型**:

1. **纯随机搜索 (PRS)**: 在可行域均匀随机采样, 选最优
2. **拉丁超立方采样 (LHS)**: 每个维度分成N个等概率区间, 保证更好空间填充
3. **马尔可夫链蒙特卡洛 (MCMC)**: Metropolis-Hastings算法构建以目标函数为稳态分布的马尔可夫链
4. **交叉熵方法 (CEM)**: 维护参数化采样分布, 每代选精英更新分布参数向最优区域集中
5. **模拟退火** (第8章) -- MCMC的特例

### 12.2 适用场景

| 场景 | 示例 |
|------|------|
| 高维积分 | 贝叶斯推断中的后验期望估计 |
| 随机优化 | 随机需求/随机价格的供应链优化 |
| 风险分析 | VaR/CVaR的模拟估计 |
| 可靠性分析 | 复杂系统的故障概率估计 |
| 仿真优化 | 离散事件系统的参数优化 |
| 全局优化 | 不规则可行域、多峰函数 |
| 参数估计 | Bootstrap置信区间 |

### 12.3 优点

- **维数独立**: 收敛速率与维数无关, 高维问题优势明显
- **通用性强**: 几乎不需要假设条件
- **易于实现**: 核心逻辑简单
- **可并行**: 各样本独立
- **提供不确定性量化**: 自然产出置信区间

### 12.4 缺点

- **收敛慢**: $O(1/\sqrt{n})$ 比确定性方法的指数收敛慢得多
- **精度有限**: 要达到高精度需要极大的样本量
- **随机误差**: 每次运行结果不同 (可通过固定种子复现)
- **效率取决于采样策略**: 朴素MC在低维比确定性方法差

```python
"""
蒙特卡洛优化方法
"""
import numpy as np
from scipy.stats import qmc
import matplotlib
matplotlib.use("Agg")
import matplotlib.pyplot as plt
import time


def pure_random_search(objective, bounds, n_samples=10000):
    """纯随机搜索全局优化"""
    dim = len(bounds)
    lo = np.array([b[0] for b in bounds])
    hi = np.array([b[1] for b in bounds])
    samples = lo + np.random.rand(n_samples, dim) * (hi - lo)
    values = np.array([objective(s) for s in samples])
    best_idx = np.argmin(values)
    return samples[best_idx], values[best_idx], samples, values


def latin_hypercube_search(objective, bounds, n_samples=10000):
    """使用LHS的蒙特卡洛搜索"""
    dim = len(bounds)
    lo = np.array([b[0] for b in bounds])
    hi = np.array([b[1] for b in bounds])
    sampler = qmc.LatinHypercube(d=dim)
    samples_unit = sampler.random(n=n_samples)
    samples = qmc.scale(samples_unit, lo, hi)
    values = np.array([objective(s) for s in samples])
    best_idx = np.argmin(values)
    return samples[best_idx], values[best_idx], samples, values


class CrossEntropyMethod:
    """交叉熵方法 (CEM) 用于连续优化"""

    def __init__(self, objective_func, bounds, pop_size=100,
                 elite_ratio=0.1, max_iter=100, smoothing=0.7,
                 verbose=True):
        self.objective_func = objective_func
        self.bounds = np.array(bounds)
        self.dim = len(bounds)
        self.pop_size = pop_size
        self.n_elite = max(1, int(pop_size * elite_ratio))
        self.max_iter = max_iter
        self.smoothing = smoothing
        self.verbose = verbose
        self.history = []

    def initialize_distribution(self):
        self.mu = np.array([(lo + hi) / 2 for lo, hi in self.bounds])
        self.sigma = np.array([(hi - lo) / 4 for lo, hi in self.bounds])

    def sample(self):
        samples = np.random.randn(self.pop_size, self.dim) * self.sigma + self.mu
        for i in range(self.dim):
            samples[:, i] = np.clip(samples[:, i], self.bounds[i][0], self.bounds[i][1])
        return samples

    def update_distribution(self, elite_samples):
        new_mu = elite_samples.mean(axis=0)
        new_sigma = elite_samples.std(axis=0) + 1e-6
        self.mu = self.smoothing * new_mu + (1 - self.smoothing) * self.mu
        self.sigma = self.smoothing * new_sigma + (1 - self.smoothing) * self.sigma
        max_sigma = (self.bounds[:, 1] - self.bounds[:, 0]) / 2
        min_sigma = max_sigma * 0.001
        self.sigma = np.clip(self.sigma, min_sigma, max_sigma)

    def run(self):
        self.initialize_distribution()
        self.history = []
        for iteration in range(self.max_iter):
            samples = self.sample()
            values = np.array([self.objective_func(s) for s in samples])
            elite_idx = np.argsort(values)[:self.n_elite]
            elite_samples = samples[elite_idx]
            best_val = values[elite_idx[0]]
            self.update_distribution(elite_samples)
            self.history.append(best_val)
            if self.verbose and (iteration % 20 == 0 or iteration == self.max_iter - 1):
                print(f"Iter {iteration:4d}: Best = {best_val:.8f}, "
                      f"|mu| = {np.linalg.norm(self.mu):.4f}, "
                      f"|sigma| = {np.linalg.norm(self.sigma):.4f}")
        return self.mu.copy(), self.history[-1]


def mcmc_optimization(objective, bounds, n_iter=10000, step_size=0.1,
                       annealing=True, T_start=10, T_end=0.01):
    """Metropolis-Hastings采样 + 退火优化"""
    dim = len(bounds)
    lo = np.array([b[0] for b in bounds])
    hi = np.array([b[1] for b in bounds])
    x = lo + np.random.rand(dim) * (hi - lo)
    f_x = objective(x)
    best_x = x.copy()
    best_f = f_x
    history = [(0, best_f)]
    accepted = 0

    for t in range(1, n_iter + 1):
        x_proposal = x + np.random.randn(dim) * step_size * (hi - lo)
        x_proposal = np.clip(x_proposal, lo, hi)
        f_proposal = objective(x_proposal)
        if annealing:
            T = T_start * (T_end / T_start) ** (t / n_iter)
        else:
            T = 1.0
        delta = f_proposal - f_x
        if delta < 0 or np.random.rand() < np.exp(-delta / T):
            x = x_proposal
            f_x = f_proposal
            accepted += 1
            if f_x < best_f:
                best_x = x.copy()
                best_f = f_x
        if t % (n_iter // 20) == 0:
            history.append((t, best_f))

    accept_rate = accepted / n_iter
    return best_x, best_f, accept_rate, history


if __name__ == "__main__":
    print("=" * 60)
    print("蒙特卡洛优化方法对比")
    print("=" * 60)

    def goldstein_price(x):
        x1, x2 = x[0], x[1]
        part1 = 1 + (x1 + x2 + 1)**2 * (19 - 14*x1 + 3*x1**2
                                          - 14*x2 + 6*x1*x2 + 3*x2**2)
        part2 = 30 + (2*x1 - 3*x2)**2 * (18 - 32*x1 + 12*x1**2
                                           + 48*x2 - 36*x1*x2 + 27*x2**2)
        return part1 * part2

    bounds_2d = [(-2, 2), (-2, 2)]

    print("\n--- 纯随机搜索 (N=20000) ---")

    best_prs, val_prs, _, _ = pure_random_search(goldstein_price, bounds_2d, n_samples=20000)
    print(f"最优: ({best_prs[0]:.4f}, {best_prs[1]:.4f}), 值={val_prs:.4f}, 耗时={time.time()-t0:.2f}s")

    print("\n--- 拉丁超立方采样 (N=20000) ---")
    t0 = time.time()
    best_lhs, val_lhs, _, _ = latin_hypercube_search(goldstein_price, bounds_2d, n_samples=20000)
    print(f"最优: ({best_lhs[0]:.4f}, {best_lhs[1]:.4f}), 值={val_lhs:.4f}, 耗时={time.time()-t0:.2f}s")

    print("\n--- 交叉熵方法 (CEM) ---")
    cem = CrossEntropyMethod(goldstein_price, bounds_2d,
                             pop_size=200, elite_ratio=0.1, max_iter=150, verbose=True)
    t0 = time.time()
    best_cem, val_cem = cem.run()
    print(f"最优: ({best_cem[0]:.4f}, {best_cem[1]:.4f}), 值={val_cem:.4f}, 耗时={time.time()-t0:.2f}s")

    print("\n--- MCMC优化 (退火) ---")
    t0 = time.time()
    best_mcmc, val_mcmc, accept_rate, _ = mcmc_optimization(
        goldstein_price, bounds_2d, n_iter=20000, step_size=0.15, annealing=True)
    print(f"最优: ({best_mcmc[0]:.4f}, {best_mcmc[1]:.4f}), 值={val_mcmc:.4f}, "
          f"接受率={accept_rate:.2%}, 耗时={time.time()-t0:.2f}s")

    print("\n" + "=" * 60)
    print("结果汇总 (理论最优值=3.0)")
    print("=" * 60)
    print(f"{'方法':<20} {'最优值':<12} {'解坐标'}")

    for name, val, x in [
```
