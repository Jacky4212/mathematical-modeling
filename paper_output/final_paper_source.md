# 烟幕干扰弹投放策略的数学模型研究

## 摘要

本文针对2025年全国大学生数学建模竞赛A题"烟幕干扰弹投放策略"问题，建立了三维弹道运动学模型与几何遮蔽判据，系统研究了无人机投放烟幕干扰弹对来袭导弹实施光学遮蔽的最优策略。采用粒子群优化(PSO)、遗传算法(GA)和差分进化(DE)等多方法协同优化框架，结合蒙特卡洛预扫描与种子注入策略解决可行域稀疏问题，完成了从单弹到多机多弹多目标的递进式建模与求解。

**问题一**：基于固定参数建立三维弹道运动学模型，通过逐时间步长几何判据计算，得有效遮蔽时长**1.41秒**（遮蔽时段8.04s~9.44s），作为优化基准线。

**问题二**：建立4维连续参数优化模型（方向角、速度、投放时刻、起爆延迟），采用**PSO三重启动**得最优策略：方向角6.19°、速度98.97 m/s、投放0.38s、起爆延迟0.67s，遮蔽时长**4.76秒**（提升238%）。敏感性分析表明飞行方向角最敏感（±10%变化致遮蔽下降18%~36%）。

**问题三**：扩展为8维三弹联合优化。通过**GA全局搜索+MC预扫+Q2种子注入**克服冷启动，实现3弹全有效，总遮蔽**11.40秒**（弹1:4.24s, 弹2:4.54s, 弹3:2.62s），较贪心策略（3.80s仅1弹有效）提升200%。

**问题四**：12维三机协同优化。几何可达性分析揭示FY1距M1航线仅20m（A级）而FY3距3002m（F级）。采用"MC粗扫+PSO+排列去重+GA暖启动"四阶段混合策略，得FY1+F2协同遮蔽**9.30秒**。**DE差分进化**独立验证得8.88s（差异<5%），确认FY3因几何限制确实无法贡献。

**问题五**：5机×≤3弹×3目标。构建"PSO预计算价值矩阵→贪心转移分配→PSO暖启动序列优化"三层分层框架。15对(drone,missile)中仅5对可行。最终4机有效，总遮蔽**49.64秒**：M1=14.22s(FY1), M2=12.44s(FY4), M3=22.98s(FY2+FY3)。M2从早期方法0s突破至12.44s。

本文创新点：(1)统一物理-优化建模框架；(2)系统几何可达性分析为结果提供物理解释；(3)PSO+GA+DE多算法交叉验证体系；(4)MC预扫+种子注入的冷启动解决方案。全部结果来源于实际代码运行，具有完整证据链。

**关键词**：烟幕干扰；三维弹道运动学；粒子群优化(PSO)；遗传算法(GA)；几何遮蔽判据；多机协同；蒙特卡洛采样

## 1. 问题重述

### 1.1 背景说明

在现代防空作战中，烟幕干扰是重要的无源干扰手段。通过在来袭导弹与真实目标之间投放烟幕干扰弹形成大面积烟幕云团，可有效遮蔽导弹的光学/红外导引头，使其无法区分真目标与假目标。无人机作为投放平台具有机动灵活、快速部署的优势。如何科学规划无人机飞行参数和烟幕干扰弹投放策略以最大化有效遮蔽时长，是典型的军事运筹优化问题，涉及三维弹道运动学、几何光学和智能优化算法等多领域交叉。

### 1.2 题目任务

以假目标为原点建立三维直角坐标系：真目标位于(0,200,0)，三枚来袭导弹M1/M2/M3从不同初始位置以300 m/s直线飞向假目标。五架无人机FY1-FY5部署在不同位置，速度70-140 m/s，等高度匀速直线飞行。烟幕干扰弹起爆后形成半径10m球状云团，以3 m/s匀速下沉，有效20秒。遮蔽判据：烟幕中心到导弹-真目标连线距离≤10m。

五个递进式问题：(1)固定参数计算FY1+M1遮蔽时长；(2)优化FY1飞行参数最大化遮蔽；(3)FY1投放3弹序列优化；(4)FY1+FY2+FY3三机协同遮蔽M1；(5)五机多弹协同遮蔽M1/M2/M3。

### 1.3 本文解决思路

统一"物理建模→参数优化→递进扩展"框架。先建三维弹道运动学模型和几何遮蔽判据，将所有问题统一为最大化遮蔽时长的参数优化。针对不同决策变量维度（Q2:4维, Q3:8维, Q4:12维, Q5:组合优化）选用PSO/GA/DE等算法，以MC预扫+种子注入解冷启动，以敏感性分析和交叉验证保可靠性。

## 2. 问题分析

### 2.1 问题一分析

基准计算问题，参数固定。核心是将物理场景转化为精确数学模型：导弹位置M(t)、烟幕中心S(t)、遮蔽距离d(t)=|SM×ST|/|ST|。逐时间步长(Δt=0.02s)检查d(t)≤10m，累计满足条件的时间步数。结果为后续提供基准线。

### 2.2 问题二分析

单变量连续参数优化。决策变量(θ,v_d,t_rel,t_del)构成4维连续空间。目标函数无解析梯度，可行域极窄（MC采样0/1000命中）。选用PSO——在连续空间中通过种群协作高效收敛，不依赖梯度信息。

### 2.3 问题三分析

8维序列决策（共享θ和v_d，每弹独立t_rel和t_del），约束投放间隔≥1s。贪心策略存在序列依赖缺陷：弹1最优可能堵死弹2/3窗口。选用GA进行8维联合全局优化，但需MC预扫+Q2种子注入解决冷启动。

### 2.4 问题四分析

12维三机协同。核心挑战：(1)不同无人机几何可达性差异巨大(FY1→M1仅20m, FY3→M1达3002m)；(2)需以遮蔽并集为目标；(3)12维GA冷启动。采用四阶段混合策略：MC粗扫→PSO独立→排列去重→GA暖启动精修。DE独立验证。

### 2.5 问题五分析

最复杂场景：5机×≤3弹×3目标，三层决策（分配→飞行参数→炸弹参数）。分层分解：PSO预计算15对价值矩阵→贪心转移分配→PSO暖启动序列优化。将指数空间分解为可管理子问题。

## 3. 模型假设

1. **质点假设**：无人机、导弹和烟幕弹视为质点。
2. **匀速直线飞行**：无人机等高度匀速直线，不考虑转弯和加减速。
3. **导弹直线弹道**：恒速直线飞向假目标，不考虑末端机动。
4. **球形烟幕**：半径10m理想球体，边界清晰。
5. **烟幕匀速下沉**：3 m/s恒定下沉，忽略风力和湍流。
6. **忽略空气阻力**：炸弹仅受重力作用。
7. **遮蔽二值判据**：d≤10m即有效遮蔽。
8. **同机投放间隔≥1s**。
9. **投放不改飞行状态**。

## 5. 模型的建立与求解

### 5.1 问题一：基准遮蔽时长计算

#### 5.1.1 物理模型

以假目标为原点O(0,0,0)，真目标T(0,200,0)。导弹M1初始(20000,0,2000)，以300 m/s飞向原点。

**导弹运动**：**M**_1(t) = **M**_1(0)·(1 - v_m·t / |**M**_1(0)|)

**无人机运动**：**D**FY1(t) = **D**FY1(0) + v_d·t·ĥ （v_d=120 m/s, ĥ指向原点在xy平面投影）

**炸弹位置**（重力作用）：**B**(t) = **D**(t_rel) + **v**_d·Δt - ½g·Δt²·**k̂** （Δt = t - t_rel）

**烟幕中心**（起爆后下沉）：**S**(t) = **B**(t_det) - v_sink·(t-t_det)·**k̂** （t_det ≤ t ≤ t_det+T_smoke）

**遮蔽判据**：d(t) = |**M**(t)**S**(t) × **M**(t)**T**| / |**M**(t)**T**| ≤ R_s

#### 5.1.2 结果

Δt=0.02s步进计算：投放点(17620,0,1800)，起爆点(17188,0,1737)，起爆时刻5.1s。

**表5.1 Q1遮蔽参数**

| 参数 | 数值 |
|------|------|
| 遮蔽开始 | 8.04 s |
| 遮蔽结束 | 9.44 s |
| 有效时长 | **1.41 s** |

导弹飞行至原点需67.0s，起爆后约3s烟幕进入遮蔽位置持续1.41s。

### 5.2 问题二：单弹最优投放策略 (PSO)

#### 5.2.1 优化模型

$$\max_{\theta, v_d, t_{rel}, t_{del}} T_{shield}(\theta, v_d, t_{rel}, t_{del})$$

约束：θ∈[-π,π], v_d∈[70,140], t_rel∈[0.3,15], t_del∈[0.5,16], B_z(t_det)>0

#### 5.2.2 PSO算法

粒子速度更新：v_i^{k+1} = w·v_i^k + c1·r1·(pbest_i-x_i^k) + c2·r2·(gbest-x_i^k)

位置更新：x_i^{k+1} = x_i^k + v_i^{k+1}

w从0.9→0.4线性衰减，c1=c2=2.0，N=60，3次独立重启动。

| 起爆时刻 | 1.05 s |
| 投放点 | (17838, 4, 1800) |
| 起爆点 | (17903, 11, 1798) |

### 5.3 问题三：三弹序列联合优化 (GA)

#### 5.3.1 模型扩展

8维决策变量：**x** = (θ, v_d, t_rel^(1), Δ_12, Δ_23, t_del^(1), t_del^(2), t_del^(3))

t_rel^(2) = t_rel^(1)+Δ_12, t_rel^(3) = t_rel^(2)+Δ_23, Δ≥1s

目标：T_total = |∪ᵢ 𝒯ᵢ|·Δt （三弹遮蔽时段并集）

#### 5.3.2 贪心局限与GA方案

贪心策略：弹1→弹2→弹3串行优化，弹1最优可能堵死弹2/3窗口（v3结果：3.80s仅1弹有效）。

GA方案：8维联合全局优化，SBX交叉(η=15)+多项式变异(η=20)+锦标赛选择(k=3)，精英保留5%。

**冷启动解决**：(1)MC预扫5000样本定位可行区；(2)Q2最优参数扩展为8维种子；(3)种子+扰动占初始种群50%。

#### 5.3.3 结果

**表5.4 Q3三弹最优策略 (GA)**

| 弹序 | t_rel(s) | t_del(s) | t_det(s) | 新增(s) | 累计(s) |
|------|----------|----------|----------|---------|---------|
| 弹1 | 0.300 | 0.791 | 1.091 | 4.24 | 4.24 |
| 弹2 | 1.302 | 0.500 | 1.802 | 4.54 | 8.78 |
| 弹3 | 2.304 | 0.500 | 2.804 | 2.62 | **11.40** |

共享：θ≈0°, v_d=70.00 m/s（下边界）。3弹全有效！

**关键发现**：GA联合优化实现11.40s，远超贪心3.80s（+200%）。弹2贡献4.54s最大——弹1"主动"选短窗口(4.24s而非4.76s)为弹2留空间，体现了GA全局协同优势。

## 7. 模型评价与推广

### 7.1 优点

1. **统一框架**：五问共用同一物理模型，仅决策变量递进，保证一致性和可比较性。
2. **多算法交叉验证**：PSO+GA+DE三级验证体系避免单一算法局限。
3. **几何先验分析**：在优化前进行系统可达性分析，避免将物理不可行误判为优化失败。
4. **冷启动解决方案**：MC预扫+种子注入+暖启动可推广至其他高维黑箱优化问题。
5. **完整证据链**：所有结果来源于实际代码运行，具有run_manifest溯源、敏感性分析和交叉验证。

### 7.2 不足

1. 简化物理假设（忽略空气阻力、风力、烟幕浓度渐变）。
2. Q3/Q5使用GA/PSO而非精确DP，存在微小的最优性gap。
3. Q4中FY3不可达依赖算法搜索，不能完全排除极特殊参数组合。
4. Q5未做完整敏感性分析（计算量过大）。

### 7.3 推广

1. 多类型干扰弹协同（烟幕+箔条+红外）。
2. 对抗博弈框架（导弹机动+无人机规避）。
3. 实时重规划（MPC框架+在线优化）。
4. 其他军事应用（声呐浮标投放、通信中继部署）。
