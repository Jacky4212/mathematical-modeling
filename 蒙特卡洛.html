2. 蒙特卡洛方法

线性回归 →

一、蒙特卡洛方法基本原理 必须掌握

1.1 什么是蒙特卡洛方法

定义

蒙特卡洛方法（Monte Carlo Method）是一类基于随机采样和统计模拟的数值计算方法。其核心思想是：通过生成大量随机样本，用统计量（如样本均值）来近似待求的数学量（如积分值、期望值、概率等）。

蒙特卡洛方法得名于摩纳哥著名赌城，暗喻其"随机"本质。其应用极为广泛：数值积分、优化问题、物理模拟、金融风险分析、人工智能（MCMC 采样）等领域均有重要作用。

1.2 理论基础

大数定律 (Law of Large Numbers)

设 $X_1, X_2, \ldots, X_N$ 是独立同分布的随机变量，期望 $E[X_i] = \mu$，则当样本数 $N \to \infty$ 时：

$$\frac{1}{N} \sum X_i \to \mu$$ （依概率收敛）

含义：样本均值随着样本量的增加依概率收敛到真实期望值（弱大数定律），这保证了 MC 方法的一致性。

中心极限定理 (Central Limit Theorem)

设 $X_1, \ldots, X_N$ i.i.d.，方差 $\sigma^2$ 有限，则：

$$\sqrt{N} \cdot (\bar{X}_N - \mu) / \sigma \to N(0, 1)$$ （依分布收敛）

含义：MC 估计误差近似服从正态分布，误差的标准差 $\propto \sigma/\sqrt{N}$。这给出了 MC 方法的误差收敛速率 $O(1/\sqrt{N})$。

1.3 MC 方法工作流程

标准 MC 四步走：
① 定义域 → 将问题表示为一个定积分或期望形式
② 采样 → 在定义域 $\Omega$ 内生成 $N$ 个均匀（或依某种分布）随机点
③ 计算 → 计算每个采样点上的被积函数值
④ 聚合 → (均值) × (定义域的体积/面积) = 积分近似值
公式： $I = \int_{\Omega} f(x)\,dx \approx \frac{V}{N} \sum f(x_i)$，其中 $V = \text{vol}(\Omega)$

1.4 优缺点分析

优点缺点
维度无关性：收敛速度 $O(1/\sqrt{N})$ 与积分维数 $d$ 无关，而确定性格点法误差为 $O(N^{-2/d})$，高维时 MC 占优 收敛缓慢：精度每提高一位，样本量需增加 100 倍
几何灵活性：可处理任意复杂形状的积分区域，无需规则网格 精度有限：单位计算成本下的精度低于低维格点法
直观易懂：方法原理简单，易于编程实现和并行化 随机误差：结果带有统计不确定性，需要误差估计
黄金法则：对于 $d \leq 3$ 的低维积分，确定性方法（梯形法则、Simpson、Gauss）通常更优；当 $d \geq 4$ 时，MC 方法的效率开始超越确定性格点法。在金融数学（上千维度）、统计物理、贝叶斯推断等场景，MC 几乎是唯一可行的数值方法。

1.5 误差与收敛性深入分析

$$\varepsilon \approx \frac{\sigma}{\sqrt{N}}, \quad \sigma = \sqrt{\text{Var}[f(X)]}, \quad X \text{ 服从采样分布}$$

MC 误差有以下关键性质：

  • 概率性：误差估计是一个置信区间（通常取 95% CI = $\pm 1.96\sigma/\sqrt{N}$），而非严格的确定性误差界
  • 与维数无关：$O(1/\sqrt{N})$ 不含维数 $d$，这是 MC 在高维问题中的核心竞争力
  • 减方差技术：可通过重要性采样（沿重要区域多采样）、控制变量法（减去一个已知均值的相关变量）、对偶变量法（利用负相关降低方差）等手段，在不增加 $N$ 的情况下减小 $\sigma^2$
目标精度 $\varepsilon$需样本量 $N$（$\sigma=1$ 时）
10% (0.1)$\sim 100$
1% (0.01)$\sim 10{,}000$
0.1% (0.001)$\sim 1{,}000{,}000$
0.01% (0.0001)$\sim 100{,}000{,}000$

表中给出的是标准误 $\sigma/\sqrt{N}$ 的近似值，95% 置信区间半宽约为 $1.96\sigma/\sqrt{N}$。可清晰看出 MC 的 "平方根瓶颈"：精度提高 10 倍需要 100 倍的样本量。

二、蒙特卡洛积分 重点

2.1 一维 MC 积分

基本公式
$$I = \int_{a}^{b} f(x) \, dx \approx (b - a) \cdot \frac{1}{N} \sum f(x_i)$$

其中 $x_i \sim \text{Uniform}(a, b)$，即从区间 $[a, b]$ 上均匀随机采样 $N$ 个点。

原理：定积分等于函数在区间上的平均值乘以区间长度。MC 方法用随机样本的算术平均来估计函数均值。

% mc1d.m — 一维 MC 积分示例 % 计算 ∫₀¹ x² dx 的精确值为 1/3 ≈ 0.3333 a = 0; b = 1; N = 10000; % 样本数 x = a + (b - a) * rand(1, N); % U(a,b) 随机点 f = x.^2; % 被积函数 f(x) = x² I_mc = (b - a) * mean(f); % MC 积分值 err = abs(I_mc - 1/3); % 绝对误差 fprintf('MC = %.6f, 误差 = %.6e\n', I_mc, err);

2.2 二维与三维 MC 积分

MC 积分在高维直接推广——只需将区间长度替换为区域的面积/体积，采样从高维均匀分布中进行。

二维 MC 积分
$$\iint_{D} f(x, y) \, dxdy \approx (\text{Area of } D) \cdot \frac{1}{N} \sum f(x_i, y_i)$$

$(x_i, y_i) \sim \text{Uniform}(D)$，即从区域 $D$ 内均匀随机采样。

% mc2d.m — 二维 MC 积分：单位圆面积验证 % 单位圆面积 = π，精确值 ≈ 3.14159 N = 100000; % 策略：在 [-1,1]×[-1,1] 正方形中采样 x = 2*rand(1, N) - 1; y = 2*rand(1, N) - 1; inside = (x.^2 + y.^2) <= 1; % 点在单位圆内的逻辑索引 count = sum(inside); area_est = 4 * count / N; % 正方形面积 4 × 命中率 fprintf('MC 面积 ≈ %.6f, 实际 π ≈ %.6f\n', area_est, pi);

2.3 误差分析：MC vs 确定性格点法

方法一维误差阶$d$ 维误差阶$d=10$, $N=10^{6}$ 时误差
Monte Carlo$O(N^{-1/2})$$O(N^{-1/2})$$\sim 10^{-3}$
梯形法则$O(N^{-2})$$O(N^{-2/d})$$\sim 10^{-1.2} \approx 0.06$
Simpson 法则$O(N^{-4})$$O(N^{-4/d})$$\sim 10^{-2.4} \approx 0.004$
Gauss 求积指数收敛随 $d$ 急剧恶化需 $\approx 10^{d}$ 个节点
关键洞察：确定性格点法在低维高效（一维 Simpson 用 100 个点可达机器精度），但误差随维度升高以 $O(N^{-2/d})$ 的速率恶化——这就是著名的 "维度灾难"（curse of dimensionality）。MC 方法的收敛速率与维数无关，使其在高维数值积分中无可替代。

2.4 减方差技术概览

技术原理适用场景
重要性采样在被积函数大的区域以更高概率采样被积函数有尖峰或奇点
控制变量法减去一个已知积分值且与原函数相关的函数存在已知解析积分的近似函数
对偶变量法利用配对负相关样本抵消部分方差被积函数在采样区间上单调时，理想情况下方差可减半
分层采样将定义域分层，每层分别采样保证覆盖均匀区域可自然分层时
拟蒙特卡洛 (QMC)用低差异序列（Halton、Sobol）替代伪随机数理论收敛速率约 $O((\ln N)^{d} / N)$，但在极高维时 $\ln^{d} N$ 因子可能抵消优势

三、经典蒙特卡洛案例 重点

3.1 圆周率 π 的 MC 估计 必须掌握

这是最经典的 MC 入门案例，也被称为"投针实验"的数字版本。利用单位正方形内接 1/4 圆的面积关系估算 π。

原理

在单位正方形 $[0,1] \times [0,1]$ 内随机投点。
1/4 单位圆面积 = $\pi/4$，正方形面积 = 1。
点数落在 1/4 圆内的比例 $\approx \pi/4$，因此：

$$\pi \approx 4 \times \frac{\text{圆内点数}}{\text{总点数}}$$
% pi_mc.m — 蒙特卡洛估计 π N = 100000; x = rand(1, N); % U(0,1) 随机点 x 坐标 y = rand(1, N); % U(0,1) 随机点 y 坐标 inside = (x.^2 + y.^2) <= 1; % 判断是否在 1/4 圆内 pi_est = 4 * sum(inside) / N; fprintf('π ≈ %.6f, 误差 ≈ %.6e\n', pi_est, abs(pi_est - pi)); % 可视化收敛过程 pi_cum = 4 * cumsum(inside) ./ (1:N); plot(1:N, pi_cum); hold on yline(pi, 'r--'); % 参考线 π ≈ 3.1416 xlabel('样本数 N'); ylabel('π 估计值'); title('MC 估计 π — 收敛过程');

3.2 中子屏蔽模拟 竞赛常见

这是 MC 方法在核物理中的典型应用。考虑一个三层屏蔽板，中子顺序经过各层，每层可能发生反射、吸收或穿透。模拟大量中子的随机行踪，统计穿透概率和能量损失。

三层屏蔽模型参数（示例）
层穿透概率 $P_i$反射概率 $R_i$吸收概率 $A_i$
第1层（外层）0.70.20.1
第2层（中层）0.50.30.2
第3层（内层）0.30.40.3

注意：$P_i + R_i + A_i = 1$，每层为一个完备事件组。

% neutron_shield.m — 中子屏蔽 MC 模拟 P = [0.7 0.5 0.3]; % 各层穿透概率 R = [0.2 0.3 0.4]; % 各层反射概率 % A = 1 - P - R (吸收概率，不显式写出) N = 100000; % 模拟中子数 penetrate = 0; % 穿透计数 absorb = 0; % 吸收计数 for i = 1:N layer = 1; % 从第1层开始 while layer >= 1 && layer <= 3 r = rand(); if r < P(layer) % 穿透 layer = layer + 1; elseif r < P(layer) + R(layer) % 反射 layer = layer - 1; else % 吸收 absorb = absorb + 1; break; end end if layer > 3 penetrate = penetrate + 1; end end fprintf('穿透率 = %.4f, 吸收率 = %.4f\n', ... penetrate/N, absorb/N);
算法要点：当一个中子在某一层反射后回退到上一层，它可能再次穿透回当前层——这形成了一个随机游走（random walk）。每步决策对应一个 $[0,1]$ 均匀随机数与概率区间的比较。这种状态机式模拟是 MC 模拟复杂物理过程的范例。

3.3 不规则图形面积估计 — 叶片面积

将不规则叶片放入已知尺寸的矩形（包围盒）内，在矩形内均匀随机撒点，统计落入叶片内部的点数比例来估计面积。

算法流程
  1. 确定包围盒：$[x_{\min}, x_{\max}] \times [y_{\min}, y_{\max}]$
  2. 生成 $N$ 个随机点 $(x_i, y_i) \sim \text{Uniform}(\text{包围盒})$
  3. 用 射线法（ray casting）判断每个点是否在叶片多边形内部
  4. 叶片面积 $\approx$ 包围盒面积 $\times$ (内部点数 / $N$)
% leaf_area.m — 叶片面积 MC 估计（核心逻辑） % 假设 leaf_x, leaf_y 构成封闭多边形顶点坐标 xmin = min(leaf_x); xmax = max(leaf_x); ymin = min(leaf_y); ymax = max(leaf_y); box_area = (xmax - xmin) * (ymax - ymin); N = 50000; xp = xmin + (xmax - xmin) * rand(1, N); yp = ymin + (ymax - ymin) * rand(1, N); inside = inpolygon(xp, yp, leaf_x, leaf_y); % MATLAB 内置 leaf_area = box_area * sum(inside) / N; fprintf('叶片面积 ≈ %.4f (包围盒 = %.1f)\n', leaf_area, box_area); % 可视化 plot(leaf_x, leaf_y, 'k-'); hold on scatter(xp(inside), yp(inside), 1, 'b.'); % 内部点蓝色 scatter(xp(~inside), yp(~inside), 1, 'r.'); % 外部点红色
Point-in-Polygon 判断：最常用的是射线法（从测试点向右发射水平射线，计算与多边形边的交点数量——奇数个交点在内部，偶数个在外部）。MATLAB 内置函数 inpolygon() 封装了该算法，可直接调用。

3.4 高维积分示例

考虑一个 10 维单位超立方体上的积分，展示 MC 如何轻松处理维度灾难：

% mc_highdim.m — 10 维超球体积估计 d = 10; % 维度 N = 100000; points = rand(N, d); % N×d 矩阵，每行一个 d 维随机点 % 计算每个点到原点距离的平方 dist_sq = sum(points.^2, 2); % 按行求和 inside = dist_sq <= 1; % 包含立方体体积为 2^d，半径 1 的超球内接于 [-1,1]^d % 命中率 ≈ V_sphere / 2^d volume_est = 2^d * sum(inside) / N; volume_true = pi^(d/2) / gamma(d/2 + 1); % d 维单位超球体积 fprintf('MC 体积 ≈ %.6f, 精确值 ≈ %.6f\n', volume_est, volume_true);

四、数学建模竞赛中的应用 高频考点

4.1 2010 CUMCM A 题：储油罐标定 重点

问题背景：加油站储油罐在使用中会发生变位（倾斜），需要通过油位高度和储油量的对应关系来标定罐容表。由于罐体几何形状不规则（两端为球冠封头），精确体积积分极为复杂。

MC 方法解题思路：
① 建立罐体的三维几何模型（圆柱体 + 两个球冠）
② 沿罐体表面及内部空间大量随机采样
③ 对每个采样点，结合变位参数（纵向倾斜角 $\alpha$、横向偏转角 $\beta$），通过坐标变换计算该点在变位后的实际高度
④ 统计位于油面以下的点数比例 → 估算储油体积
优势：几何建模困难时，MC 可以"暴力"绕过复杂的解析几何推导，直接给出数值结果。
% oiltank_mc_core.m — 储油罐 MC 标定核心逻辑 % 参数：L = 罐长, R = 半径, r = 球冠半径, alpha = 倾斜角 N = 500000; % 步骤1：在罐体包围盒内随机采样 x_p = rand(N,1) * L; % 沿罐长方向 y_p = (rand(N,1)*2 - 1) * R; % 径向 z_p = (rand(N,1)*2 - 1) * R; % 步骤2：筛选在罐体内的点 in_tank = (y_p.^2 + z_p.^2) <= R^2; % 步骤3：变位坐标变换（绕 y 轴倾斜 α） x_new = x_p * cos(alpha) + z_p * sin(alpha); z_new = -x_p * sin(alpha) + z_p * cos(alpha); % 步骤4：统计油面以下点数 % 假设油位高度为 h（从罐底算起） below_oil = (z_new <= h - R) & in_tank; volume = 2*R * 2*R * L * sum(below_oil) / N;
竞赛中 MC 的定位：当精确解析推导过于复杂（如 2010A 需要推导含球冠的变位体积公式，涉及繁杂的积分），MC 可作为数值验证工具。先用解析方法推导近似模型，再用 MC 数值验证模型的精度，是竞赛中非常稳妥的策略。

4.2 2005 CUMCM D 题：DVD 租赁优化

问题背景：在线 DVD 租赁公司需要优化库存配置：购买多少张 DVD 才能以给定概率满足会员需求？会员按偏好排队，DVD 归还后可再借出。

随机模拟思路：
① 为每位会员生成随机偏好序列（按某种分布随机排列 DVD 编号）
② 按时间步进模拟租赁流程：分配 DVD → 会员使用 → 归还 → 再次入库
③ 多轮重复模拟（如 1000 次），统计每种库存配置下的满足率
④ 以满足率 $\geq$ 阈值（如 95%）为目标，反向搜索最优库存配置
本质：这是一个随机库存系统的离散事件模拟（DES），MC 用于处理需求和归还的不确定性。
