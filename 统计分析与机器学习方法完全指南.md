## 第一部分：统计分析

### 1. 相关性分析

#### 数学原理
| 方法 | 公式 | 数据类型 | 线性关系 |
|------|------|----------|----------|
| Pearson | $r = \frac{\sum(x_i-\bar{x})(y_i-\bar{y})}{\sqrt{\sum(x_i-\bar{x})^2\sum(y_i-\bar{y})^2}}$ | 连续正态 | 仅线性 |
| Spearman | $\rho = 1 - \frac{6\sum d_i^2}{n(n^2-1)}$ | 有序/秩 | 单调即可 |
| Kendall's τ | $\tau = \frac{2(C-D)}{n(n-1)}$ | 有序 | 单调即可 |

#### 适用场景
| 方法 | 适用场景 |
|------|----------|
| Pearson | 两连续变量，正态分布，线性相关 |
| Spearman | 有序变量、非正态、非线性单调 |
| Kendall | 小样本、多位次相同（ties多） |

#### 注意事项
1. Pearson对异常值敏感，做之前先画散点图
2. 相关≠因果
3. 强相关不意味着关系有意义（检查散点图）
4. 小样本时Pearson不稳健

#### 参考代码
```python
from scipy.stats import pearsonr, spearmanr, kendalltau

r, p = pearsonr(x, y)
rho, p = spearmanr(x, y)
tau, p = kendalltau(x, y)
```

### 2. 独立样本T检验

#### 数学原理
$$t = \frac{\bar{x}_1 - \bar{x}_2}{\sqrt{\frac{s_1^2}{n_1} + \frac{s_2^2}{n_2}}}$$

**方差齐性检验（Levene检验）**：p>0.05 → 方差齐 → Student's t；p<0.05 → 方差不齐 → Welch's t

#### 适用场景
两组独立样本均值差异比较（实验组vs对照组、男女比较）

#### 实现注意事项
1. 必须先做方差齐性检验
2. 报告效应量：Cohen's d = (mean1 - mean2) / pooled_std
3. d=0.2小效应, 0.5中等, 0.8大效应

#### 参考代码
```python
from scipy.stats import ttest_ind, levene

# 方差齐性检验
stat, p_levene = levene(group1, group2)
equal_var = p_levene > 0.05

# T检验
t_stat, p_value = ttest_ind(group1, group2, equal_var=equal_var)
# Cohen's d
d = (np.mean(group1) - np.mean(group2)) / np.sqrt((np.var(group1, ddof=1) + np.var(group2, ddof=1)) / 2)
```

### 3. 方差分析 ANOVA

#### 数学原理
$$F = \frac{MS_{between}}{MS_{within}} = \frac{SS_{between}/(k-1)}{SS_{within}/(N-k)}$$

$SS_{total} = SS_{between} + SS_{within}$（总变异 = 组间变异 + 组内变异）

#### 事后比较
| 方法 | 特点 |
|------|------|
| Tukey HSD | 控制族系误差率，所有配对比较 |
| Bonferroni | 保守，检验次数多时 |
| Dunnett | 与对照组比较 |

#### 适用场景
三组及以上均值差异比较

#### 实现注意事项
1. 需检验方差齐性（Levene）
2. 残差正态性（Q-Q图或Shapiro-Wilk）
3. 显著后必须做事后比较确定差异到底在哪里
4. 报告η²效应量

#### 参考代码
```python
from scipy.stats import f_oneway
from statsmodels.stats.multicomp import pairwise_tukeyhsd

# 单因素ANOVA
F, p = f_oneway(group1, group2, group3)

# Tukey HSD
tukey = pairwise_tukeyhsd(all_values, all_labels, alpha=0.05)
print(tukey)
```

### 4. 卡方检验

#### 数学原理
$$\chi^2 = \sum \frac{(O_{ij} - E_{ij})^2}{E_{ij}}, \quad E_{ij} = \frac{R_i \times C_j}{N}$$

#### 适用场景
两分类变量独立性检验、拟合优度检验

#### 实现注意事项
1. 期望频数 < 5 的格子数不超过20%
2. 2×2表用Yates连续性校正
3. 小样本用Fisher精确检验
4. 报告Cramér's V效应量

#### 参考代码
```python
from scipy.stats import chi2_contingency, fisher_exact

chi2, p, dof, expected = chi2_contingency(contingency_table)
# 验证期望频数
print("最小期望频数:", expected.min())
```

## 第二部分：回归分析

### 6. 线性回归 (OLS)

#### 数学原理
$$y = X\beta + \varepsilon, \quad \hat{\beta} = (X^TX)^{-1}X^Ty$$

$$R^2 = 1 - \frac{SS_{res}}{SS_{tot}}$$

### 多重共线性诊断
$$VIF_j = \frac{1}{1 - R_j^2}$$

VIF > 10 → 严重共线性；VIF > 5 → 需要注意

#### 适用场景
线性关系、自变量与因变量均连续

#### 实现注意事项
1. 残差检验：正态性(Q-Q)、独立性(Durbin-Watson≈2)、等方差性
2. VIF > 10 时考虑Ridge/Lasso或删除变量
3. 异常值：Cook's distance > 4/n需关注

#### 参考代码
```python
import statsmodels.api as sm
from statsmodels.stats.outliers_influence import variance_inflation_factor

X_sm = sm.add_constant(X)
model = sm.OLS(y, X_sm).fit()
print(model.summary())

# VIF
vif = pd.DataFrame({'feature': X.columns,
                     'VIF': [variance_inflation_factor(X.values, i)
                             for i in range(X.shape[1])]})
```

### 7. 岭回归 (Ridge)

#### 数学原理
$$\hat{\beta}_{ridge} = \arg\min_\beta \left\{ \sum (y_i - x_i\beta)^2 + \alpha \sum \beta_j^2 \right\}$$

L2正则化将所有系数向0收缩但不归零。

#### 适用场景
多重共线性严重（VIF>10）、特征多样本少

#### 优缺点
| 优点 | 缺点 |
|------|------|
| 解决多重共线性 | 不自动特征选择 |
| 降低过拟合 | α需调参 |
| 闭式解 | 所有特征保留 |

#### 参考代码
```python
from sklearn.linear_model import RidgeCV

ridge = RidgeCV(alphas=[0.01, 0.1, 1, 10, 100], cv=5)
ridge.fit(X_train_scaled, y_train)
print(f"最优α: {ridge.alpha_}")
```

### 8. Lasso回归

#### 数学原理
$$\hat{\beta}_{lasso} = \arg\min_\beta \left\{ \sum (y_i - x_i\beta)^2 + \alpha \sum |\beta_j| \right\}$$

L1正则化可将不重要系数直接压缩为0，具有内置特征选择。

#### Ridge vs Lasso
| 特性 | Ridge (L2) | Lasso (L1) |
|------|-----------|-----------|
| 特征选择 | 不归零 | 可压缩到0 |
| 共线性处理 | 分散权重 | 随机选择其一 |
| 适用 | 所有特征都有用 | 需要稀疏解 |

#### 适用场景
高维数据（p>n）、需要特征选择

#### 参考代码
```python
from sklearn.linear_model import LassoCV

lasso = LassoCV(cv=5, random_state=42, max_iter=10000)
lasso.fit(X_train_scaled, y_train)
selected = X.columns[lasso.coef_ != 0]
print(f"选中{len(selected)}个特征: {selected.tolist()}")
```

### 9. 逻辑回归

#### 数学原理
$$P(Y=1|X) = \frac{1}{1 + e^{-(\beta_0 + \beta_1X_1 + \cdots + \beta_kX_k)}}$$

$$\text{Odds Ratio(OR)} = e^{\beta_j}$$

#### 评估指标
| 指标 | 公式 | 说明 |
|------|------|------|
| Accuracy | (TP+TN)/Total | 类别平衡时用 |
| Precision | TP/(TP+FP) | 假阳性成本高 |
| Recall | TP/(TP+FN) | 假阴性成本高 |
| F1 | 2PR/(P+R) | 综合指标 |
| AUC-ROC | 曲线下面积 | **不平衡数据首选** |

#### 适用场景
二分类/多分类、医学诊断、信用评分、流失预测

#### 实现注意事项
1. 多重共线性：VIF>10需处理
2. 类别不平衡：SMOTE/调整class_weight
3. Hosmer-Lemeshow拟合优度检验
4. AUC>0.7可接受, >0.8良好, >0.9优秀

#### 参考代码
```python
from sklearn.linear_model import LogisticRegression
from sklearn.metrics import roc_auc_score, classification_report

lr = LogisticRegression(class_weight='balanced', max_iter=1000)
lr.fit(X_train_scaled, y_train)
y_prob = lr.predict_proba(X_test_scaled)[:, 1]
auc = roc_auc_score(y_test, y_prob)
print(f"AUC: {auc:.4f}")

# OR值
odds_ratios = np.exp(lr.coef_[0])
```

### 10. 分层回归

#### 数学原理
将自变量按理论/时间顺序分层引入模型：

**步骤**：
1. Model 1：仅控制变量 → $R^2_1$
2. Model 2：控制变量 + 新变量 → $R^2_2$
3. $\Delta R^2 = R^2_2 - R^2_1$，检验是否显著

#### 适用场景
控制混杂因素后检验新增变量的增量解释力

#### 参考代码
```python
import statsmodels.api as sm

# Model 1: 仅控制变量
X1 = sm.add_constant(df[['age', 'gender']])
m1 = sm.OLS(y, X1).fit()

# Model 2: + 核心变量
X2 = sm.add_constant(df[['age', 'gender', 'treatment']])
m2 = sm.OLS(y, X2).fit()

delta_r2 = m2.rsquared - m1.rsquared
print(f"ΔR² = {delta_r2:.4f}")
```

## 第三部分：聚类分析

### 11. K-Means 聚类

#### 数学原理
$$\min_{C_1,...,C_k} \sum_{i=1}^{k} \sum_{x \in C_i} \|x - \mu_i\|^2$$

#### K值选择方法
| 方法 | 原理 |
|------|------|
| **肘部法则** | SSE随K增大递减，找"拐点" |
| 轮廓系数 | 衡量簇内紧致+簇间分离，越近1越好 |
| CH指数 | 类间方差/类内方差，越大越好 |
| Gap Statistic | 与随机分布对比 |

#### 适用场景
球形簇、大数据集、需预知K值

#### 优缺点
| 优点 | 缺点 |
|------|------|
| 简单快速（O(n)） | 需预设K |
| 大数据友好 | 球形簇假设 |
| 可解释性强 | 对初始中心敏感 |

#### 参考代码
```python
from sklearn.cluster import KMeans
from sklearn.metrics import silhouette_score

# 肘部法则
inertias = []
for k in range(1, 11):
    inertias.append(KMeans(n_clusters=k, random_state=42).fit(X).inertia_)

# 轮廓系数
silhouettes = []
for k in range(2, 11):
    labels = KMeans(n_clusters=k, random_state=42).fit_predict(X)
    silhouettes.append(silhouette_score(X, labels))

# 最终聚类
kmeans = KMeans(n_clusters=best_k, random_state=42)
labels = kmeans.fit_predict(X)
```

### 12. 分层聚类

#### 数学原理
**凝聚法（自底向上）**：每个点初始为一簇 → 逐步合并最近的簇

#### 距离度量
| 方法 | 定义 | 特点 |
|------|------|------|
| Single-link | min(d(a,b)) | 链状效应 |
| Complete-link | max(d(a,b)) | 紧凑簇 |
| Average-link | mean(d(a,b)) | 中庸 |
| Ward | min(ΔSSE) | **推荐**，球形 |

#### 适用场景
需要层次结构、小数据集、不需要预知K

#### 优缺点
| 优点 | 缺点 |
|------|------|
| 树状图可视化层次 | O(n²)复杂度 |
| 不需预设K | 合并后不能撤销 |
| 任意形状（Single-link） | 大数据不适用 |

#### 参考代码
```python
from scipy.cluster.hierarchy import dendrogram, linkage, fcluster
from scipy.spatial.distance import pdist

Z = linkage(X_scaled, method='ward')
dendrogram(Z)
labels = fcluster(Z, t=n_clusters, criterion='maxclust')
```

### 13. DBSCAN 密度聚类

#### 数学原理
基于密度可达性：核心点（eps内≥min_samples）、边界点、噪声点

#### 关键参数
| 参数 | 含义 | 选择方法 |
|------|------|----------|
| eps | 邻域半径 | **k-distance图拐点** |
| min_samples | 最小样本数 | ≥ 维度+1，通常2*dim |

#### 适用场景
任意形状簇、含噪声数据、不需预知K

#### 优缺点
| 优点 | 缺点 |
|------|------|
| 任意形状 | 密度差异大效果差 |
| 自动识别噪声 | 高维数据效果差 |
| 不需预设K | eps/min_samples敏感 |

#### 参考代码
```python
from sklearn.cluster import DBSCAN
from sklearn.neighbors import NearestNeighbors

# k-distance图选eps
nn = NearestNeighbors(n_neighbors=min_samples)
nn.fit(X_scaled)
distances = np.sort(nn.kneighbors()[0][:, -1])
# 找拐点...

dbscan = DBSCAN(eps=eps_optimal, min_samples=min_samples)
labels = dbscan.fit_predict(X_scaled)
n_noise = (labels == -1).sum()
print(f"噪声点: {n_noise} ({n_noise/len(labels)*100:.1f}%)")
```

### 14. 决策树

#### 数学原理
- **分类（CART）**：Gini指数 $Gini = 1 - \sum p_k^2$
- **回归（CART）**：MSE = $\frac{1}{n}\sum(y_i - \bar{y})^2$

#### 剪枝
- **预剪枝**：max_depth、min_samples_split、min_samples_leaf
- **后剪枝**：代价复杂度剪枝（ccp_alpha）

#### 优缺点
| 优点 | 缺点 |
|------|------|
| 完全可解释 | 容易过拟合 |
| 不需要标准化 | 不稳定（小变化大不同） |
| 处理混合类型 | 偏向多值特征 |

#### 参考代码
```python
from sklearn.tree import DecisionTreeClassifier, plot_tree

tree = DecisionTreeClassifier(max_depth=4, min_samples_leaf=10, random_state=42)
tree.fit(X_train, y_train)
# 特征重要性
for name, imp in zip(X.columns, tree.feature_importances_):
    print(f"{name}: {imp:.4f}")
```

### 15. 随机森林

#### 数学原理
Bagging + 随机特征子集。Bootstrap抽样训练多棵树，每节点分裂从随机子集中选最优。

#### 关键优势
- OOB误差：无需单独验证集，36.8%样本未被抽中
- 特征重要性：基于 impurity reduction 或 permutation

#### 优缺点
| 优点 | 缺点 |
|------|------|
| 不易过拟合 | 模型文件大 |
| 高精度 | 训练较慢 |
| 无需标准化 | 外推能力弱 |

#### 参考代码
```python
from sklearn.ensemble import RandomForestClassifier
from sklearn.model_selection import cross_val_score

rf = RandomForestClassifier(n_estimators=200, max_depth=10,
                            min_samples_leaf=5, oob_score=True,
                            random_state=42)
rf.fit(X_train, y_train)
print(f"OOB Score: {rf.oob_score_:.4f}")
```

### 16. 支持向量机 (SVM)

#### 数学原理
**最大间隔分类器（硬间隔）**：
$$\min \frac{1}{2}\|w\|^2 \quad \text{s.t.} \quad y_i(w^Tx_i + b) \ge 1$$

**软间隔（引入松弛变量）**：
$$\min \frac{1}{2}\|w\|^2 + C\sum \xi_i$$

**核技巧**：$\phi(x_i)^T\phi(x_j) \to K(x_i, x_j)$

| 核 | K(x,y) | 适用 |
|----|--------|------|
| Linear | $x^T y$ | 线性可分 |
| RBF | $\exp(-\gamma\|x-y\|^2)$ | **默认首选** |
| Poly | $(x^Ty + r)^d$ | 已知多项式关系 |
| Sigmoid | $\tanh(\gamma x^Ty + r)$ | 神经网络 |

#### 适用场景
高维数据、小样本、复杂非线性边界

#### 优缺点
| 优点 | 缺点 |
|------|------|
| 高维表现好 | 大数据训练慢 |
| 核技巧处理非线性 | C和γ需仔细调参 |
| 最大间隔有理论保证 | 概率输出需校准 |

#### 参考代码
```python
from sklearn.svm import SVC
from sklearn.model_selection import GridSearchCV

svm = SVC(kernel='rbf', probability=True, random_state=42)
params = {'C': [0.1, 1, 10], 'gamma': [0.01, 0.1, 1]}
grid = GridSearchCV(svm, params, cv=5, scoring='accuracy')
grid.fit(X_train_scaled, y_train)
```

### 17. K近邻 (KNN)

#### 数学原理
$$\hat{y} = \frac{1}{K}\sum_{i \in N_K(x)} y_i$$（回归）

$$\hat{y} = \text{mode}\{y_i : i \in N_K(x)\}$$（分类）

#### 距离度量
Euclidean（连续）/ Manhattan / Minkowski / Hamming（类别）

#### 优缺点
| 优点 | 缺点 |
|------|------|
| 原理简单，无需训练 | 预测慢（需全量计算） |
| 无参数假设 | 维度灾难 |
| 天然多分类 | 特征需标准化 |

#### 关键：K值选择
- K=1：过拟合，对噪声敏感
- K=N：欠拟合，总是预测多数类
- 推荐K=√N，交叉验证确认

#### 参考代码
```python
from sklearn.neighbors import KNeighborsClassifier
from sklearn.preprocessing import StandardScaler

# 必须标准化
X_scaled = StandardScaler().fit_transform(X)

# K值选择
scores = []
for k in range(1, 31):
    knn = KNeighborsClassifier(n_neighbors=k)
    scores.append(cross_val_score(knn, X_scaled, y, cv=5).mean())
best_k = np.argmax(scores) + 1
```

### 18. 朴素贝叶斯

#### 数学原理
$$P(C_k|X) = \frac{P(C_k)\prod P(x_i|C_k)}{P(X)}$$

**条件独立性假设**：$P(X|C_k) = \prod P(x_i|C_k)$（"朴素"的来源）

| 变体 | 似然假设 | 适用 |
|------|----------|------|
| Gaussian NB | $P(x_i|C_k) \sim N(\mu_{ik}, \sigma^2_{ik})$ | 连续特征 |
| Multinomial NB | 多项式分布 | 文本分类 |
| Bernoulli NB | 伯努利分布 | 二元特征 |

#### 优缺点
| 优点 | 缺点 |
|------|------|
| 训练极快 | 条件独立假设常不成立 |
| 小样本也好用 | 概率估计不准（校准差） |
| 天然多分类 | 零概率问题（需拉普拉斯平滑） |

#### 适用场景
文本分类（垃圾邮件）、实时预测、基线模型

#### 参考代码
```python
from sklearn.naive_bayes import GaussianNB, MultinomialNB

gnb = GaussianNB()
gnb.fit(X_train, y_train)
y_prob = gnb.predict_proba(X_test)
```

### 19. 集成学习对比总结

#### 三大范式
| 范式 | 代表 | 方式 | 降低 |
|------|------|------|------|
| **Bagging** | 随机森林 | 并行独立 | 方差 |
| **Boosting** | XGB/LGB/CatBoost | 串行纠错 | 偏差 |
| **Stacking** | 多层集成 | 元学习器融合 | 两者 |

#### XGBoost vs LightGBM vs CatBoost
| 维度 | XGBoost | LightGBM | CatBoost |
|------|:---:|:---:|:---:|
| 树生长 | Level-wise | **Leaf-wise** | Symmetric |
| 类别特征 | 需编码 | 原生支持 | **最强** |
| 缺失值 | 自动 | 自动 | 自动 |

### 20. 信度分析 (Cronbach's α)

#### 数学原理
$$\alpha = \frac{k}{k-1}\left(1 - \frac{\sum \sigma^2_i}{\sigma^2_{total}}\right)$$

k=题项数，$\sigma^2_i$=第i题方差，$\sigma^2_{total}$=总分方差

#### 判断标准
| α值 | 信度 |
|-----|------|
| >0.9 | 极好 |
| 0.8-0.9 | 良好 |
| 0.7-0.8 | 可接受 |
| 0.6-0.7 | 需要修改 |
| <0.6 | 不合格 |

#### 参考代码
```python
import pandas as pd

def cronbach_alpha(df):
    """Cronbach's α信度系数"""
    k = df.shape[1]
    item_var = df.var(axis=0, ddof=1).sum()
    total_var = df.sum(axis=1).var(ddof=1)
    alpha = (k / (k - 1)) * (1 - item_var / total_var)
    return alpha

# CITC（修正题总相关）
def citc(df):
    total = df.sum(axis=1)
    return [df.drop(columns=[c]).sum(axis=1).corr(df[c]) for c in df.columns]
```

### 21. 效度分析 (KMO + Bartlett)

#### 数学原理
**KMO（Kaiser-Meyer-Olkin）**：
$$KMO = \frac{\sum_{i \neq j} r_{ij}^2}{\sum_{i \neq j} r_{ij}^2 + \sum_{i \neq j} a_{ij}^2}$$

$r_{ij}$=相关系数，$a_{ij}$=偏相关系数

| KMO | 适合度 |
|-----|--------|
| >0.9 | 极好 |
| 0.8-0.9 | 良好 |
| 0.7-0.8 | 中等 |
| 0.6-0.7 | 勉强 |
| <0.5 | 不适合 |

**Bartlett球形检验**：H₀: 相关矩阵=单位矩阵。p<0.05 → 拒绝 → 适合因子分析。

#### 参考代码
```python

kmo_all, kmo_model = calculate_kmo(df)
chi2, p_bartlett = calculate_bartlett_sphericity(df)
print(f"KMO: {kmo_model:.4f}")
print(f"Bartlett p值: {p_bartlett:.6f}")
```

### 22. 因子分析 (Factor Analysis)

#### 数学原理
因子分析是主成分分析的推广，用有限个不可观测的**公共因子**描述原始变量间的相关关系。

**正交因子模型**：$X = AF + \varepsilon$

- $X = (X_1, \ldots, X_p)^T$：标准化后的原始变量
- $F = (F_1, \ldots, F_m)^T$：公共因子（$m < p$，不可观测）
- $A = (a_{ij})_{p \times m}$：**因子载荷矩阵**，$a_{ij} = \text{Corr}(X_i, F_j)$（正交时）
- $\varepsilon$：特殊因子（变量独有部分）

**关键统计量**：
- **变量共同度**：$h_i^2 = \sum_{j=1}^{m} a_{ij}^2$，所有公因子对 $X_i$ 的总方差贡献（越接近1越好）
- **因子方差贡献**：$g_j^2 = \sum_{i=1}^{p} a_{ij}^2$，$F_j$ 对所有变量的总影响（衡量因子重要性）
- **因子得分函数**：$F_j = b_{j1}X_1 + \cdots + b_{jp}X_p$（用变量估计因子取值）

#### 因子载荷估计方法
| 方法 | 原理 | 特点 |
|------|------|------|
| **主成分法** | 用相关矩阵R的前m个特征值和特征向量构造A | 最常用，简单直观 |
| **主轴因子法** | 从约相关矩阵出发迭代估计 | 更关注共同度，对初值敏感 |
| **最大似然法** | 假设数据多元正态，最大化似然 | 需大样本（>1500），MATLAB `factoran`默认 |

#### 因子旋转
因子载荷矩阵**不唯一**——可通过正交旋转获得等价但更易解释的因子结构。

| 旋转方法 | 目标 | 效果 |
|----------|------|------|
| **方差最大法 (Varimax)** | 简化载荷矩阵的**列** | 每列载荷向0/1两极分化，因子含义清晰 |
| **四次方最大法 (Quartimax)** | 简化载荷矩阵的**行** | 每行只有一个高载荷，变量归属明确 |

旋转**不改变变量共同度**，只重新分配公因子的方差贡献。SPSS默认使用Varimax。

#### 因子分析步骤
1. 标准化数据 → 计算相关系数矩阵
2. **适用性检验**：KMO > 0.5，Bartlett Sig < 0.05
3. **提取公因子**：特征值 > 1 或累计贡献率 ≥ 60%（SPSS默认）
4. **因子旋转**：使载荷结构清晰，便于命名
5. **计算因子得分**：将因子表示为变量的线性组合
6. **应用**：综合评分排序、聚类、回归等后续分析

#### 适用场景
- 揭示隐藏的数据结构（哪些变量共享同一个潜在因素）
- 量表/指标体系的构念效度验证
- 客观确定多指标体系的权重（替代AHP等主观赋权）
- 将高维指标压缩为少数因子得分，用于后续聚类或回归
- 问卷调查的因素分析

#### 优缺点
| 优点 | 缺点 |
|------|------|
| 揭示潜在结构，因子可命名解释 | 对样本量要求较高（n ≥ 5p，最好n ≥ 100） |
| 通过旋转优化可解释性 | 因子含义可能因旋转方法不同而变化 |
| 客观赋权，避免主观偏差 | 假设线性关系，可能遗漏非线性结构 |
| 消除多重共线性 | 因子得分是估计值，存在不确定性 |

#### 与主成分分析(PCA)的本质区别
| 维度 | PCA | 因子分析 |
|------|-----|----------|
| 数学本质 | 变量线性组合 $Y = A^TX$ | 生成模型 $X = AF + \varepsilon$ |
| 目标 | 最大化方差，降维 | 解释潜在结构 |
| 方差 | 关注总方差（全部解释） | 只关注共同方差（分出特殊方差） |
| 旋转 | 不可旋转 | 可以且应该旋转 |
| 模型方向 | X → Y（正向） | F → X（反向） |

#### 参考代码
```python
from factor_analyzer import FactorAnalyzer
from factor_analyzer.factor_analyzer import calculate_kmo, calculate_bartlett_sphericity
from sklearn.preprocessing import StandardScaler
import numpy as np

# 1. 适用性检验
kmo_all, kmo_model = calculate_kmo(df)
chi2, p_value = calculate_bartlett_sphericity(df)
print(f"KMO: {kmo_model:.4f}, Bartlett p: {p_value:.6f}")

# 2. 确定因子个数（特征值>1准则）
fa_full = FactorAnalyzer(rotation=None)
fa_full.fit(df)
ev, _ = fa_full.get_eigenvalues()
n_factors = sum(ev > 1)
print(f"特征值>1的因子数: {n_factors}")

# 3. 因子分析 + Varimax旋转
fa = FactorAnalyzer(n_factors=n_factors, rotation='varimax')
fa.fit(df)

# 4. 因子载荷矩阵
loadings = pd.DataFrame(
    fa.loadings_,
    index=df.columns,
    columns=[f'F{i+1}' for i in range(n_factors)]
)
print("旋转后因子载荷:\n", loadings)

# 5. 计算因子得分
scores = fa.transform(df)

# 6. 综合得分（以方差贡献率为权重）
variance_ratio = fa.get_factor_variance()[0]  # 各因子方差贡献
weights = variance_ratio / variance_ratio.sum()
composite = scores @ weights
```

### 23. 中介效应分析

#### 数学原理
**Baron & Kenny三步法**：
1. X → Y（总效应c）：Y = cX + e₁，c必须显著
2. X → M（a路径）：M = aX + e₂，a必须显著
3. X + M → Y：Y = c'X + bM + e₃

中介效应 = a×b，总效应 = c = c' + a×b

**Bootstrap检验**（现代标准，替代Sobel）：
- 对样本做5000次重抽样
- 计算每次的a×b
- 95% CI不包含0 → 中介效应显著

#### 适用场景
检验变量间的间接影响路径

#### 参考代码
```python
import statsmodels.api as sm

# Step 1: X → Y
m1 = sm.OLS(y, sm.add_constant(x)).fit()
c = m1.params['x']

# Step 2: X → M
m2 = sm.OLS(m, sm.add_constant(x)).fit()
a = m2.params['x']

# Step 3: X + M → Y
X_step3 = sm.add_constant(pd.DataFrame({'x': x, 'm': m}))
m3 = sm.OLS(y, X_step3).fit()
b = m3.params['m']
c_prime = m3.params['x']

indirect = a * b
direct = c_prime
total = c
print(f"总效应(c)={total:.4f}, 直接效应(c')={direct:.4f}, 间接效应(a×b)={indirect:.4f}")

# Bootstrap置信区间
import numpy as np
n_boot = 5000
ab_boot = []
n = len(y)
idx = np.arange(n)
for _ in range(n_boot):
    boot_idx = np.random.choice(idx, n, replace=True)
    # 重抽样后重算a和b...
    ab_boot.append(a_boot * b_boot)

ci_low = np.percentile(ab_boot, 2.5)
ci_high = np.percentile(ab_boot, 97.5)
significant = ci_low * ci_high > 0
print(f"95% CI: [{ci_low:.4f}, {ci_high:.4f}], 显著: {significant}")
```

## 方法选型速查

### 按问题类型
| 问题 | 方法 | 备注 |
|------|------|------|
| 两连续变量关系 | Pearson | 需正态 |
| 两分类变量关系 | 卡方检验 | 需期望频数≥5 |
| 两组均值差 | T检验 | 先验方差齐性 |
| 多组均值差 | ANOVA | 显著后做事后比较 |
| 非正态组间比较 | Mann-Whitney/Kruskal-Wallis | 非参数 |
| 连续Y预测 | 线性/Ridge/Lasso | 看VIF决定 |

| 无监督分组 | K-Means/DBSCAN | 看簇形状 |
| 高精度预测 | CatBoost/XGBoost | 结构化数据 |
