17. 遗传算法（Genetic Algorithm, GA）

聚类分析 →

一、遗传算法概述

遗传算法（Genetic Algorithm, GA）

遗传算法是通过模拟生物进化过程来完成优化搜索的仿生类算法。它将"优胜劣汰、适者生存"的生物进化原理引入优化参数形成的编码群体中，按适应度函数通过选择（复制）、交叉和变异对个体进行筛选，使适应度高的个体被保留并产生更优的后代。算法简单、可并行处理、能收敛到全局最优解。

1.1 三种基本遗传操作

操作生物学类比实现方式作用
选择（复制）适者生存按适应度比例随机选择（轮盘赌）保留优良个体，淘汰劣质个体
交叉繁殖（基因重组）两个染色体交换部分基因片段产生新的个体，在解空间中搜索
变异基因突变以很小概率随机改变某位基因增加种群多样性，防止陷入局部最优

1.2 算法流程图

遗传算法流程：
初始化种群 → 计算适应度 → 循环：选择 → 交叉 → 变异 → 计算适应度 → 检查终止条件 → 输出最优解

二、基本运行参数

参数符号推荐范围说明
种群大小$M$20 ~ 100太小采样不足，太大计算量大收敛慢
终止代数$G$100 ~ 500进化多少代后停止
交叉概率$P_c$0.4 ~ 0.99太大破坏优良个体，太小搜索停滞
变异概率$P_m$0.0001 ~ 0.1太小难产生新模式，太大退化为随机搜索

2.1 遗传算法应用步骤（七步法）

Step 1：确定决策变量及约束条件（表现型 $X$ 和解空间）
Step 2：建立优化模型（目标函数类型和数学描述）
Step 3：确定染色体编码方法（基因型 $x$ 和搜索空间）
Step 4：确定解码方法（基因型 → 表现型的映射）
Step 5：确定个体适应度的量化评价方法（目标函数值 → 适应度）
Step 6：设计遗传算子（选择、交叉、变异的具体操作方法）
Step 7：确定运行参数（$M, G, P_c, P_m$ 等）

三、收敛性分析与参数影响

3.1 各因素对收敛性的影响

因素太小太大
种群规模 $M$采样点不足，算法性能差增加计算量，收敛缓慢
选择操作收敛速度慢配合最优保存策略，以概率1收敛于全局最优
交叉概率 $P_c$搜索停滞，不收敛优良个体被快速破坏
变异概率 $P_m$难产生新模式算法退化为随机搜索

3.2 遗传算法的本质

遗传算法本质上是对染色体模式所进行的一系列运算：选择算子将优良模式遗传到下一代，交叉算子进行模式重组，变异算子进行模式突变。通过这三类操作，模式逐步向更优方向进化，最终得到问题的最优解。

3.3 遗传欺骗问题

遗传欺骗（Deception）

在进化过程中，有时会产生一些超常个体，因竞争力过强而控制了选择过程，导致种群多样性迅速丧失，算法收敛到局部最优解而非全局最优解。

四、遗传算法的改进

4.1 编码方式的改进

编码方式优点缺点
二进制编码编码解码简单，交叉变异易实现精度高时编码串过长，搜索空间急剧扩大
格雷编码克服二进制编码的不连续问题编解码稍复杂
浮点数编码改善计算复杂性，适合高精度连续优化遗传算子设计更复杂

4.2 遗传算子的改进

改进方法
排序选择按适应度降序排列，设计概率分配表分配选择概率，再用轮盘赌产生下一代
均匀交叉产生与编码长度相同的二进制屏蔽字，$W_i=0$ 继承A基因，$W_i=1$ 交换AB基因
逆序变异随机选择两个断点，将断点间的基因序列反转

4.3 自适应遗传算法

Srinivas 等人提出：$P_c$ 和 $P_m$ 随适应度自动调整——适应度趋于一致时增大 $P_c$ 和 $P_m$，适应度分散时减小。对高于平均适应度的个体采用较低的 $P_c$ 和 $P_m$（保护优良个体），低于平均的采用较高值（淘汰劣质个体）。

五、MATLAB 遗传算法工具箱

5.1 基本语法

x = ga(fitnessfcn, nvars) % 最简形式 x = ga(fitnessfcn, nvars, A, b) % 线性不等式约束 x = ga(fitnessfcn, nvars, A, b, Aeq, beq) % 线性等式约束 x = ga(fitnessfcn, nvars, A, b, Aeq, beq, LB, UB) % 变量上下界 x = ga(fitnessfcn, nvars, A, b, Aeq, beq, LB, UB, nonlcon, intcon, options)

5.2 参数设置

options = gaoptimset('PropertyName1', PropertyValue1, ... 'PropertyName2', PropertyValue2, ...); [x, fval, reason, output, final_pop] = ga(@fitnessfun, nvars, options); % 将上次的最终种群作为下次的初始种群 options = gaoptimset('InitialPopulation', final_pop);

5.3 图形界面

在 MATLAB 命令窗口输入 optimtool，单目标选 ga，多目标选 gamultiobj，即可通过图形界面设置参数并运行。

5.4 示例：求函数最大值

问题：求 $f(x) = 10\sin(5x) + 7\cos(4x)$ 在 $x \in [0, 10]$ 的最大值。

function [y] = fitness1(x) y = x + 10*sin(x) + 7*cos(x); % ga 默认求最小值 end

将 $x$ 用 10 位二进制编码，分辨率 $\approx 0.01$。注意：初始种群越大，优化效果通常越明显。

六、优缺点总结

6.1 优点

优点说明
全局搜索能力强基于种群搜索，不易陷入局部最优（相比梯度下降法）
无需梯度信息不需要目标函数可微、连续，适合黑箱优化
可并行处理种群中各个体可独立评估，天然适合并行计算
通用性强对问题类型（连续/离散/混合）几乎无限制
易于与其它方法结合可与局部搜索（memetic算法）、多目标优化（NSGA-II）等结合

6.2 局限性

局限性说明
收敛速度较慢相比梯度类方法，需要更多函数评估次数
参数调优困难$M, G, P_c, P_m$ 等参数对性能影响大，需反复试调
缺乏严格收敛性保证启发式算法，不能保证一定找到全局最优
编码方式影响性能不恰当的编码会导致搜索效率低下
竞赛提示：遗传算法在竞赛中适用于复杂约束优化、组合优化、多目标优化等问题，特别是传统优化方法（LP/NLP）难以处理的非凸、离散问题。建议使用 MATLAB 自带的 ga 函数或 Python 的 geneticalgorithm 库。
