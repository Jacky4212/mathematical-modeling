10. 灰色系统理论及其应用

模拟退火算法 →

一、灰色系统概论

1.1 什么是灰色系统

定义

部分信息已知、部分信息未知的系统称为灰色系统。与之对应：信息完全已知为白色系统，信息完全未知为黑色系统。客观世界中绝对的白色或黑色系统很少，灰色系统大量存在。

系统类型特征示例
白色系统信息完全已知，因素间关系确定牛顿力学系统（F=ma）
灰色系统部分信息已知，部分未知社会经济系统、农业系统、生态系统
黑色系统信息完全未知完全未知的内部机理

1.2 灰色系统理论的创新

与传统统计方法的区别：
• 回归分析要求大样本且分布规律 → 灰色系统对样本量无要求，不需要典型分布
• 概率统计是经验性的统计规律 → 灰色系统是现实性的生成律
• 核心思想：将随机量看作灰变量 → 数据生成（AGO）→ 发现规律 → 建模

二、灰色关联分析（GRA）

2.1 数据变换（无量纲化）

为保证建模质量，需对原始数据消除量纲并使其具有可比性。

变换方法公式
初值化变换$y(k) = x(k)/x(1)$
均值化变换$y(k) = x(k)/\bar{x}$
区间值化变换$y(k) = \frac{x(k) - \min x}{\max x - \min x}$
百分比变换$y(k) = x(k)/\max x(k)$
归一化变换$y(k) = x(k)/x_0$（x₀>0为参考值）

2.2 关联系数与关联度

关联系数公式
$$\xi_i(k) = \frac{\min_s\min_t|x_0(t)-x_s(t)| + \rho \cdot \max_s\max_t|x_0(t)-x_s(t)|}{|x_0(k)-x_i(k)| + \rho \cdot \max_s\max_t|x_0(t)-x_s(t)|}$$

其中 $\rho \in [0,1]$ 为分辨系数（通常取0.5），$\rho$越大分辨率越大。

关联度
$$r_i = \frac{1}{n}\sum_{k=1}^{n}\xi_i(k)$$

关联度越大，比较数列与参考数列的关联越紧密。

2.3 应用示例：铅球运动员因素分析

案例：对女子铅球运动员1982-1986年成绩与16项素质指标做关联分析。
以铅球专项成绩为参考数列，16项素质为比较数列，计算关联度排序：
全蹲(0.933) > 3kg滑步(0.896) > 高翻(0.855) > 4kg原地(0.854) > 挺举(0.847) > 立定跳远(0.776) > 30米(0.745) > 100米(0.726)
结论：训练应着重这八项指标，减少盲目性。

三、优势分析

当参考数列不止一个、比较因素也不止一个时，构建关联矩阵 R = (rᵢⱼ)。

  • 优势子因素：某一列元素均大于其他列 → 该子因素对所有母因素影响最大
  • 优势母因素：某一行元素均大于其他行 → 该母因素最易受所有子因素影响
  • 潜在优势分析：通过稀疏化关联矩阵（将小元素近似为零），识别潜在优势因素
案例：某地区5种投资（子因素）对6种收入（母因素）的关联分析得出：
• 交通投资对国民收入影响最大（r₁₅=0.936）
• 科技对工业影响最大（r₂₄=0.885），但对农业影响最小（r₃₄=0.577）
• 农业需要多方面配合（固定资产投资r₃₁=0.891，工业投资r₃₂=0.858）

四、生成数

4.1 累加生成（AGO）

1次累加生成（1-AGO）
$$x^{(1)}(k) = \sum_{i=1}^{k} x^{(0)}(i), \quad k = 1,2,...,n$$

累加生成可使非负的摆动数列或任意无规律数列转化为非减的单增数列，削弱随机性，发现内在规律。r次累加记作r-AGO。

4.2 累减生成（IAGO）

$$x^{(r-1)}(k) = x^{(r)}(k) - x^{(r)}(k-1), \quad k = 2,3,...,n$$

用于还原生成数列。AGO和IAGO互为逆运算。

4.3 紧邻均值生成

$$z^{(1)}(k) = 0.5x^{(1)}(k) + 0.5x^{(1)}(k-1), \quad k = 2,3,...,n$$

等权邻值生成数，是构造GM模型的关键中间变量。

五、GM(1,1)模型（核心预测模型）

5.1 模型定义

GM(1,1) — 1阶1变量灰色模型

最经典、应用最广泛的灰色模型。适用于具有较强指数规律的序列，只需4个以上数据点即可建模。

灰微分方程：

$$x^{(0)}(k) + a z^{(1)}(k) = b$$

其中 a 为发展系数，b 为灰作用量。

白化微分方程：

$$\frac{dx^{(1)}}{dt} + a x^{(1)} = b$$

5.2 参数估计（最小二乘法）

$$\hat{u} = (B^T B)^{-1} B^T Y, \quad \hat{u} = [\hat{a}, \hat{b}]^T$$

其中 $Y = [x^{(0)}(2), x^{(0)}(3), ..., x^{(0)}(n)]^T$，$B = \begin{bmatrix} -z^{(1)}(2) & 1 \\ -z^{(1)}(3) & 1 \\ \vdots & \vdots \\ -z^{(1)}(n) & 1 \end{bmatrix}$

5.3 时间响应函数（预测公式）

$$\hat{x}^{(1)}(k+1) = \left(x^{(0)}(1) - \frac{b}{a}\right) e^{-ak} + \frac{b}{a}$$
$$\hat{x}^{(0)}(k+1) = \hat{x}^{(1)}(k+1) - \hat{x}^{(1)}(k) \quad \text{（累减还原）}$$

六、灰色预测的标准步骤

步骤一：级比检验

$$\lambda(k) = \frac{x^{(0)}(k-1)}{x^{(0)}(k)}, \quad k = 2,3,...,n$$

所有 $\lambda(k)$ 必须落入可容覆盖 $\Theta = (e^{-2/(n+1)}, e^{2/(n+2)})$ 内。若不满足，需对数列做平移变换 $y^{(0)}(k) = x^{(0)}(k) + c$。

步骤二：建立GM(1,1)模型

1-AGO → 构造B和Y → 最小二乘求a,b → 得时间响应式 → 计算预测值。

步骤三：模型检验

检验方法标准精度等级
残差检验相对误差 < 0.01一级（优）
相对误差 < 0.05二级（良）
相对误差 < 0.10三级（合格）
相对误差 < 0.20四级（勉强）
$$\varepsilon(k) = \frac{x^{(0)}(k) - \hat{x}^{(0)}(k)}{x^{(0)}(k)}$$

级比偏差检验：$\rho(k) = 1 - \frac{1-0.5a}{1+0.5a}\lambda(k)$，要求 < 0.1。

均方差比检验：$C = S_2/S_1$（残差标准差/原始数据标准差），C越小越好。

步骤四：预测预报

由模型得到指定时区内的预测值，结合实际给出预测预报。

七、GM(1,1)计算实例

7.1 城市交通噪声预测

数据

北方某城市1986-1992年道路交通噪声平均声级[dB(A)]：71.1, 72.4, 72.4, 72.1, 71.4, 72.0, 71.6

求解过程：
1. 级比检验：λ ∈ [0.982, 1.0098]，均落入可容覆盖 → 可建模
2. 1-AGO：x⁽¹⁾ = (71.1, 143.5, 215.9, 288, 359.4, 431.4, 503)
3. 最小二乘：â = 0.0023, b̂ = 72.6573
4. 时间响应式：x̂⁽¹⁾(k+1) = −30929e⁻⁰·⁰⁰²³ᵏ + 31000
5. 模型还原值：71.1, 72.4, 72.2, 72.1, 71.9, 71.7, 71.6
6. 精度：相对误差最大0.7%，模型精度较高

7.2 MATLAB实现

clc,clear x0=[71.1 72.4 72.4 72.1 71.4 72.0 71.6]'; % 原始数据（列向量） n=length(x0); lamda=x0(1:n-1)./x0(2:n) % 级比检验 x1=cumsum(x0); % 1-AGO B=[-0.5*(x1(1:n-1)+x1(2:n)),ones(n-1,1)]; Y=x0(2:n); u=B\Y % 参数估计 [a; b] % 预测 x1_hat=(x0(1)-u(2)/u(1))*exp(-u(1)*(0:n-1))+u(2)/u(1); x0_hat=[x0(1),diff(x1_hat)] % 还原 epsilon=x0'-x0_hat % 残差 delta=abs(epsilon./x0') % 相对误差

八、其他灰色模型

8.1 GM(1,N) — 多变量灰色模型

用途

分析N个变量（N-1个子因素）对1个主因素的综合影响。1阶N变量模型。

$$x_1^{(0)}(k) + a z_1^{(1)}(k) = \sum_{i=2}^{N} b_i x_i^{(1)}(k)$$

参数向量 $\hat{u} = [\hat{a}, \hat{b}_2, ..., \hat{b}_N]^T$ 同样用最小二乘法估计。

8.2 Verhulst模型 — S形过程预测

适用场景

描述具有饱和状态的S形过程，如人口预测、生物生长、产品经济寿命、交通事故死亡人数等。

$$x^{(0)} + a z^{(1)} = b(z^{(1)})^2$$
$$\hat{x}^{(1)}(k+1) = \frac{a x^{(0)}(1)}{b x^{(0)}(1) + (a - b x^{(0)}(1)) e^{ak}}$$
Verhulst vs GM(1,1)：1990-2003年交通事故死亡人数预测中，Verhulst模型平均相对误差3.74%，GM(1,1)为3.6%，精度相当。但Verhulst更适合S形饱和趋势的长期预测。

8.3 GM(2,1) — 二阶灰色模型

$$\alpha^{(1)}x^{(0)}(k) + a_1 x^{(0)}(k) + a_2 z^{(1)}(k) = b$$

适用于非单调的摆动发展序列，是GM(1,1)的高阶扩展。

8.4 灾变预测

不预测数据本身的数值，而是预测异常值出现的时间。例如：给定降雨量≤320mm为旱灾阈值，构建下限灾变数列的时刻序列，用GM(1,1)预测下一次旱灾的发生年份。

九、经典案例：SARS疫情经济影响评估

9.1 问题

2003年SARS疫情对某市商品零售业、旅游业和综合服务业造成影响。已知1997-2002年数据，需评估2003年各月损失。

9.2 方法

用GM(1,1)由1997-2002年年平均值预测2003年月均值 → 由历史月份比例 $u_i$ 分解到各月 → 与实际值比较得出损失。

9.3 结果

行业最严重月份预测损失恢复时间
商品零售业4-6月~60.1亿元8月基本恢复
旅游业4-7月~162万人（约16.2亿美元）年底基本恢复
综合服务业5-8月~70亿元部分行业影响较小

模型验证：零售业4-6月预测损失60.1亿元与专家估计的62亿元基本一致。

十、数学建模竞赛中的应用策略

灰色系统的竞赛优势场景：
1. 小样本预测：只有4-10个数据点时，回归和时序模型不可靠，GM(1,1)是最佳选择
2. 因素关联分析：数据量不足以做回归分析时，用GRA找出主要影响因素
3. 贫信息系统：信息大量缺乏或紊乱的情况下构建预测评估模型
4. 灾变/异常预测：不预测数值而预测事件发生时间
5. S形趋势：饱和增长趋势用Verhulst模型优于GM(1,1)

Python实现

import numpy as np def gm11(x0): """GM(1,1)灰色预测模型""" n = len(x0) # 级比检验 lamda = x0[:-1] / x0[1:] if not np.all((lamda > np.exp(-2/(n+1))) & (lamda < np.exp(2/(n+2)))): print('警告：级比不在可容覆盖内，需平移变换') # 1-AGO x1 = np.cumsum(x0) # 紧邻均值 z1 = 0.5 * (x1[1:] + x1[:-1]) # 最小二乘估计 B = np.column_stack([-z1, np.ones(n-1)]) Y = x0[1:] u = np.linalg.inv(B.T @ B) @ B.T @ Y a, b = u[0], u[1] # 预测 k = np.arange(n) x1_hat = (x0[0] - b/a) * np.exp(-a*k) + b/a x0_hat = np.diff(x1_hat, prepend=x0[0]) # 检验 epsilon = x0 - x0_hat[:n] delta = np.abs(epsilon / x0) return x0_hat, delta, (a, b) # === 灰色关联分析 === def grey_relational(x0, xi, rho=0.5): """x0: 参考数列, xi: 比较数列(2D array), rho: 分辨系数""" # 初值化 x0_norm = x0 / x0[0] xi_norm = xi / xi[:, [0]] diff = np.abs(xi_norm - x0_norm) mn, mx = diff.min(), diff.max() ksi = (mn + rho*mx) / (diff + rho*mx) r = ksi.mean(axis=1) return r
