6. 时间序列分析

图与网络 →

1. 时间序列概述

📖 定义

时间序列 (Time Series)：按时间顺序记录的一组数据 $\{Y_t, t=1,2,\dots,n\}$。观测时间 $t$ 可以是年份、季度、月份或任何时间单位。

1.1 时间序列的四种成分

成分英文特征典型场景
趋势 TTrend持续向上或向下的长期变动GDP增长、人口增长
季节变动 SSeasonal一年内重复出现的周期性波动空调销量（夏高冬低）
循环波动 CCyclical非固定长度的周期性变动（>1年）经济周期（繁荣→衰退）
不规则波动 IIrregular随机波动，不可预测自然灾害、突发事件
🔑 乘法模型：$Y_i = T_i \times S_i \times C_i \times I_i$ — 四种成分相互影响，常用且符合实际
加法模型：$Y_i = T_i + S_i + C_i + I_i$ — 四种成分相互独立
平稳序列 (Stationary Series)：不含趋势的序列，波动主要来自随机成分。均值不随时间变化。

2. 平稳时间序列理论 必须掌握

2.1 严平稳 vs 宽平稳

严平稳 (Strict Stationarity)

序列所有统计性质不随时间推移而变化。任意 $m$ 个时刻的联合分布与平移 $\tau$ 后的联合分布完全相同：

$F(x_{t_1}, \dots, x_{t_m}) = F(x_{t_1+\tau}, \dots, x_{t_m+\tau})$

⚠️ 条件严格，实际数据太难达到，竞赛用宽平稳即可。

宽平稳 (Weak Stationarity) 竞赛最常用

只保证序列低阶矩的平稳性，满足三个条件：

  1. $E(X_t) = \mu$（常数均值，与 $t$ 无关）
  2. $E(X_t^2) < \infty$（二阶矩存在，方差有限）
  3. $\gamma(t, s) = \gamma(k)$，$k = |t-s|$（自协方差仅依赖于时间间隔 $k$，与起止点无关）

推论：宽平稳序列的均值和方差为常数，自相关系数 $\rho_k = \gamma_k / \gamma_0$ 只依赖于滞后阶数 $k$。

2.2 平稳性的意义

  1. 减少变量个数：$n$ 个观测 → 仅需估计 $\mu$ 和 $\gamma_k$（$k=0,1,\dots$），大幅增加样本容量
  2. 估计稳定：历史数据的参数估计可直接用于未来预测
  3. 简化难度 + 提高精度

2.3 平稳性检验方法 必考

方法原理判定标准
时序图检验 肉眼观察序列是否在常数值附近随机波动 波动范围有界、无明显趋势和周期 → 平稳
自相关图检验 平稳序列具有短期相关性：ACF 随延迟 $k$ 增大快速衰减到 0 ACF 快速落入 2 倍标准差内 → 平稳
ACF 缓慢衰减或不衰减 → 非平稳
ADF 单位根检验 $H_0$: 存在单位根（非平稳） p-value < 0.05 → 拒绝 $H_0$ → 平稳
📌 非平稳序列的典型特征：
1. 时序图有明显趋势或周期性 → 非平稳
2. ACF 图衰减缓慢，长时间不落入 2 倍标准差 → 非平稳
3. 平稳序列的 ACF 通常在 $k \leq 3$ 内就衰减到 0 附近

3. 纯随机性检验（白噪声检验）建模前必做

3.1 白噪声序列

纯随机序列（白噪声 White Noise）

序列值彼此之间没有任何相依性——没有记忆，过去行为对将来发展没有影响。满足：

$E(X_t) = \mu,\quad Var(X_t) = \sigma^2,\quad \gamma(k) = 0 \; (\forall k \neq 0)$

3.2 白噪声检验的两个时机

  1. 建模前：检验序列是否有分析价值。若为白噪声 → 无规律可提取 → 终止分析
  2. 建模后：检验残差是否为白噪声。若残差是白噪声 → 有用信息提取充分，模型成功；否则 → 信息未提取完 → 重新建模

3.3 检验统计量

$$\begin{aligned} Q_{BP} &= n \sum_{k=1}^{m} \hat{\rho}_k^2 \sim \chi^2(m) \quad &\text{(Box-Pierce)}\\ Q_{LB} &= n(n+2) \sum_{k=1}^{m} \frac{\hat{\rho}_k^2}{n-k} \sim \chi^2(m) \quad &\text{(Ljung-Box, 小样本矫正)} \end{aligned}$$

其中 $n$ 为观测期数，$m$ 为指定延迟期数，$\hat{\rho}_k$ 为延迟 $k$ 阶的样本自相关系数。

假设检验框架：
$H_0: \rho_1 = \rho_2 = \dots = \rho_m = 0$（序列为纯随机/白噪声）
$H_1:$ 至少存在某个 $\rho_k \neq 0$（序列非白噪声）
判定：$Q$ 统计量的 p-value < $\alpha$ → 拒绝 $H_0$ → 序列非白噪声，有分析价值 ✓

4. 平稳序列的预测方法（平滑法）

📌 适用条件：序列不含趋势和季节变动，波动主要来自随机成分。通过平滑消除随机波动。

4.1 移动平均法 (Moving Average)

选择固定移动间隔 $k$，将最近 $k$ 期的简单平均作为下一期预测值：

$$F_{t+1} = \frac{Y_t + Y_{t-1} + \dots + Y_{t-k+1}}{k}$$

特点：

  • 每期数据等权重（均为 $1/k$），旧数据 $k$ 期后完全丢弃
  • 只适合较为平稳的序列预测
  • 选择 $k$ 的方法：试验不同 $k$，选使 MSE 最小者

4.2 简单指数平滑法 (Simple Exponential Smoothing)

核心思想：近期数据权重大，远期数据权重呈指数衰减。

$$F_{t+1} = \alpha Y_t + (1-\alpha) F_t$$

展开可得：$F_{t+1} = \alpha Y_t + \alpha(1-\alpha)Y_{t-1} + \alpha(1-\alpha)^2 Y_{t-2} + \dots$

$\alpha$ 取值效果适用场景
$\alpha \to 1$侧重近期值，响应快但波动大序列波动大、需要快速跟踪
$\alpha \to 0$侧重历史值，更平滑但滞后随机波动较大时

选择 $\alpha$ 的方法：试验多个值，选 MSE 最小的。

🔑 移动平均 vs 指数平滑
移动平均：等权重、固定窗口、旧数据突然丢弃 → 简单直观
指数平滑：指数衰减权重、无限记忆、平滑过渡 → 更符合"近期影响更大"的实际
竞赛建议：两种都试，选 MSE 更小的。

5. 趋势序列的预测

5.1 线性趋势预测

时间序列按固定斜率增长或下降时，拟合线性趋势方程：

$\hat{Y}_t = b_0 + b_1 t$

用最小二乘法估计 $b_0$ 和 $b_1$。预测时直接将未来时刻 $t$ 代入方程。

5.2 Holt 双参数指数平滑 有趋势时必用

简单指数平滑在序列有趋势时会系统滞后。Holt 模型用两个平滑系数同时跟踪水平和趋势：

$$\begin{aligned} L_t &= \alpha Y_t + (1-\alpha)(L_{t-1} + b_{t-1}) \quad &\text{（水平方程）}\\ b_t &= \gamma(L_t - L_{t-1}) + (1-\gamma) b_{t-1} \quad &\text{（趋势方程）}\\ F_{t+m} &= L_t + b_t \cdot m \quad &\text{（$m$ 步向前预测）} \end{aligned}$$

$\alpha \in (0,1)$ 控制水平平滑，$\gamma \in (0,1)$ 控制趋势平滑。两个参数都通过最小化 MSE 确定。

5.3 非线性趋势预测

曲线类型方程适用场景
指数曲线$Y_t = b_0 e^{b_1 t}$几何级数增长/衰减（可线性化后 OLS）
修正指数曲线$Y_t = K + b_0 b_1^t$有渐近线的增长
生长曲线 (Logistic)$Y_t = \frac{K}{1 + a e^{-rt}}$S 型增长，有上限（市场饱和）
多阶多项式$Y_t = b_0 + b_1 t + \dots + b_k t^k$$k-1$ 个拐点的复杂形态

5.4 趋势模型选择策略

  1. 绘制时序图，观察大体形态（直线/指数/S型/多个拐点）
  2. 选择对应的模型族，用 OLS（或线性化后 OLS）拟合
  3. 比较各模型的 MSE 和 $R^2$
  4. 注意：模型不是越复杂越好 — 多项式阶数 $k$ 过高会导致过拟合

6. 多成分序列的分解预测 (Decomposition)

📌 适用场景：序列同时包含趋势 + 季节 + 循环多种成分。分解法是处理此类序列的古典经典方法，易于理解，结果易于解释。

6.1 乘法模型分解步骤

  1. 确定并分离季节成分：计算季节指数 (Seasonal Index)，将每个观测值除以对应的季节指数，消除季节性
  2. 建立趋势预测模型：对消除季节后的序列拟合线性趋势方程 $\hat{Y}_t = b_0 + b_1 t$
  3. 计算最终预测值：将趋势预测值乘以对应季节的季节指数

6.2 季节指数的计算

季节指数反映某季节相对于全年平均水平的偏离程度：

  • 季节指数 > 1 → 该季节高于平均水平（旺季）
  • 季节指数 < 1 → 该季节低于平均水平（淡季）
  • 所有季节指数之和 = 季节数（如季度数据 = 4，月度数据 = 12）

6.3 加法模型 vs 乘法模型

加法模型乘法模型
形式$Y = T + S + C + I$$Y = T \times S \times C \times I$
季节波动幅度恒定，不随趋势变化幅度随趋势增大而增大
适用序列波动幅度稳定时序列波动幅度与水平成正比
分解操作$Y - S$ 消除季节$Y / S$ 消除季节

7. Box-Jenkins 方法：AR / MA / ARMA 模型 竞赛核心

📌 核心思想：不直接拟合时间 $t$ 的函数，而是利用序列自身的自相关结构建模。前提是序列平稳且非白噪声。

7.1 AR(p) — 自回归模型

p 阶自回归模型 AR(p)
$$X_t = \phi_0 + \phi_1 X_{t-1} + \phi_2 X_{t-2} + \dots + \phi_p X_{t-p} + \varepsilon_t$$

其中 $\varepsilon_t \sim WN(0, \sigma^2)$ 为白噪声，$\phi_1,\dots,\phi_p$ 为自回归系数。

当 $\phi_0 = 0$ 时，称为中心化 AR(p) 模型。

延迟算子形式：$\Phi(B) X_t = \varepsilon_t$，其中 $\Phi(B) = 1 - \phi_1 B - \dots - \phi_p B^p$（$B$ 为延迟算子：$B X_t = X_{t-1}$）。

7.2 MA(q) — 移动平均模型

q 阶移动平均模型 MA(q)
$$X_t = \mu + \varepsilon_t + \theta_1 \varepsilon_{t-1} + \theta_2 \varepsilon_{t-2} + \dots + \theta_q \varepsilon_{t-q}$$

式中 $\varepsilon_t$ 是白噪声序列，$\theta_1,\dots,\theta_q$ 为移动平均系数。

MA(q) 模型表明 $X_t$ 是当前和过去 $q$ 期随机误差的加权和。

7.3 ARMA(p,q) — 自回归移动平均模型

ARMA(p,q) 模型
$$X_t = \phi_0 + \sum_{i=1}^{p} \phi_i X_{t-i} + \varepsilon_t + \sum_{j=1}^{q} \theta_j \varepsilon_{t-j}$$

AR 部分捕获序列自身的记忆效应，MA 部分捕获随机扰动的持续影响。

7.4 模型族总览

模型全称适用条件参数
AR(p)自回归平稳序列，当前值与过去 $p$ 期相关$p$: 自回归阶数
MA(q)移动平均平稳序列，当前值与过去 $q$ 期误差相关$q$: 移动平均阶数
ARMA(p,q)自回归移动平均平稳非白噪声序列$(p, q)$
ARIMA(p,d,q)差分 ARMA$d$ 阶差分后平稳$(p, d, q)$
SARIMA季节 ARIMA含季节周期的非平稳序列$(p,d,q)(P,D,Q)_s$

7.5 AR/MA 模型的统计性质 定阶关键

模型ACF（自相关函数）PACF（偏自相关函数）
AR(p)拖尾（指数衰减/振荡收敛到 0）截尾于 $p$ 阶（$k>p$ 后为 0）
MA(q)截尾于 $q$ 阶（$k>q$ 后为 0）拖尾（指数衰减/振荡收敛到 0）
ARMA(p,q)拖尾拖尾
📌 定阶口诀：
ACF 截尾 + PACF 拖尾 → MA(q)，截尾阶数 = $q$
ACF 拖尾 + PACF 截尾 → AR(p)，截尾阶数 = $p$
两者都拖尾 → ARMA(p,q)，需用 AIC/BIC 自动定阶

7.6 ARIMA 建模完整流程

  1. 平稳性检验：时序图 + ACF 图 + ADF 检验。若不平稳，进行差分（ARIMA 的 $d$）
  2. 白噪声检验：Ljung-Box 检验。若为白噪声 → 无规律可提取 → 终止
  3. 模型识别（定阶）：观察 ACF/PACF 截尾拖尾模式确定 $p,q$；或遍历 $(p,q)$ 组合选 AIC/BIC 最小者
  4. 参数估计：最大似然估计 (MLE) 或条件最小二乘 (CLS)
  5. 模型诊断：残差白噪声检验（Ljung-Box p>0.05）+ 残差正态性检验（Q-Q 图）
  6. 预测：点预测 + 置信区间预测

8. 预测方法的选择与评估

8.1 方法选择路线

8.2 预测误差度量

$$\begin{aligned} \text{ME} &= \frac{1}{n}\sum (Y_t - F_t) \quad &\text{（平均误差，反映偏差方向）}\\ \text{MAD} &= \frac{1}{n}\sum |Y_t - F_t| \quad &\text{（平均绝对误差）}\\ \text{MSE} &= \frac{1}{n}\sum (Y_t - F_t)^2 \quad &\text{（均方误差，竞赛首选）}\\ \text{MAPE} &= \frac{100\%}{n}\sum \left|\frac{Y_t - F_t}{Y_t}\right| \quad &\text{（适合跨量纲比较）} \end{aligned}$$

9.1 平稳性检验 + 白噪声检验

from statsmodels.tsa.stattools import adfuller from statsmodels.stats.diagnostic import acorr_ljungbox from statsmodels.graphics.tsaplots import plot_acf, plot_pacf # ADF 平稳性检验 result = adfuller(df['value']) print('ADF statistic: {:.4f}, p-value: {:.4f}'.format(result[0], result[1])) # Ljung-Box 白噪声检验 lb_result = acorr_ljungbox(df['value'], lags=[6, 12]) print(lb_result) # p<0.05 → 非白噪声 ✓ # 绘制 ACF / PACF 图 plot_acf(df['value'], lags=20) plot_pacf(df['value'], lags=20)

9.2 平滑法与 Holt

from statsmodels.tsa.holtwinters import SimpleExpSmoothing, Holt # 简单指数平滑 ses = SimpleExpSmoothing(df['value']).fit( smoothing_level=0.3, optimized=False) # Holt 双参数（自动优化 α, γ） holt = Holt(df['value']).fit() forecast = holt.forecast(5) # 预测未来 5 期 # 移动平均 df['MA_3'] = df['value'].rolling(window=3).mean()

9.3 ARIMA 完整建模

from statsmodels.tsa.arima.model import ARIMA # 若不平稳，先差分 df['diff1'] = df['value'].diff() # 一阶差分 # 拟合 ARIMA(1,1,1) model = ARIMA(df['value'], order=(1, 1, 1)) result = model.fit() print(result.summary()) # 残差白噪声检验（模型诊断） lb_resid = acorr_ljungbox(result.resid, lags=[10]) print(lb_resid) # p>0.05 → 残差为白噪声，模型充分 ✓ # 预测 + 置信区间 forecast = result.get_forecast(steps=5) print(forecast.predicted_mean) # 点预测 print(forecast.conf_int()) # 95% 置信区间

10. 数学建模竞赛应用要点

竞赛完整决策路线

  1. 画时序图 → 观察趋势、季节、周期、异常值
  2. 平稳性检验 → 时序图 + ACF + ADF 检验
  3. 白噪声检验 → Ljung-Box。若白噪声 → 无分析价值，换方法
  4. 平稳 → 平滑法 或 ARMA
  5. 非平稳 → 差分 + ARIMA，或 Holt / 分解法 / SARIMA
  6. 多方法对比 → 选 MSE/MAPE 最小者
  7. 残差检验 → 残差必须为白噪声（Ljung-Box p>0.05）
  8. 论文要求 → 给出点预测 + 区间预测 + 模型诊断

常见陷阱与对策

陷阱后果对策
不检验平稳性直接建 ARMA伪回归，模型无效先 ADF + 差分
不做白噪声检验对无规律序列硬建模建模前 LB 检验
只看 AIC 不看残差诊断残差非白噪声，信息未提完残差 LB 检验必须做
过度差分（$d$ 太大）丢失信息，模型变差ADF p<0.05 就停
小样本（$n<30$）硬上 ARIMA参数估计不可靠用指数平滑或灰色预测 GM(1,1)
忽略季节性月度/季度数据预测失败SARIMA 或季节分解
多项式阶数过高过拟合，预测性能反而差限制 $k \leq 3$，交叉验证
