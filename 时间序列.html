1. 时间序列概述

📖 定义

时间序列 (Time Series)：按时间顺序记录的一组数据。观测时间 $t$ 可以是年份、季度、月份或任何时间单位，观测值记为 $Y_t$。

1.1 时间序列的四种成分

成分英文特征典型场景
趋势 TTrend持续向上或向下的长期变动GDP增长、人口增长
季节变动 SSeasonal一年内重复出现的周期性波动空调销量（夏高冬低）
循环波动 CCyclical非固定长度的周期性变动（>1年）经济周期（繁荣→衰退）
不规则波动 IIrregular随机波动，不可预测自然灾害、突发事件
🔑 乘法模型：$Y_i = T_i \times S_i \times C_i \times I_i$
加法模型：$Y_i = T_i + S_i + C_i + I_i$
平稳序列：只含不规则波动、不含趋势的序列。均值不随时间变化。

2. 平稳时间序列理论

2.1 严平稳 vs 宽平稳

严平稳 (Strict Stationarity)

序列所有统计性质都不会随时间推移而变化。任意 $m$ 个时刻的联合分布与时间平移 $\tau$ 后的联合分布完全相同：

$F(x_{t_1}, \dots, x_{t_m}) = F(x_{t_1+\tau}, \dots, x_{t_m+\tau})$

⚠️ 条件严格，实际数据太难达到。

宽平稳 (Weak Stationarity) 竞赛最常用

只保证序列低阶矩（均值、方差、协方差）的平稳性，满足三个条件：

  1. $E(X_t) = \mu$（常数均值）
  2. $E(X_t^2) < \infty$（二阶矩存在）
  3. $\gamma(t, s) = \gamma(k)$，$k = |t-s|$（自协方差只依赖于时间间隔）

2.2 平稳性的意义

  1. 极大减少随机变量的个数，增加待估参数的样本容量
  2. 历史数据的参数估计结果稳定，可直接用于未来预测
  3. 简化难度 + 提高精度

3. 平稳序列的预测方法（平滑法）

📌 适用条件：序列不含趋势和季节变动，波动主要来自随机成分。通过平滑消除随机波动。

3.1 移动平均法 (Moving Average)

选择固定移动间隔 $k$，将最近 $k$ 期数据的简单平均作为下一期预测值：

$$F_{t+1} = \frac{Y_t + Y_{t-1} + \dots + Y_{t-k+1}}{k}$$

特点：

  • 每期数据等权重
  • 只用最近 $k$ 期，旧数据自动丢弃
  • $k$ 越大 → 越平滑但越滞后；$k$ 越小 → 越敏感但噪声大
  • 选 $k$ 方法：试验使 MSE 最小的 $k$

3.2 简单指数平滑法 (Simple Exponential Smoothing)

基本思想：近期数据权重大，远期数据权重呈指数衰减。

$$F_{t+1} = \alpha Y_t + (1-\alpha)F_t$$

其中 $\alpha \in (0,1)$ 为平滑系数：

$\alpha$ 取值效果适用场景
$\alpha \to 1$侧重近期值，响应快波动大、需要快速跟踪
$\alpha \to 0$侧重历史值，更平滑随机波动较大时
🔑 移动平均 vs 指数平滑
移动平均：等权重、固定窗口、旧数据突然丢弃
指数平滑：指数衰减权重、无限记忆、平滑过渡
竞赛建议：两种都试，选 MSE 更小的。

4. 趋势序列的预测

4.1 线性趋势预测

时间序列按固定斜率增长或下降时，拟合线性趋势方程：

$$\hat{Y}_t = b_0 + b_1 t$$

用最小二乘法估计 $b_0$（截距）和 $b_1$（斜率）。预测时直接将未来时刻 $t$ 代入方程。

4.2 Holt 双参数指数平滑 有趋势时必用

简单指数平滑在序列有趋势时会系统滞后。Holt 模型用两个平滑系数同时跟踪水平和趋势：

$$\begin{aligned} L_t &= \alpha Y_t + (1-\alpha)(L_{t-1} + b_{t-1}) \quad &\text{（水平方程）}\\ b_t &= \gamma(L_t - L_{t-1}) + (1-\gamma)b_{t-1} \quad &\text{（趋势方程）}\\ F_{t+m} &= L_t + b_t \cdot m \quad &\text{（m 步预测）} \end{aligned}$$

$\alpha$ 控制水平平滑，$\gamma \in (0,1)$ 控制趋势平滑。两个参数都通过最小化 MSE 选择。

4.3 非线性趋势预测

当序列不是线性增长时，可选用：

  • 指数曲线：$Y_t = b_0 e^{b_1 t}$（几何级数增长/衰减）
  • 多项式曲线：$Y_t = b_0 + b_1 t + b_2 t^2 + \dots$（二次、三次等）
  • Logistic 曲线：$Y_t = \frac{K}{1 + ae^{-rt}}$（S 型增长，有上限）

5. Box-Jenkins 方法：ARMA / ARIMA 竞赛重点

📌 核心思想：不直接拟合时间 $t$ 的函数，而是利用序列自身的自相关结构建模。

5.1 模型族

模型全称适用数学形式
AR(p)自回归当前值与过去值相关$X_t = c + \sum_{i=1}^p \phi_i X_{t-i} + \varepsilon_t$
MA(q)移动平均当前值与过去误差相关$X_t = \mu + \varepsilon_t + \sum_{i=1}^q \theta_i \varepsilon_{t-i}$
ARMA(p,q)自回归移动平均平稳序列$X_t = c + \sum_{i=1}^p \phi_i X_{t-i} + \varepsilon_t + \sum_{i=1}^q \theta_i \varepsilon_{t-i}$
ARIMA(p,d,q)差分自回归移动平均非平稳→差分后平稳$d$ 阶差分后变 ARMA
SARIMA季节 ARIMA有季节周期ARIMA × 季节差分

5.2 ARIMA 建模流程

  1. 平稳性检验：ADF 检验（$H_0$: 非平稳，p<0.05 拒绝 → 平稳）
  2. 差分：若不平稳，做 $d$ 阶差分直到通过 ADF 检验
  3. 确定 p, q：观察 ACF（截尾→MA）和 PACF（截尾→AR）图的拖尾/截尾模式；或用 AIC/BIC 自动选择
  4. 参数估计：最大似然或最小二乘
  5. 残差诊断：残差应为白噪声（Ljung-Box 检验 p>0.05）
  6. 预测：用拟合模型做点预测和区间预测

5.3 ACF / PACF 定阶规则

模型ACF（自相关）PACF（偏自相关）
AR(p)拖尾（指数衰减或振荡）截尾于 p 阶
MA(q)截尾于 q 阶拖尾（指数衰减或振荡）
ARMA(p,q)拖尾拖尾

6. 预测方法的选择与评估

6.1 方法选择路线

序列特征推荐方法备选
平稳（无趋势/无季节）移动平均 / 指数平滑ARMA
有线性趋势Holt 双参数平滑线性回归 / ARIMA
有趋势+季节Winter 三参数平滑SARIMA
非线性趋势指数/多项式曲线拟合Logistic
复杂非平稳ARIMA / SARIMAProphet / 深度学习

6.2 误差度量

$$\begin{aligned} \text{MAE} &= \frac{1}{n}\sum |Y_t - F_t| \\ \text{MSE} &= \frac{1}{n}\sum (Y_t - F_t)^2 \\ \text{MAPE} &= \frac{100\%}{n}\sum \left|\frac{Y_t - F_t}{Y_t}\right| \end{aligned}$$

竞赛首选 MSE，MAPE 适合跨量纲比较。

7. Python 代码实现

7.1 移动平均与指数平滑

# 移动平均 import pandas as pd import numpy as np # 简单移动平均 k=3 df['MA_3'] = df['value'].rolling(window=3).mean() # 简单指数平滑 (α=0.3) from statsmodels.tsa.holtwinters import SimpleExpSmoothing model = SimpleExpSmoothing(df['value']).fit( smoothing_level=0.3, optimized=False ) df['SES'] = model.fittedvalues # Holt 双参数平滑 from statsmodels.tsa.holtwinters import Holt model = Holt(df['value']).fit() forecast = model.forecast(5) # 预测未来5期

7.2 ARIMA 建模

from statsmodels.tsa.arima.model import ARIMA from statsmodels.tsa.stattools import adfuller # 1. ADF 平稳性检验 result = adfuller(df['value']) print('ADF p-value: {:.4f}'.format(result[1])) # 2. 若不平稳，做差分 df['diff1'] = df['value'].diff() # 一阶差分 # 3. 拟合 ARIMA(1,1,1) model = ARIMA(df['value'], order=(1, 1, 1)) result = model.fit() print(result.summary()) # 4. 预测 forecast = result.forecast(steps=5) # 5. 残差诊断 from statsmodels.stats.diagnostic import acorr_ljungbox lb_test = acorr_ljungbox(result.resid, lags=[10]) print(lb_test) # p>0.05 → 残差为白噪声 ✓

7.3 自动 ARIMA

8. 数学建模竞赛应用要点

🎯 什么时候用时间序列方法？
赛题涉及"预测未来数值"且数据是按时间顺序排列的（如历年GDP、月度销量、日温记录）。

竞赛快速决策路线

  1. 画图 → 观察趋势、季节、周期
  2. ADF 检验 → 判断是否平稳
  3. 平稳 → ARMA / 指数平滑
  4. 非平稳 → 差分后 ARIMA / Holt / SARIMA
  5. 多方法对比 → 选 MSE/MAPE 最小的
  6. 残差检验 → 确保残差为白噪声
常见陷阱解决方案
直接把非平稳序列放进 ARMA先做 ADF 检验 + 差分
只看 AIC 不看残差残差白噪声检验是必须的
过度差分（d 太大）ADF p<0.05 就停，不要差分过度
样本太小（n<30）用简单方法（指数平滑/灰色预测），别硬上 ARIMA
不考虑季节因素月度/季度数据 → 必须考虑 SARIMA 或季节差分
