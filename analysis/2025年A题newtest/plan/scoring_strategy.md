# 评分闭环与模型路线策略

本文件由 `modeling-paper-rubric-and-model-selector` 根据结构化题意分析生成，用于指导后续 QA 与微单元写作。

## 全局原则

- 每一问必须形成“任务定义 -> 模型建立 -> 求解结果 -> 验证检验 -> 回答原问”的闭环。
- `model_route.json` 是后续写作的模型路线交接单，不能在正文生成时随意偷换主模型。
- `rubric_alignment.json` 是评分点与证据形式的映射，QA 应逐条检查是否落实。
