# 2025年CUMCM A题：烟幕干扰弹的投放策略 — 解题思路

## 一、问题重述与建模目标

### 场景描述
来袭空地导弹以300 m/s飞向假目标，真目标位于假目标后方200m处。无人机挂载烟幕干扰弹，需在导弹与真目标之间形成烟幕遮蔽。

### 核心参数
| 对象 | 参数 | 数值 |
|------|------|------|
| 导弹 M1/M2/M3 | 速度 | 300 m/s，直指假目标原点 |
| 无人机 FY1-FY5 | 速度 | 70~140 m/s，等高度直线飞行 |
| 烟幕干扰弹 | 投放间隔 | ≥1 s（同机） |
| 烟幕云团 | 半径/持续时间 | 10m / 20s（起爆后） |
| 烟幕云团 | 下沉速度 | 3 m/s（匀速） |
| 假目标 | 形状/尺寸 | 圆柱，半径7m，高10m |
| 真目标 | 位置 | 下底面圆心 (0, 200, 0) |

### 坐标与初始位置
- 假目标：原点 (0, 0, 0)
- 真目标下底面圆心：(0, 200, 0)
- 导弹：M1(20000,0,2000), M2(19000,600,2100), M3(18000,-600,1900)
- 无人机：FY1(17800,0,1800), FY2(12000,1400,1400), FY3(6000,-3000,700), FY4(11000,2000,1800), FY5(13000,-2000,1300)

## 二、物理建模

### 2.1 导弹运动模型

导弹以恒定速度 300 m/s 朝向假目标原点 (0,0,0) 飞行。

导弹 Mi 初始位置 $(x_{m0}, y_{m0}, z_{m0})$，速度方向指向原点：

$$\vec{v}_m = -300 \cdot \frac{(x_{m0}, y_{m0}, z_{m0})}{\sqrt{x_{m0}^2 + y_{m0}^2 + z_{m0}^2}}$$

t 时刻导弹位置：

$$\vec{P}_m(t) = (x_{m0}, y_{m0}, z_{m0}) + \vec{v}_m \cdot t$$

**导弹命中假目标时间**：$t_{hit} = \frac{\sqrt{x_{m0}^2 + y_{m0}^2 + z_{m0}^2}}{300}$

### 2.2 无人机运动模型

无人机受领任务后，立即调整航向，以恒定速度 $v_d \in [70, 140]$ m/s 等高度匀速直线飞行。

无人机从初始位置 $(x_{d0}, y_{d0}, z_{d0})$，以速度 $v_d$ 沿方向角 $\theta$（水平面内）飞行：

$$\vec{P}_d(t) = (x_{d0} + v_d \cos\theta \cdot t,\; y_{d0} + v_d \sin\theta \cdot t,\; z_{d0})$$

### 2.3 烟幕干扰弹运动模型（投放→起爆）

干扰弹脱离无人机后，在重力作用下做**抛体运动**。

投放时刻 $t_{drop}$，投放点即无人机位置 $\vec{P}_d(t_{drop})$，干扰弹具有无人机的瞬时速度 $\vec{v}_d$。

起爆时刻 $t_{burst} = t_{drop} + \Delta t_{delay}$（$\Delta t_{delay}$ 为延迟起爆时间）。

起爆点位置（仅重力影响，忽略空气阻力）：

$$x_b = x_d(t_{drop}) + v_{dx} \cdot \Delta t_{delay}$$
$$y_b = y_d(t_{drop}) + v_{dy} \cdot \Delta t_{delay}$$
$$z_b = z_d(t_{drop}) + v_{dz} \cdot \Delta t_{delay} - \frac{1}{2}g(\Delta t_{delay})^2$$

其中 $g = 9.8 \text{ m/s}^2$，$\vec{v}_d = (v_d\cos\theta,\; v_d\sin\theta,\; 0)$（等高度飞行，$v_{dz}=0$）。

简化：因为无人机等高度飞行，$v_{dz} = 0$，所以：
$$z_b = z_{d0} - \frac{1}{2}g(\Delta t_{delay})^2$$

### 2.4 烟幕云团模型（起爆后）

起爆后瞬时形成**球状**烟幕云团，半径为 $R = 10$ m。云团中心以 $v_{sink} = 3$ m/s 匀速下沉。

$$\vec{P}_{cloud}(t) = \vec{P}_b + (0, 0, -3) \cdot (t - t_{burst}), \quad t \ge t_{burst}$$

有效遮蔽条件：$t - t_{burst} \le 20$ s 且导弹到真目标的视线穿过云团球体。

### 2.5 遮蔽判断（核心几何问题）

导弹到真目标的视线是否被烟幕云团遮挡？

**视线参数方程**（导弹 → 真目标下底面圆心）：

$$\vec{P}_{LOS}(s) = \vec{P}_m(t) + s \cdot (\vec{P}_{target} - \vec{P}_m(t)), \quad s \in [0, 1]$$

其中 $\vec{P}_{target} = (0, 200, 0)$。

**遮蔽条件**：视线与球体（云团）相交，即存在 $s \in [0, 1]$ 使得：

$$\|\vec{P}_{LOS}(s) - \vec{P}_{cloud}(t)\| \le R$$

这等价于球心到直线的距离 $d \le R$，且交点在线段范围内。

**球心到直线的距离**：

$$d = \frac{\|(\vec{P}_{cloud} - \vec{P}_m) \times (\vec{P}_{target} - \vec{P}_m)\|}{\|\vec{P}_{target} - \vec{P}_m\|}$$

其中 $\times$ 为向量叉积。

**遮蔽时间窗口**：导弹飞行过程中，满足 $d \le R$ 且 $t \in [t_{burst}, t_{burst}+20]$ 的连续时间段。

## 三、逐题分析与求解策略

### 问题 1（确定性计算）

**条件**：FY1 以 120 m/s 朝向假目标飞行。受领任务后 1.5s 投放，间隔 3.6s 起爆。

**求解步骤**：

1. 计算 FY1 飞行方向：从 FY1(17800,0,1800) 朝向 (0,0,0)
   - $\vec{v}_d = -120 \cdot \frac{(17800, 0, 1800)}{\sqrt{17800^2 + 1800^2}}$
   
2. 投放点：$\vec{P}_{drop} = \vec{P}_{FY1} + \vec{v}_d \cdot 1.5$

3. 起爆点：考虑重力下落
   - $\vec{P}_{burst} = \vec{P}_{drop} + \vec{v}_d \cdot 3.6 + (0, 0, -\frac{1}{2}g \cdot 3.6^2)$

4. 导弹 M1 轨迹：$\vec{P}_{M1}(t) = (20000,0,2000) + \vec{v}_{m1} \cdot t$

5. 计算视线-球体相交的时间窗口

6. **输出**：有效遮蔽时长 = 相交窗口 ∩ [起爆时刻, 起爆+20s] 的总时长

**方法**：纯运动学计算，无优化。

### 问题 2（单无人机单弹优化）

**决策变量**：
- 飞行方向角 $\theta$（水平面内）
- 飞行速度 $v_d \in [70, 140]$
- 投放时刻 $t_{drop}$
- 起爆延迟 $\Delta t_{delay}$

**目标**：最大化遮蔽时长

**约束**：
- $v_d \in [70, 140]$
- $t_{drop} \ge 0$
- 起爆时无人机已飞离安全距离（可能需要考虑）
- 烟幕云团中心在导弹视线路径附近

**求解策略**：

1. **网格搜索 + 局部优化**
   - 对 $\theta$ 和 $v_d$ 做粗粒度网格搜索
   - 对每组 $(\theta, v_d)$，优化 $t_{drop}$ 和 $\Delta t_{delay}$
   
2. **几何分析简化**
   - 理想遮蔽位置：烟幕云团在导弹与真目标连线的"中途"
   - 起爆时机需精确匹配导弹到达遮蔽区域的时间
   
3. **推荐方法**：
   - 先用**几何推导**缩小搜索空间：理想的起爆点应在导弹飞行路径附近
   - 再用**遗传算法(GA)或粒子群(PSO)** 做精细优化
   - 也可使用**蒙特卡洛采样** + 局部爬山

### 问题 3（单无人机多弹时序优化）

**新增约束**：
- 同机投放间隔 ≥ 1 s
- 3枚弹可形成多次遮蔽（可不连续）

**决策变量**：
- $\theta, v_d$（共用）
- $(t_{drop}^{(1)}, \Delta t_{delay}^{(1)}), (t_{drop}^{(2)}, \Delta t_{delay}^{(2)}), (t_{drop}^{(3)}, \Delta t_{delay}^{(3)})$

**求解策略**：

1. **序列优化**
   - 先优化第1枚弹（同问题2），得到其遮蔽窗口
   - 在第1枚遮蔽窗口之外，优化第2枚弹填补助空白
   - 类似地优化第3枚
   
2. **联合优化**
   - 所有变量一起优化，使用 **PSO 或 GA**
   - 目标函数 = 3枚弹的遮蔽时长并集总长度

3. 输出到 result1.xlsx（按模板格式）

### 问题 4（多无人机协同，各1枚弹）

**决策变量**：
- FY1, FY2, FY3 各有一组 $(\theta_i, v_{di}, t_{drop}^{(i)}, \Delta t_{delay}^{(i)})$

**新增复杂度**：3架无人机从不同位置出发，需协同覆盖导弹M1的飞行路径

**求解策略**：

1. **空间分工**
   - 将导弹飞行路径分成3段（近/中/远）
   - 每架无人机负责一段的遮蔽
   
2. **联合优化**
   - 直接用 **PSO/GA** 对全部12个变量做联合优化
   - 或先对每架单独优化，再微调协调

3. 输出到 result2.xlsx

### 问题 5（全系统协同，5机×3弹，3枚导弹）

**规模**：
- 5架无人机，每架≤3枚弹 → 最多15枚弹
- 3枚来袭导弹需分别遮蔽
- 决策变量：每架无人机的 $(\theta_i, v_{di})$ + 每枚弹的 $(t_{drop}, \Delta t_{delay})$

**求解策略**：

1. **分层优化**
   - 第1层：导弹分配 — 哪几架无人机负责哪枚导弹
   - 第2层：每枚导弹的遮蔽计划（类似问题4）
   - 第3层：弹的投放时序

2. **整体优化**
   - 使用 **GA（遗传算法）** 处理离散+连续混合变量
   - 编码方案：每架无人机 → (航向角, 速度, [弹1投放时刻, 弹1延迟, 弹2投放时刻, ...])
   - 适应度函数：3枚导弹各自遮蔽时长之和（或加权）

3. 输出到 result3.xlsx

## 四、数学模型总结

### 核心方程

**1. 遮蔽判断函数**（最关键）：

```python
def is_shielded(missile_pos, cloud_center, cloud_radius, target_pos):
    """判断导弹→目标的视线是否穿过烟幕球体"""
    # 球心到直线的距离
    v = target_pos - missile_pos
    w = cloud_center - missile_pos
    t_proj = np.dot(w, v) / np.dot(v, v)
    t_proj = np.clip(t_proj, 0, 1)  # 限制在线段范围内
    closest = missile_pos + t_proj * v
    dist = np.linalg.norm(closest - cloud_center)
    return dist <= cloud_radius
```

**2. 遮蔽时长计算**：
- 对时间做离散采样（步长 0.01s）
- 统计满足遮蔽条件的时间点数
- 对多枚弹取并集

**3. 优化目标**：
$$\max \sum_{i} \text{coverage\_duration}_i$$
其中 $i$ 为被掩护的导弹编号。

### 推荐求解算法

| 问题 | 方法 | 原因 |
|------|------|------|
| 问题1 | 直接计算 | 所有参数已给定 |
| 问题2 | PSO 或 GA | 4变量连续优化 |
| 问题3 | GA + 贪心序列 | 混合整数（时序约束） |
| 问题4 | GA/PSO | 12变量协同优化 |
| 问题5 | GA（分层编码） | 大规模混合优化 |

### 关键技巧

1. **时间离散化**：以 0.01-0.05s 为步长，将连续遮蔽判断转为离散统计
2. **几何预处理**：计算每条导弹轨迹到达各位置的时间，预判遮蔽窗口
3. **约束处理**：投放间隔约束用罚函数法
4. **遮蔽效率**：理想的起爆点应在导弹视线路径上或非常接近
